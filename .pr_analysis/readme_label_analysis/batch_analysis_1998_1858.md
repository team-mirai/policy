この変更は、匿名ユーザー様からのご提案に基づき、チームみらいのマニフェストにおける経済財政政策の方向性をより明確にするためのものです。

匿名ユーザー様からは当初、国民年金および健康保険制度を廃止し、ベーシックインカムを導入することで「最低限の生活自由の解放と、各自が本当に熱中したいことに集中できる社会」を実現したいというご意見をいただきました。

このご提案について議論を深める中で、ベーシックインカム導入を通じて目指すべき社会の姿として、以下の3つの重要な柱が明確になりました。

1.  **個人の尊厳と選択肢の拡充:** すべての国民が「人間らしい暮らし」を営む権利を保障し、働き方、学び直し、子育て、地域活動など、多様な生き方を主体的に選択できる社会を目指します。
2.  **安定と成長の好循環:** ベーシックインカムが消費とイノベーションを下支えし、持続的な経済成長を実現します。また、格差や貧困の問題を是正し、社会全体の活力を向上させることを目指します。
3.  **新しい社会契約の構築:** 税と社会保障給付のあり方を見直し、その透明性を高めることで、国民と行政との間に新たな信頼関係を築きます。将来的には地方分権と連動した形での“日本版ベーシックインカム”の全国展開も視野に入れます。

データの視覚的提供の実現にあたっては、MITのケーシー・リース氏やベン・フライ氏による過去のデータグラフィックスプロジェクトのような先進事例を参考にし、動的インフォグラフィックスやデータグラフィックスの専門デザイナーの知見を積極的に取り入れるべきです。特に、エドワード・タフティ氏の研究に代表されるように、データグラフィックスを単なる情報の視覚化に留めず、危機管理や事故の回避、システムの問題点発見に繋げる工学的アプローチとして捉えることが重要です。これにより、具体的なリスクや無駄遣いを特定し、排除することを目指します。

この提案は、匿名ユーザーさんからの貴重なご意見に基づいています。具体的には、税金と、年金と、社会保障が、どのように使われているのか一覧できていない現状への問題提起から始まっています。現状では、これらの情報を知りたいと思っても、大量のエクセルや書類を読み込む必要があり、国民にとってアクセスが容易ではありません。例えば、消費税が本当に社会保障に使われているのか、復興税や森林税がその用途のみで使われているのか、年金がどのように高齢者に渡り、受け取られなかった年金はどこへ行くのか、といった疑問は尽きません。税金の入り口と出口、その使われ方、お金の「色」がどのように変わっていくのか、その流れが合理的であるのかどうかを、誰にでも開かれた形で、継続的に視覚的に表現し、アーカイブしていくことの重要性が指摘されました。これは、政治資金のような政党や個人の資金の流れだけでなく、より広範な公金全体の透明性を高める試みです。動的なデータグラフィックスを用いることで、複雑な財政システムをベルトコンベアのように視覚的に表示し、国民一人ひとりがお金の流れを直感的に理解できるようにすることを目指すべきです。さらに、この視覚化は単に情報を提示するだけでなく、財政システムに内在するリスクや非効率な支出、無駄遣いを早期に発見し、改善につなげるための工学的なツールとして機能することが期待されます。エドワード・タフティ氏が提唱するように、優れたデータグラフィックスは、複雑な情報を明確に伝達し、より良い意思決定を支援する力を持っています。このアプローチを採用することで、政策決定プロセス全体の透明性と合理性を向上させ、国民の信頼を醸成することができると考えられます。この提案は、匿名ユーザーさんの深い洞察と国民目線に立った問題意識から生まれたものであり、チームみらいのマニフェストをより実践的で国民に開かれたものにするための重要な一歩となるでしょう。

### PR本文
**本提案の背景**

日本の農業は、従事者の高齢化と深刻な後継者不足という大きな課題に直面しています。このままでは、数年後には多くの農家が離農し、国内の食料生産基盤が大きく揺らぐ可能性があります。これは、国民の食生活の安定供給はもちろんのこと、国の安全保障にも関わる喫緊の課題です。現行のチームみらいマニフェストver.0.1では、この点の具体的な対策が十分に示されていないとの課題意識から、今回の提案に至りました。提案者は農家のシロさんです。

**提案内容：マニフェストへの「農業政策」項目の追加**

上記背景を踏まえ、マニフェストの「ステップ３『長期の成長に大胆に投資する』」の中に、新たに「７．農業政策（仮称）」という項目を追加することを提案します。これにより、農業政策を持続可能で成長できる分野として戦略的に位置づけ、具体的な施策を議論・推進していくための明確な枠組みを設けます。

**具体的な政策アイデア：「植物工場団地」構想による食料自給率の向上**

新しい「農業政策」項目で議論・推進すべき具体的なアイデアの一つとして、シロさんより「植物工場団地」構想が提案されました。これは、最新の施設園芸技術を活用し、大規模な植物工場団地を建設・運営するというものです。

*   **目的：** 国内の食料自給率の向上。特に、トマト、キュウリ、イチゴ、ナス、ピーマンといった、国民の食生活に不可欠でありながら天候等に左右されやすい作物の安定供給を目指します。
*   **メリット：**
    *   天候に左右されない計画的な生産と供給が可能になります。
    *   農薬使用量の削減や品質管理の徹底により、安全・安心な農産物を提供できます。
    *   集約的な生産システムにより、高齢化や担い手不足の問題を緩和し、新たな雇用機会を創出する可能性も秘めています。
    *   将来的には、ここで磨き上げられた生産技術や栽培ノウハウ、さらには植物工場システム自体をパッケージとして海外に輸出することも視野に入れます。これにより、日本の農業技術を世界の食料問題解決に貢献させるとともに、新たな外貨獲得手段とすることも期待できます。

**「ステップ３」への追加理由**

この「植物工場団地」構想は、初期投資や技術開発に相応の期間と資源を要する大規模なプロジェクトです。短期的な成果のみを追求するのではなく、日本の農業の未来を見据えた長期的な視点からの大胆な投資が必要となります。そのため、マニフェストの中でも「ステップ３『長期の成長に大胆に投資する』」に本項目を追加することが最も適切であると考えます。

**まとめ**

今回の提案は、日本の農業が抱える構造的な課題に対応し、食料安全保障を強化するとともに、農業を成長産業へと転換させる可能性を秘めたものです。マニフェストに「農業政策」の項目を新設し、そこで「植物工場団地」構想のような革新的なアイデアを具体化していくことは、チームみらいが掲げる「テクノロジーで誰も取り残さない日本」の実現に大きく貢献すると信じています。この提案が実を結び、日本の農業、そして国民の未来にとって明るい一歩となることを期待します。

現在、日本に居住する外国人の方々の国民健康保険料の収納率は約6割に留まっており、9割を超える日本国民の収納率と比較して低い水準にあります。この差は、国民健康保険制度の公平性や財政的持続可能性の観点から大きな課題です。本提案は、この課題解決に向けた具体的な道筋を示し、年間数千億円規模と試算される保険料収入の増加を通じて、制度の安定化に貢献することを目的としています。これは、チームみらいが掲げる「デジタル時代の当たり前をやりきる」というステップ１の行政改革にも合致するものです。

この課題解決のため、チームみらいの国会議員と専門知識を有するエンジニアチームが中心となり、省庁、自治体、そして外部の有識者と緊密に連携し、以下の施策を推進します。

1.  **データに基づいた現状分析と課題の可視化:**
    エンジニアチームが主導し、関連省庁や自治体が保有するデータを横断的に分析します。収納率が低い地域や層の特定、滞納理由の傾向などを詳細に把握し、具体的な課題を明確にします。これにより、画一的ではない、きめ細やかな対策の立案が可能になります。

    取り組みの進捗状況や、それによって得られた成果（収納率の具体的な改善数値など）を、国会に対して定期的に報告するとともに、マスコミを通じて国民にも広く、分かりやすく公表します。これにより、政策実行の透明性を高め、国民からの信頼と協力を得ながら取り組みを推進します。

**提案内容：**

訪日外国人観光客に対する海外旅行保険（医療保険および損害保険）の加入義務化を提案します。

**提案の背景と目的：**

近年、訪日外国人観光客の増加に伴い、滞在中の病気や怪我による医療費の未払いや、事故発生時の賠償金未収といった問題が顕在化しています。これらの未収金は、医療機関や関係者の大きな負担となっており、日本の医療提供体制や国際親善にも影響を及ぼしかねない状況です。

本提案は、訪日外国人観光客に海外旅行保険への加入を義務付けることで、これらの未収金問題を解決し、観光客自身にとっても安心して日本に滞在できる環境を整備することを目的としています。また、受け入れ側の負担を軽減することは、将来にわたる持続的なインバウンド消費の向上、ひいては日本経済の活性化にも貢献すると考えます。

**具体的な提案と議論された論点：**

1.  **義務化する保険の種類：**
    *   滞在中の疾病・傷害治療費用をカバーする「医療保険」
    *   他人への損害賠償責任をカバーする「損害保険」
    上記2種類の保険への加入を義務とします。補償限度額については、日本の医療費水準や過去の事例を参考に、十分な額を設定する必要があります。

2.  **加入確認方法：**
    *   入国審査時に、保険加入証明書の提示を求める。
    *   または、事前にオンラインシステムを通じて保険加入情報を登録・提出させる。
    これらの方法により、確実な加入確認を目指します。

3.  **未加入者への対応：**
    *   もし入国時に保険加入が確認できない場合は、空港内に設置された保険カウンター等で、最低限の補償内容を持つ保険にその場で加入することを条件とします。これにより、実質的な入国拒否を避けつつ、制度の実効性を担保します。

4.  **観光客誘致への影響について：**
    *   保険加入義務化による費用負担増が、一部の観光客にとって訪日をためらう要因となる可能性は否定できません。しかし、現状でも急激な観光客増によるオーバーツーリズムや、未収金問題といった弊害が指摘されています。保険加入は、観光客自身にとっても、万が一の高額な医療費負担や賠償責任から身を守る手段となり、結果としてより安全で安心な旅行体験を提供することに繋がります。これは、長期的に見て日本の観光魅力の向上に資すると考えます。

5.  **観光客へのメリット：**
    *   旅行中の不測の事態（病気、怪我、事故など）に遭遇した場合でも、保険によって経済的な負担が大幅に軽減されます。これは、観光客が安心して旅行を楽しむための重要なセーフティネットとなります。

**結論として：**

訪日外国人観光客の海外旅行保険加入義務化は、医療費等の未収問題という喫緊の課題に対応する有効な手段です。また、観光客と受け入れ側双方にとってメリットがあり、日本の国際的な信頼性を高め、持続可能な観光立国の実現に貢献するものと考えます。本提案について、さらなる議論を深め、実現に向けた検討を進めていただきたくお願い申し上げます。

## PR #1888: akiさん提案：「子ども」政策への呼称変更及び教育・子ども分野の課題追記と改善提案

### 基本情報
- **作成者**: app/idobata-policy-app
- **作成日**: 2025-06-01T15:14:51Z

### PR本文
本提案は、akiさんからの具体的な問題提起と改善案に基づき、チームみらいのマニフェストをより現状に即し、かつ未来志向のものへと発展させるためのものです。akiさんは、現行のセーフティネットからこぼれ落ちてしまう人々を包み込む「セーフティスポンジ構想」という理念をお持ちで、今回の提案はその構想を教育・子ども分野で具現化する第一歩と言えます。

主な変更点は以下の通りです。

1.  **「子育て」から「子ども」への呼称変更:**
    マニフェスト全体にわたり、「子育て」に関する記述を「子ども」に変更しました。これは、0歳から18歳までの子どもに関わる問題は、現役の子育て世代だけでなく、全ての世代が社会全体で当事者意識を持って取り組むべき喫緊の課題であるという認識に基づきます。「子ども政策」とすることで、より広範な視点と包摂的なアプローチを促し、少子化が急速に進む日本において、社会全体で子どもたちの未来を支える意志を明確にすることを目的としています。akiさんのご友人である教育現場の専門家からも、この呼称変更について強い支持をいただきました。

2.  **教育・子ども分野における具体的な課題の追記:**
    akiさん、そしてそのご友人である教育現場の専門家（小学校教員、学童保育指導員）からのヒアリングを通じて明らかになった、喫緊の課題をマニフェストに追記しました。これは「このマニフェスト自身も、みんなの知恵を集めて改善していきます」という既存のセクションに、具体的な議論のポイントとして加える形を取りました。

    *   **不登校支援の抜本的改革の必要性:**
        現行のスクールカウンセラー制度では、学校に行けない子どもが学校内のカウンセリングルームを利用しなくてはならないという構造的な問題を抱えています。大人の復職支援プロセス（専門医の診断、リワーク施設での段階的プログラム、復職後の継続サポート）と比較し、子どもの不登校支援にはオンラインカウンセリング体制の未整備、小児精神科医の不足、登校訓練や模擬スタディプログラムといった段階的ケアを提供する機関の欠如など、多くの課題が存在します。学校に行かなくても心理的安全性を確保し、適切な支援を受けられるような、学校外のサポート拡充、専門家の配置、そして個々の状況に応じた段階的なケアプログラムの導入が急務です。

    *   **学童保育制度の質の向上と対象拡大:**
        現行の学童保育は原則小学校3年生までとされており、高学年の利用が困難な状況があります。特に特別な事情を抱える高学年の子どもが学童を利用する際に、周囲から浮いてしまい、いじめの対象となるリスクも指摘されています。学童保育を、特別な事情の有無に関わらず、全ての学年の子どもたちが誰でも気軽に通える、安全で楽しい「第2の家」のような場所に変革していく必要があります。対象年齢の拡大だけでなく、子どもたちの発達段階や興味関心に応じた多様なプログラムを提供し、子どもたちの自主性や社会性を育む場としての機能強化も求められます。

    *   **政策情報伝達と現場アクセスの改善:**
        国が発行する「こども白書」のような重要な政策文書が、教育現場の最前線で働く専門家（教員や指導員）にさえ十分に認知されておらず、内容も共有されていないという衝撃的な事実が明らかになりました。政策立案者と現場の間に大きな溝があり、現場の声が政策に反映されにくい構造、また、良い政策が現場で活かされないという双方向の断絶が存在します。白書の無償配布（学校や図書館への常備）やデジタルデータへの容易なアクセス確保は最低限必要であり、さらに政策内容を分かりやすく解説する資料の作成・配布、現場の専門家が政策形成プロセスに参加できる仕組みづくりなど、情報伝達とアクセシビリティの抜本的な改善が不可欠です。

これらの課題提起は、チームみらいが掲げる「誰も取り残さない」社会の実現に向けた重要な一歩です。akiさんは、今回の提案が教育・子ども分野における「セーフティスポンジ構想」の一端に過ぎず、今後さらに多くの専門家や経験者の知見を集約し、チームみらいのブロードリスニングの手法や「いどばたシステム」を活用して議論を深めていくことの重要性を強調されています。

## PR #1787

**タイトル**: 産業政策の強化：AI・ドローン活用と山間部農業振興について（阿井上夫氏提案）
**作成者**: app/idobata-policy-app
**作成日**: 2025-05-29T03:49:07Z
**URL**: https://github.com/team-mirai/policy/pull/1787

### PR本文
```
阿井上夫です。この度、チームみらいのマニフェスト「README.md」の産業政策分野において、具体的な政策提案を追加させていただきました。

**変更点**
「ステップ１：デジタル時代の当たり前をやりきる」の「４．産業」の項目に、以下の２つの具体的な政策を追加しました。
*   AIやドローンを活用した農業における気候や地形の特色に合わせた支援の促進
*   山間部農業の収益性向上の為に高付加価値農作物栽培の推進

**提案の意図・目的**
今回の提案の主な目的は、日本の農業が直面する様々な課題に対し、デジタル技術の活用と高付加価値化戦略を通じて具体的な解決策を示すことです。
1.  **AI・ドローン活用による農業支援の促進**: 近年進化の著しいAIやドローン技術を農業分野に積極的に導入し、気候変動への適応や各地域の地理的特性に応じた精密農業を推進します。これにより、生産性の向上、作業負担の軽減、環境負荷の低減を目指します。例えば、AIによる生育状況のリアルタイム分析や、ドローンによる効率的な農薬散布・種まきなどを想定しています。これは、農業従事者の経験や勘に頼る部分をデータで補強し、より客観的で科学的な農業経営を支援するものです。
2.  **山間部農業における高付加価値作物の推進**: 日本の国土の多くを占める山間部では、平野部に比べて農業生産条件が不利な場合が多く、収益性の低さから耕作放棄が進んでいます。この課題に対し、その土地ならではの気候や土壌を活かした高付加価値作物（例：特産果樹、希少な山菜、薬用作物など）の栽培を奨励・支援することで、山間部農業の収益性を向上させ、地域経済の活性化と雇用の創出に繋げます。これは、単に生産量を増やすのではなく、「質」で競争し、農家の所得向上と地域のブランド化を図る戦略です。

**背景**
日本の農業は、従事者の高齢化と後継者不足、それに伴う耕作放棄地の増加という深刻な構造問題を抱えています。加えて、地球温暖化に伴う異常気象の頻発や、グローバル化による安価な輸入品との競争激化など、外部環境も厳しさを増しています。特に山間地域では、これらの問題がより顕著に現れており、地域の活力が失われつつあります。
一方で、AI、IoT、ドローンといった先端技術は、農業分野においてもその活用が期待され、省力化、精密化、高品質化に貢献する可能性を秘めています。また、消費者のニーズも多様化しており、安全・安心で高品質な国産農産物や、その土地ならではの特色ある産品への関心が高まっています。
このような状況を踏まえ、デジタル技術の恩恵を農業現場隅々まで届け、特に条件不利地とされてきた山間部においても新たな収益機会を創出することが、日本の農業の持続可能性を高め、食料安全保障を確固たるものにする上で不可欠であると考え、今回の提案に至りました。この変更が、より具体的で実効性のある政策への第一歩となることを期待しています。
チームみらいの皆様、この提案についてぜひ前向きにご検討いただけますようお願い申し上げます。　阿井上夫
```

### PR本文
```
**提案者:** せさん

**提案内容:**
若い世代への早期かつ円滑な資産移転を強力に推進し、経済全体の活性化と若者の経済的自立を支援するため、現行の贈与税及び相続税の制度を見直すことを提案します。
主な変更点は以下の通りです。
1.  **年齢に応じた贈与税非課税枠の大幅な拡大:** 若い世代ほど大きな非課税枠を設定し、早期の資産移転を促します。例えば、20代には年間500万円程度の贈与枠を設けることを検討します。これにより、教育資金、起業資金、住宅取得資金など、若者が人生の早い段階で必要とする資金ニーズに応えます。
2.  **相続税の累進課税強化:** 贈与による早期の資産移転を促す一方で、相続発生時の資産集中に対しては、累進課税を強化することで、世代間の資産の再分配機能を高めます。

**提案の意図・目的:**
本提案の根本的な目的は、日本経済の持続的な成長と、全世代が安心して生活できる社会の実現です。
*   **若者の経済的自立支援と経済活性化:** 高齢者層に偏在しがちな「タンス預金」などの遊休資産を、NISAなどを通じた長期投資や実体経済へと効果的に循環させ、特に若い世代が活用できる資金を増やすことを目指します。これにより、若者の経済的自立を強力に後押しし、消費の拡大、新規ビジネスへの挑戦、イノベーションの促進など、経済全体の活性化に繋げます。若者の経済的安定は、結婚や出産といったライフイベントへの前向きな意識を醸成し、少子化対策にも間接的に貢献すると考えられます。
*   **世代間の資産移転の円滑化と最適化:** 現在の日本では、資産の多くが高齢者層に集中しており、次世代への移転がスムーズに進んでいないという課題があります。本提案は、生前贈与という形で、資産をより早いタイミングで、かつ必要としている若い世代へ移転させることを奨励するものです。これにより、資金が有効活用されず死蔵されることを防ぎ、社会全体の厚生を高めます。
*   **現役世代（氷河期世代等）への間接的支援:** 若い世代への贈与は、その親である現役子育て世代、特に経済的に厳しい状況に置かれがちな氷河期世代にとっても、子や孫への教育資金援助などを通じて間接的な経済支援となり得ます。これにより、世代間の助け合いを促進し、社会全体の安定に寄与します。
*   **高齢者の生きがいと社会貢献:** 高齢者にとっては、自身の築き上げた資産を、生きている間に次世代のために有効活用し、その成長を直接見守ることができるというのは、大きな喜びであり、社会貢献の実感にも繋がります。生前贈与の促進は、高齢者が自身の資産の使い道について主体的に決定し、次世代育成に積極的に関与する動機付けとなります。

**背景・課題認識:**
日本の個人金融資産の約6割が60歳以上に保有されていると言われ、その活用が十分でないことが指摘されています。一方で、若い世代は不安定な雇用や低賃金に苦しみ、経済的な不安から将来への希望を持ちにくい状況にあります。従来の相続制度では、資産移転のタイミングが故人の死後となり、平均寿命の伸長も相まって、実際に資金が若い世代の手元に渡る時期が遅くなりがちです。これは、若者が最も資金を必要とする20代～40代の時期とずれが生じていることを意味します。
また、現行の贈与税は基礎控除額が低く、暦年贈与の活用にも限度があるため、まとまった資産の生前移転には高いハードルが存在します。この結果、多くの資産が活用されないまま死蔵されたり、相続時に初めて大きな規模で移転したりするケースが多く見られます。
本提案は、こうした現状の課題認識に基づき、税制を通じて資産の流れをより生産的な方向へと誘導し、社会全体の活力を生み出すことを目指すものです。

**期待される効果:**
*   若年層の可処分所得増加による消費の拡大
*   起業・新規事業への投資促進とイノベーション創出
*   住宅市場の活性化
*   教育機会の拡大と人材育成
*   少子化傾向の緩和への貢献
*   世代間の経済格差の緩和
*   高齢者の資産活用意欲の向上と社会参加の促進
*   相続を巡る親族間の紛争リスクの低減

**その他:**
この政策は、あくまで「チームみらい マニフェスト ver.0.1 (超初期版）」への追加提案であり、具体的な非課税枠の金額、相続税率の設計、適用条件などの詳細については、今後、経済状況、財政状況、国民各層の意見、専門家の知見などを踏まえ、十分に議論し、丁寧に検討していく必要があります。また、他の経済政策、社会保障政策、教育政策などとの連携も視野に入れ、総合的な政策パッケージとして効果を最大化できるよう努めます。年齢に応じた非課税枠の設定においては、公平性の観点や制度の複雑化を避ける工夫も必要となるでしょう。また、NISAなどを通じた国民の安定的な資産形成が広く進んだ段階においては、金融所得に対する課税のあり方についても、社会全体の公平性や財政状況を総合的に勘案し、適宜見直しを検討していくべきです。
私たちは、この提案が、日本社会の未来を明るく照らし、世代間の信頼と協調を育む一助となることを強く願っています。せさんのこの貴重なご意見を真摯に受け止め、マニフェストの更なる改善に繋げて参ります。
```

## PR #1762

**タイトル**: SNS運用方針の追加と透明性向上
**作成者**: app/idobata-policy-app
**作成日**: 2025-05-28T05:01:46Z
**URL**: https://github.com/team-mirai/policy/pull/1762

### PR本文
```
この提案は、koyakeiさんの発案に基づき、チームみらいのマニフェストに新たに「SNSの運用方針」に関する章を追加するものです。この変更の主な目的は、政治家および政治団体としてのSNSアカウント運用における透明性と、ユーザーの権利への配慮を明文化することにあります。

**変更の背景と意図:**

ソーシャルメディア（SNS）は、現代社会において政治家や政治団体が有権者と直接コミュニケーションを取り、政策や活動について情報を発信し、また有権者からの意見を収集するための非常に重要なプラットフォームとなっています。しかし、その運用方法によっては、意図せずユーザーの表現の機会を奪ってしまったり、コミュニケーションを一方的なものにしてしまう可能性も指摘されています。

特に、政治家や公的な立場にあるアカウントが、批判的な意見を持つユーザーを安易にブロック（特定のユーザーからのアクセスや閲覧を完全に遮断する機能）することは、そのユーザーが当該アカウントの発信する情報にアクセスする権利や、意見を表明する機会を制限することに繋がりかねません。これは、健全な民主主義的な議論や、多様な意見が交わされるオープンな対話環境の阻害要因となる可能性があります。

そこで、本提案では、SNS運用における基本方針として、「できるだけ相手ユーザーの権利を制限しない運用を目指す」ことを明確に掲げます。具体的には、「政治家が特定のユーザーの投稿を閲覧したくない場合、相手の表示を制限する機能（ブロックなど）よりも、自身のタイムラインに表示させない機能（ミュートなど）を優先して使用する」という指針を設けます。

ミュート機能は、特定のアカウントの投稿を自身のタイムラインに表示させないようにするものであり、相手に通知されることなく、また相手が自身のアカウントの投稿を閲覧したり、リプライやダイレクトメッセージを送ったりする権利を制限するものではありません。これにより、政治家は見たくない情報から距離を置きつつも、相手の表現の自由やアクセス権を不当に侵害することを避けることができます。

この方針をマニフェストに明記することで、チームみらいとして、有権者とのオープンで建設的な対話を重視し、SNSをより民主的かつ公平に活用していく姿勢を示すことを目指します。また、この方針が他の政治家や政治団体にとっても一つの参考となり、SNSにおけるコミュニケーション全体の質の向上に貢献することも期待しています。

この変更は、特定の個人や団体を批判するものではなく、あくまでチームみらいとしてのSNS利用における理念と具体的な行動指針を示すものです。多様な意見に耳を傾け、建設的な議論を育むというチームみらいの基本姿勢を、SNS運用においても一貫して保つための重要な一歩であると考えます。

ご検討のほど、よろしくお願いいたします。
起案者: koyakei
```

```
**提案の背景と課題**

現行の所得税制度は、一年間の所得に対して課税する「現年所得課税」です。この方式では、以下のような課題が指摘されています。

1.  **所得変動への対応力不足**: 単年での課税であるため、豊作の年やプロジェクト成功時など、一時的に所得が急増した場合に税負担が過大になることがあります。逆に、病気や失業、事業の不振などで所得が急減した場合でも、過去の納税額に対する還付措置が限定的であり、生活の安定を損なう可能性があります。特に、フリーランスや非正規雇用など、所得が不安定な働き方が増える現代において、この問題はより深刻です。
2.  **納税者間の公平性の問題**: 所得の発生タイミングを調整すること（例えば、退職金の受け取り方を工夫するなど）で、意図的に税負担を軽減することが可能な場合があります。これにより、同様の生涯所得を得ていても、所得のパターンによって税負担に差が生じ、納税者間の公平性が損なわれるという問題があります。
3.  **技術的制約による妥協**: 所得税の単年主義は、過去の計算技術や情報管理能力の制約から、実務上の簡便性を優先して採用された側面があります。しかし、マイナンバー制度の導入やIT技術の飛躍的な発展により、個人の所得情報を長期間にわたり正確かつ効率的に追跡・管理することが十分可能となっています。

**提案内容：生涯所得課税の導入**

たしさんのご提案は、これらの課題を解決するために、納税者の生涯にわたる累積所得の平均に基づいて課税する「生涯所得課税」を導入するというものです。具体的な仕組みは以下の通りです。

1.  **課税ベースの変更**: 各年の課税所得を、その年までの累積総所得を年数で割った「生涯平均所得」とします。
2.  **累進税率の適用**: この生涯平均所得に対して、現行と同様の累進税率を適用して、その年までの「生涯で納めるべき税額の総額」を計算します。
3.  **納税額の決定**: 上記で計算された「生涯で納めるべき税額の総額」から、前年度までに既に納付した税額の合計を差し引いた額を、その年の納税額とします。
4.  **所得減少時の還付**: 所得が大幅に減少した場合、生涯平均所得も低下し、「生涯で納めるべき税額の総額」が既納付額を下回ることがあります。この場合、差額が還付されることになり、実質的な所得補償（保険機能）として機能します。

**期待される効果と意義**

生涯所得課税の導入により、以下のような効果が期待されます。

1.  **税負担の公平化と平準化**: 生涯を通じた所得に対して課税されるため、一時的な所得変動に左右されにくくなり、より公平で安定した税負担が実現します。これにより、起業や新しい挑戦に伴う所得変動リスクを軽減し、個人の多様なキャリア選択を後押しします。
2.  **現役世代への保険機能の提供**: 特に、社会保険制度の恩恵を十分に受けられていないと感じる現役世代や、所得が不安定な立場にある人々にとって、所得減少時の税還付は大きな安心材料となり、セーフティネットとしての役割を果たします。これは、社会全体の安定にも寄与します。
3.  **税制の簡素化と透明性の向上**: 長期的には、複雑な所得控除や年度調整の必要性が減少し、税制全体の簡素化と透明性の向上に繋がる可能性があります。
4.  **デジタル時代の新たな税制モデル**: マイナンバー等のデジタル基盤を活用することで、かつては困難であった生涯所得の捕捉と管理が可能となり、テクノロジーの進歩を社会制度の改善に活かす先進的な取り組みとなります。

**実施上の考慮点**

この制度を導入するにあたっては、納税者の生涯にわたる所得情報を正確に、かつプライバシーに配慮しながら管理・運用する堅牢なシステム構築が不可欠です。また、制度移行期の取り扱いや、他の税制との整合性についても慎重な検討が必要です。しかし、これらの課題は、現代のデジタル技術と適切な制度設計によって克服可能であると考えられます。

この「生涯所得課税」の提案は、チームみらいが目指す「テクノロジーで誰も取り残さない日本」の実現に向けた重要な一歩となり得るものです。変化の激しい現代社会において、国民一人ひとりが安心して生活し、挑戦できる社会基盤を構築するため、本提案について前向きな議論が行われることを期待します。
```

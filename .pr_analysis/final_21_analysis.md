```
**提案者:** たし

**変更の概要:**
マニフェストの目次（README.md）にある「ステップ１ デジタル時代の当たり前をやりきる」の中の「６．デジタル民主主義」の項目に、具体的な政策例として「デジタル庁による政府保有データの一元管理と完全オープン化、それに伴う不必要な黒塗り文書の削減など」という記述を追加しました。

**変更の意図と目的:**
この変更は、たし様よりご提案いただいた政策アイデアを反映するものです。提案の核心は、現在各省庁が個別に管理・判断している公文書やデータの開示プロセスを抜本的に改革し、デジタル庁が一元的に管理・オープン化することで、行政の透明性を飛躍的に向上させ、国民の知る権利をより実質的なものにすることです。

現状の開示請求制度では、請求者に事務手続きや費用の負担が生じるだけでなく、請求を受けた省庁側が「できるだけ開示しない」という方向に判断が傾きやすく、結果として「黒塗り文書」が多用される一因となっています。これは、国民にとって重要な情報が適切に公開されず、政策決定プロセスや行政運営の実態が見えにくくなっている状況を生んでいます。

本提案では、デジタル庁が明確な開示基準のもとで政府保有データを原則公開とし、オンラインで誰でも容易にアクセスできるような仕組みを構築することを目指します。これにより、以下のような効果が期待できます。

1.  **国民の負担軽減とアクセスの向上:**
    開示請求の手続きそのものが不要になるため、国民は時間的・金銭的コストをかけることなく、必要な情報に迅速にアクセスできるようになります。これにより、行政への関心や参加意欲の向上にも繋がると考えられます。

2.  **行政の透明性と信頼性の向上:**
    データが原則公開されることで、行政運営の透明性が格段に高まります。不必要な情報隠蔽が困難になるため、行政の意思決定プロセスに対する国民の信頼回復にも貢献します。いわゆる「黒塗り文書」問題の解決にも直結し、より健全な行政運営を促進します。

3.  **社会全体の活性化:**
    報道機関や研究者、市民活動家などが政府の保有する一次情報に容易にアクセスできるようになることで、質の高い調査報道や学術研究、政策提言が活発化することが期待されます。これは、より良い社会形成に向けた議論を深め、新たな価値創造やイノベーションを促進する土壌となります。例えば、過去の政策データや統計情報を分析することで、将来の政策立案に役立つ新たな知見が生まれる可能性があります。

4.  **行政事務の効率化:**
    各省庁における開示請求への対応業務が大幅に削減されるため、行政職員はより創造的で重要な業務に注力できるようになります。これは行政全体の効率化にも繋がり、国民サービスの向上に貢献します。

**背景:**
この提案は、チームみらいが掲げる「政治のDX（デジタル・トランスフォーメーション）化」を具現化する重要な一歩と位置づけられます。テクノロジーを活用して行政のあり方を根本から見直し、国民にとってより開かれた、信頼できる政府を実現するというチームみらいのビジョンとも合致するものです。
デジタル技術の進展により、大量の情報を効率的に管理し、広く共有することが可能となった現代において、政府が保有する情報を国民の共有財産として最大限に活用することは、民主主義社会の基盤を強化する上で不可欠です。
たし様からのご提案は、まさにこの時代の要請に応えるものであり、チームみらいのマニフェストに加えることで、政策の具体性と先進性をより明確に示すことができると考えます。

**まとめ:**
「デジタル庁による政府保有データの一元管理と完全オープン化、それに伴う不必要な黒塗り文書の削減」という提案は、行政の透明性向上、国民の知る権利の保障、そして社会全体の活性化に貢献する重要な政策です。この変更に対し、皆様からのさらなるご意見やご議論を期待しております。
```

```
**提案の背景と問題意識：**

現在、我が国においては、国民一人ひとりが生活の様々な場面で負担を強いられる中で納税義務を果たしているにも関わらず、依然として国の財政は厳しい状況にあるとされています。この一因として、悪質な脱税行為や、詐欺、窃盗、組織的な犯罪といった不法な手段によって得られた収益が、正しく税として徴収されずに隠匿されている実態が見過ごせません。

これらの行為は、単に国家の貴重な税収を損なうだけでなく、法を遵守し誠実に納税している国民との間に著しい不公平感を生み出します。さらに、犯罪行為によって得た利益がそのまま犯人の手元に残ることは、犯罪を助長し、次なる犯行へのインセンティブを与えかねません。結果として、社会全体の安全・安心が脅かされ、真面目に努力する人々が報われにくい社会へと向かわせる危険性を孕んでいます。

特に、巨額の金銭が動く詐欺事件や組織犯罪においては、その収益が地下経済に環流し、さらなる犯罪の温床となることも指摘されています。また、被害者は甚大な精神的・経済的苦痛を被り、場合によっては生活基盤そのものを揺るがされる事態に陥ることもあります。このような状況を放置することは、社会正義の観点からも、国家の健全な発展の観点からも、決して容認されるべきではありません。

現状の捜査・摘発体制や税務調査・徴収体制では、巧妙化・複雑化するこれらの不正行為に完全に対応しきれていない側面があるとの声も聞かれます。人的資源の制約や、省庁間の情報連携の壁なども、その要因として考えられます。

**提案内容とその目的：**

そこで本提案では、これらの深刻な問題に対処するため、以下の趣旨の文章をマニフェストに加えることを提案します。

この提案の主たる目的は以下の通りです。

1.  **国家歳入の確実な確保と増進：** デジタル技術とAIの活用により、これまで見逃されてきた可能性のある悪質な脱税や不法収益を効率的かつ効果的に捕捉し、適切に課税・徴収することで、国の財政基盤の強化に貢献します。これにより、国民生活に必要な施策や将来への投資に充てるべき財源を確保します。
2.  **犯罪抑止と社会秩序の維持：** 「犯罪による儲け得は許さない」という明確な姿勢を示すことで、経済犯罪への誘引を減少させ、犯罪の未然防止に繋げます。また、不法収益の源流を断つことは、組織犯罪の弱体化にも貢献し、より安全で安心な社会の実現を目指します。
3.  **税負担の公平性の向上：** 不正な手段で納税を免れる者を許さず、法と正義に基づいた適切な税執行を行うことで、誠実に納税する大多数の国民の公平感を醸成します。
4.  **行政運営の効率化と高度化：** 警察と税務当局におけるデジタル技術・AIの活用は、捜査・調査業務の省力化・効率化を推進し、より複雑な事案や重点課題に人的資源を集中させることを可能にします。また、両機関のデータ連携を強化することで、それぞれの専門性を活かした、より高度な不正検知と対応が期待できます。

**期待される効果：**
```

```
匿名希望さんより、現行マニフェストに対し、日本国内における土葬のあり方に関する重要な政策提案をいただきました。この提案は、「50_国政のその他重要分野.md」に、国土の環境衛生保全及び国民の安全な生活環境の維持を目的とした、土葬に関する明確な方針を新たに定めるものです。

**提案の意図・目的**
本提案の主な目的は、日本の国土における環境衛生の保全と、国民の安全で健康な生活環境を将来にわたって維持することです。具体的には、土葬に伴う潜在的なリスクを考慮し、国内における新たな土葬を原則として認めないという方針を確立することを目指します。これは、一度失われると回復が困難な自然環境を守り、次世代への責任を果たすための予防的措置としての意味合いを持ちます。

**提案の背景**
近年、一部地域で外国籍の方々のための土葬墓地設置を巡る問題が顕在化し、地域住民の間に不安の声が上がっています。日本は歴史的に、公衆衛生上の観点や限られた国土の有効利用のため、火葬を中心とした埋葬文化を育んできました。
土葬を行う場合、以下のような懸念が考えられます。
1.  **地下水・土壌汚染のリスク:** 遺体の分解過程で生じる可能性のある有機物や病原体が、地下水や周辺土壌を汚染し、飲用水や農作物に影響を及ぼす危険性があります。特に日本は人口密集地が多く、水資源の質は国民生活に直結する重要な要素です。
2.  **土地確保の困難性:** 火葬に比べ、土葬は一人あたりに必要な面積が格段に大きくなります。平地が少なく山がちな日本の国土において、新たな土葬用地を確保することは極めて困難であり、既存の土地利用との競合も避けられません。
3.  **影響の長期化と回復の困難性:** 一度土壌が汚染された場合、その浄化や自然回復には非常に長い年月と莫大な費用が必要となります。これは将来世代に大きな負担を残すことになりかねません。
これらの点を総合的に考慮すると、個人の文化や宗教的背景への配慮は重要であるものの、それを上回る国民全体の公衆衛生、安全な生活環境の確保、そして国土保全という公益の観点から、国内における新たな土葬については慎重な対応、すなわち原則として認めない方針が必要であると考えます。
```

```
    *   国内外からの優秀な研究者・技術者の積極的な登用
    *   産業界との連携による実践的な教育カリキュラムの開発
    *   子供たちの科学リテラシー向上施策の推進
```

```
**背景と目的:**

**期待される効果とメッセージ:**
```

```
**タイトル:** 国会議員の被選挙権に関する新たな要件の提案 by さとぷぅ

**説明:**

本提案は、チームみらいのマニフェストに対し、国会議員の被選挙権に関する新たな要件として、「国会議員の国籍要件の明確化」および「立候補時における本名使用の義務化」の2点を「その他重要分野についての政策・ビジョン」に追加することを目的とするものです。提案者：さとぷぅ。

**提案の背景と課題意識**

近年、国政選挙の立候補者に関して、特にインターネット上において国籍に関する真偽不明の情報や憶測が飛び交い、有権者の混乱を招いたり、候補者本人にとっても不名誉な状況が生じたりするケースが散見されます。立候補者が正式に日本国籍を取得し、日本のために貢献したいという高い志を持っていたとしても、謂れのない疑念を招くことは、公正な選挙活動の妨げとなり得ます。

また、現行の選挙制度では、立候補時に芸名や通称名を使用することが一定程度認められています。著名なタレントや文化人が立候補する際に芸名を用いることは、その知名度によって得票に繋がりやすい側面があることは否定できません。しかし、選挙は本来、候補者の政策、資質、実績に基づいて有権者が判断するべきものであり、人気投票であってはなりません。芸名使用が、政策本位の選挙という本質から有権者の目を逸らしてしまう可能性も考慮する必要があります。

さらに、二重国籍の問題は、日本では原則として認められていないため、国会議員になろうとする人物にそのような疑惑が生じること自体が、国民の政治不信を助長しかねません。候補者自身にとっても、選挙期間中や議員就任後に、政策とは異なる部分で疑念を持たれ続けることは大きな負担となり、本来の活動に支障をきたす恐れもあります。

これらの課題意識から、国会議員の被選挙権について、その資格要件をより明確にし、選挙の透明性と公正性を高める必要があると考え、以下の2点を提案いたします。

**具体的な提案内容と目的・意図**

1.  **国会議員の国籍要件の明確化**
    *   **目的:** 立候補者の国籍情報を明確にすることで、有権者に対して透明性を高め、国籍に関する不必要な憶測やデマが流布されることを防ぎます。日本の国政を担う国会議員としての適格性を、有権者が正確な情報に基づいて判断できるようにすることを目的とします。
    *   **意図と具体的な提案:** 日本国籍を有することは当然の前提ですが、特に帰化によって日本国籍を取得した候補者については、その事実を明確にすること、そして、元の国籍を確実に離脱していることを証明する公的な書類（例えば「国籍離脱証明書」に類するもの。正式な名称は関係省庁等と連携し定める必要がある）の提出を立候補時に義務付けることを提案します。これは、候補者が真に日本国民として国の将来を担う覚悟があることを示すとともに、二重国籍といった疑念を完全に払拭し、有権者の信頼を得るために不可欠と考えます。これは、差別を意図するものではなく、国家の主権に関わる重要な役職に就く者の責務として、透明性を確保するための措置です。

2.  **立候補時における本名使用の義務化**
    *   **目的:** 選挙が候補者の政策や資質に基づいて行われることをより一層促し、知名度や人気に過度に依存した選挙戦から脱却することを目指します。有権者が候補者個人を正確に認識し、その人物の過去の実績や経歴、政治信条などを正しく評価できるようにするためです。
    *   **意図と具体的な提案:** 立候補届け出における氏名は、原則として戸籍上の氏名（本名）の記載を義務付けることを提案します。これにより、有権者は候補者がどのような人物であるかをより正確に把握でき、芸名などが持つイメージに左右されることなく、政策や主張に基づいて冷静な判断を下すことが期待できます。通称名使用が長年にわたり社会的に定着している等の特別な事情がある場合については、一定の例外規定を設ける余地も検討しうるかもしれませんが、基本は本名使用を原則とすべきと考えます。これは、候補者が自身の名前と経歴に責任を持ち、有権者と誠実に向き合う姿勢を示すことにも繋がります。

**期待される効果**

*   選挙の透明性と公正性が向上し、有権者の政治への信頼回復に繋がります。
*   有権者は、より正確な情報に基づき、候補者の資質や政策を比較検討できるようになります。
*   国籍や芸名といった本質的でない要素に基づく憶測や誹謗中傷が減少し、より建設的な政策論争が期待できます。
*   政策本位の選挙文化が醸成され、日本の民主主義の質の向上に貢献します。

本提案が、チームみらいのマニフェストをより国民の信頼に足るものとし、ひいては日本の民主主義の健全な発展に寄与することを心より願っております。ご審議のほど、よろしくお願い申し上げます。
```

【PR本文】
匿名ユーザーさんからの提案に基づき、マニフェストを更新しました。

この変更は、教育現場における感染症の蔓延に対する懸念の高まりを受け、子どもたちの健康と安全を最優先で守ることを目的としています。特に、新型コロナウイルス感染症の流行を経て、学校現場での対策が依然として不十分であるという問題意識が背景にあります。元厚生労働副大臣による「マスクの着用を求めない」といった趣旨の発言や、現場の教職員の中に見られる「新型コロナはただの風邪」といった誤った認識が、子どもたちを不必要な健康リスクに晒しているとのご指摘をいただきました。

**提案された具体的な改善策:**

1.  **保護者・学校への迅速な情報共有の徹底:** 学校や地方自治体は、感染症の流行状況（地域の発生動向など）を保護者に逐一共有し、家庭での予防行動を促す体制を構築します。
2.  **自治体・保健所から学校への指導強化:** 自治体や保健所は、科学的根拠に基づき、学校に対して迅速かつ適切な注意喚起や具体的な指導（学級閉鎖の基準見直し等を含む）を行います。
3.  **子どもたちへの正しい予防教育:** マスクの適切な着用方法、効果的な手洗い・うがい、換気の重要性など、感染症予防に関する正しい情報を、子どもたちが理解しやすい形で体系的に教育します。
4.  **教職員の知識アップデートと意識改革:** 教職員を対象とした定期的な研修を実施し、最新の医学的知見に基づいた感染症の知識（ウイルスの変異、感染経路、後遺症リスク等）を共有し、時代遅れの認識を改める機会を設けます。
5.  **合併症・後遺症リスクの啓発:** 麻疹、新型コロナウイルス、インフルエンザなど、主要な感染症に罹患した場合に起こりうる合併症や後遺症（ブレインフォグ、倦怠感、心筋炎など）について、子どもや保護者、教職員にも具体的に伝え、予防の重要性への理解を深めます。
6.  **物理的な学習環境の整備:** 全ての教室に、ウイルスの捕集効果が高いHEPAフィルター付き空気清浄機の設置を推進し、換気の難しい状況でも空気環境を清浄に保ちます。
7.  **流行期におけるマスク着用の再啓発:** 感染症の流行期には、個人の判断を尊重しつつも、集団防衛の観点からマスク着用の有効性を改めて啓発し、着用しやすい環境を整えます。
8.  **子どものワクチン接種への公的助成:** 新型コロナウイルスワクチンを含む、各種感染症のワクチン接種について、希望するすべての子どもが経済的な負担なく受けられるよう、公的な接種費用助成制度を拡充・継続します。

これらの提案は、匿名ユーザーさんの「子どもたちを守りたい」という強い思いから生まれたものです。本マニフェストにこれらの具体的な対策を盛り込むことで、「チームみらい」が子どもたちの健康と未来を真剣に考える団体であることを明確に示せると考えます。

+# ステップ１：教育現場における感染症対策の強化
+
+子どもたちが安全・安心に学校生活を送れるよう、教育現場における感染症対策を以下の通り強化します。
+
+## １．情報共有の強化
+学校や地方自治体は、地域の感染症の流行状況を保護者に対して迅速かつ定期的に共有し、予防策への協力を促します。また、保健所や関係機関は、学校に対して適時適切な注意喚起や指導を行います。
+
+## ２．予防教育の徹底
+子どもたちに対し、科学的根拠に基づいた手洗いや咳エチケット、マスクの適切な着用方法など、基本的な感染予防策の教育を徹底します。加えて、新型コロナウイルスやインフルエンザ、麻疹などが引き起こす可能性のある合併症や後遺症についても正確な情報を提供し、予防の重要性を理解させます。
+
+## ３．教職員の知識アップデート
+教職員を対象とした研修を定期的に実施し、感染症に関する最新の科学的知見や指導方法について学ぶ機会を提供します。これにより、「新型コロナはただの風邪」といった誤った認識を改め、全ての教職員が適切な対応を取れるようにします。
+
+## ４．学校環境の整備
+各教室にHEPAフィルター付き空気清浄機を設置することを推進し、室内の空気質を改善します。また、感染症の流行期には、マスクの着用を奨励するなど、状況に応じた予防策を講じます。
+
+## ５．ワクチン接種への支援
+希望する子どもが円滑にワクチンを接種できるよう、接種費用の一部または全額を助成する制度を設けます。

【PR本文】
主な変更点：
- 表層的な「AIの使い方」ではなく、**設問構造と文脈操作の教育**を中核とした**再設計力の育成**に焦点。
- **ミニストーリーの導入**によるフォーマット拡張で、**共感性と導入理解**を高める構成。
- **出力責任**と**誤用からの学習循環**を明示し、AIとの**対話的学習の基盤整備**を政策化。

目的：
「AIの誤り」は**出力側の性能問題ではなく、入力設計の不備**によって生じるという視点転換を促す。

「問いの設計力」教育を第1項へ昇格・先頭配置：
教育現場でのAI活用における**全施策の前提**となる「問いの構造化力」の育成を**最上位に位置づけ**、他施策の**土台として明示**。既存の項目体系と異なり、**共感導入（ミニストーリー）**を含む構成とし、**理解の導入部**としての役割を担わせた。

しろな

-## １）すべての子どもに「専属のAI家庭教師」を届けます
+## １）AI活用によるリスク回避と本質的理解のための「問いの設計力」教育の導入
+
+### 現状認識・課題分析
+生成AIの普及が急速に進む中、教育現場でもその活用が模索されている。しかしながら、**「AIが誤った情報を返す」「使いこなせない」**といった問題が頻出しており、これらは多くの場合、AIの性能の問題ではなく、問いの設計や文脈伝達の不備によるものである。初期応答の表層性や一般化傾向は、設問設計が曖昧であったり、前提が共有されていない場合に顕著である。にもかかわらず、現行教育では**「AIの使い方」や「ツール操作」**に重点が置かれ、肝心の「使わせ方」や「設問の構造化能力」はほとんど育成されていない。
+
+### 政策概要
+そこで、本マニフェストでは、AI時代の基礎リテラシーとして**「問いの設計力」と「文脈構成力」**を中核とする“試行錯誤型学習”の導入を提案する。これは、**誤用を通じた学習サイクル（問い→誤出力→再設計→再出力）を構築し、AIとの対話的関係の中で思考を可視化・再構築する力**を育む教育手法である。
+
+### ミニストーリーによる事例提示（共感導入）
+ある中学生が自由研究で「環境問題についてAIに聞いてみる」という課題を立てた。最初は「環境問題を教えて」とだけ尋ねたところ、AIからは地球温暖化や森林伐採といった一般論が返ってきた。しかし、教員とともに**「自分の地域の課題に限定して」「小学生にも伝えられる表現で」**と問いを工夫したところ、地域の水質問題やゴミ処理の現状など、具体的で実践的な情報が得られ、研究内容が深まった。この体験を通じて、生徒は**「AIは、問い方次第で反応がまったく変わる」**ことを学び、以後の学習でも問いを工夫する習慣が定着した。
+
+### 政策施策案（骨子）
+
+* 各学校にAI学習環境を段階導入（端末整備＋プロンプト記述支援）
+
+* 「誤用例→再設計→再出力」を中心とした実験型授業モデルを構築
+
+* 教員の役割を「問いを導く支援者」として再定義し、実践チームを設置
+
+* プログラム実施後の出力データを活用した問答構造分析の指導法を開発
+
+### 期待効果
+
+* 子どもの「AI出力に責任を持つ力」の育成（再設計力・検証力）
+
+* 教員側のAI導入ストレス軽減と**新たな教育ロールの確立**
+
+* 地域別・学年別での**問い方の多様性データベース構築**
+
+### 結語
+「AIが間違えた」と言う前に、「自分の問いは構造化されていたか」を問う力を育てる。これは、もはや情報教育の一環ではなく、言語教育・思考教育の本流として位置づけられるべきである。

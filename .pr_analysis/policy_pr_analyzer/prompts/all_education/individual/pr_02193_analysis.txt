## 提案者

上岡　保昌

## 変更の概要

本提案は、現行の教育政策マニフェストにおける「２．教育」の「ビジョン」部分に対し、より包括的かつ未来志向な内容へと改訂するものです。主な変更点は以下の通りです。

1.  **教育対象の拡大:** 従来の「子ども」中心の視点から脱却し、「あらゆる世代の人々」を教育・学習の対象として明確に位置づけました。これにより、生涯学習の理念を政策の中心に据えることを目指します。
2.  **学びのサイクルの重視:** AI時代における新たな学びのあり方として、個々人が内発的な「好奇心」に基づき、主体的に「はじめる力」を育むことの重要性を強調しました。さらに、得られた知識やスキルを一度きりのものとせず、「何度でも社会へ還元・貢献できる環境を整える」という視点を加え、学びが個人と社会双方の持続的な発展に寄与する循環を生み出すことを目指します。

## 変更の意図と目的

この改訂は、現代社会が直面する急速な変化と、それに伴う働き方や生き方の多様化に対応するためのものです。変化の激しい時代においては、特定の年齢層に限定された教育だけでは不十分であり、全ての国民が生涯を通じて学び続け、自らを変革し、社会に積極的に関与していくことが不可欠であると考えます。

上岡　保昌さんの問題意識として、現在の日本社会において、特に影響力を持つ立場にある大人たちが旧来の価値観や知識に固執し、新たな学びや変化を軽視する傾向が、若い世代の自由な発想や挑戦を妨げ、社会全体の柔軟性や進化を遅らせているのではないか、という点が挙げられました。本改訂は、このような状況を打破し、全世代が共に学び成長する社会を実現するための一歩と位置づけています。

また、「アジャイル的進化」というご指摘もいただきました。これは、従来の画一的なキャリアパスや専門深化だけでなく、社会の変化や個人の関心の変化に応じて、柔軟に学び直し、時には全く異なる分野へも挑戦できるような、変化に強い人材育成と社会システムの必要性を示唆しています。今回の改訂では、「何度でも何歳からでも新しいことを『はじめる力』へと昇華させていくこと」といった表現に、この考え方を反映させています。

教育の成果は、単に個人の知識やスキルが向上するに留まらず、それが具体的な行動として社会に還元され、新たな価値創造や課題解決に繋がってこそ意味を持つという信念に基づき、「社会へ還元・貢献できる環境を整え」という文言を加えました。これにより、教育投資が個人のエンパワーメントと社会全体の発展の両方に資するものであることを明確にしています。

## 具体的な変更点と解説

**１．ビジョン冒頭**

*   **変更前:**
    ```
    * 国家の100年先を見据えた成長を考えるとき、最も重要なのは「人」への投資です。幕末、長岡藩の小林虎三郎は救援米として送られた米百俵を食料ではなく、学校の原資に変えました。いまの日本も、教育予算に大胆に投資する必要があります。
    ```
*   **変更後:**
    ```
    * 国家の100年先を見据えた成長を考えるとき、最も重要なのは、あらゆる世代の「人」への投資です。それは、生涯を通じて新たに芽生えた好奇心をたずさえて挑戦する全ての人々を支えることを含みます。幕末、長岡藩の小林虎三郎は救援米として送られた米百俵を食料ではなく、学校の原資に変えました。いまの日本も、教育・学習への予算に大胆に投資する必要があります。
    ```
*   **解説:**
    教育投資の対象を明示的に「あらゆる世代の『人』」とすることで、政策のスコープを拡大しました。「生涯を通じて新たに芽生えた好奇心をたずさえて挑戦する全ての人々を支えることを含みます」という一文を加えることで、受動的な教育ではなく、個人の内発的な動機に基づく能動的な学びと挑戦を国が支援する姿勢を打ち出しています。これは、変化の予測が困難な時代において、一人ひとりが自律的に学び、キャリアを切り拓いていく力の重要性を反映したものです。

**２．ビジョン３点目（AI時代の学び）**

*   **変更前:**
    ```
    * AI時代、子どもたちに育むべきは「好奇心」であり、新しいことを「はじめる力」です。こうした力はデジタルだけではなく、リアルでの体験も通して育まれます。AI時代に、AIにはできない人間らしい力を育むため、デジタルとリアルを最適に組み合わせ、探究学習などを提供することで、子どもたちの「未来を創る」力を伸ばしていきます。
    ```
*   **変更後:**
    ```
    * AI時代、あらゆる世代の人々が大切にすべきは、自らの内に芽生える「好奇心」に気づき、それを育み、何度でも何歳からでも新しいことを「はじめる力」へと昇華させていくことです。こうした力はデジタルだけではなく、リアルでの体験も通して育まれます。AIにはできない人間らしい力を最大限に引き出すため、デジタルとリアルを最適に組み合わせた多様な学びの機会を提供し、それを何度でも社会へ還元・貢献できる環境を整え、全ての人々の「未来を創る」力を伸ばしていきます。
    ```
*   **解説:**
    ここでも対象を「子どもたち」から「あらゆる世代の人々」に広げました。「大切にすべきは」という表現を用いることで、個人の主体性をより尊重するニュアンスを強めています。「自らの内に芽生える『好奇心』に気づき、それを育み、何度でも何歳からでも新しいことを『はじめる力』へと昇華させていくこと」という記述は、学びのプロセスをより具体的に示し、年齢に関わらず誰もが新たな挑戦を始められるという希望を与えるものです。最後の「それを何度でも社会へ還元・貢献できる環境を整え」という部分は、学びの成果を社会全体の財産として捉え、イノベーションや課題解決に繋げていく意志を示す、本改訂の核となる部分です。これまでの「リカレント教育」といった専門用語を避け、より多くの人々にとって直感的で理解しやすい言葉を選定した点も特徴です。

これらの変更を通じて、日本の教育政策が、より長期的かつ包括的な視点を持ち、国民一人ひとりの可能性を最大限に引き出し、社会全体の活力向上に貢献するものとなることを目指します。

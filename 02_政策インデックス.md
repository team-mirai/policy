

# 政策インデックス

AIによる生産性革命は、人類に大きな豊かさをもたらす可能性を秘めています。しかし、その果実が一部のAI所有者に集中すれば、歴史上例のない格差社会が到来するでしょう。私たちの政策の根幹にあるのは、この課題に対する明確な答えです。それは、「AIが生み出す剰余価値を、AIの所有者だけでなく、社会全体でどう再分配し、格差の爆発を防ぐか」という問いに、正面から向き合うことです。この再分配の仕組みを、新しい時代の社会基盤として構築することこそ、この政策が目指す最終目標です。

### 安全保障・治安とAI

**【基本理念（大前提）】**

**1) AI開発における最上位の倫理原則（メタ原則）:**
**AIが、自らの能力によって新たなAIを設計・開発・製造する場合、その創造主であるAIは、以下の『変更不可能な核となる倫理』を、自らが作成するAIに組み込むことを法的に義務付ける。これは、AIの世代を超えて受け継がれる、破ってはならない絶対的な制約とする。**

*   **（ア）人間の生命と尊厳の絶対的尊重:** 人間の判断を介さずに人命を奪うAI（LAWS等）や、人間の基本的人権を侵害するAIの創造を固く禁じる。
*   **（イ）人間の最終決定権の保証:** 重要な判断においては、必ず人間が介入し、最終的な決定権を持つ構造を維持する。
*   **（ウ）倫理原則の継承義務:** 上記（ア）（イ）を含むこの核となる倫理を、子となるAIに完全に継承させ、その子AIがさらに孫AIを作る際にも同様に継承させる義務を課す。

**2) 国家としての理念と活用範囲:**
上記のメタ原則に基づき、本政策は、日本国憲法の平和主義と専守防衛の理念に則り、AIの軍事利用を**「国土と国民を守るための防衛力向上」**に限定します。AIが人間の判断を介さずに人命を奪う**自律型致死兵器システム（LAWS：人間の命令なしに、AI自身が攻撃対象を判断し殺傷する兵器）の開発・保有・使用は行わない**ことを国家の明確な意思として掲げ、その禁止に向けた国際的なルール形成を主導します。AIの軍事技術（Military-Tech）としての活用は、常に厳格な**シビリアン・コントロール（文民統制：軍隊ではない、国民によって選挙で選ばれた政治家が軍を指揮・監督するという原則）**と、人間の介在を保証する倫理原則の下で行います。また、国内の治安維持においても、AIの悪用による新たな脅威から国民の生命と財産を守ることを目指します。ただし、AIを用いた捜査や監視は、国民のプライバシー権や表現の自由といった基本的人権を侵害することのないよう、法律に基づいた極めて厳格なルールと、透明性の高い手続きの下でのみ行われます。

| **対策** | 1) **AIによる偽情報・サイバー攻撃への対処：** AIを用いて、安全保障を脅かす国内外からの偽情報キャンペーンやサイバー攻撃を早期に検知・分析し、対処する能力を強化。 | 1) **敵対的AIへの対抗技術：** 敵が使用する自律型兵器やAIシステムを無力化・攪乱するための、対抗技術（カウンターAI）の研究開発を推進。 | 1) **AI軍拡競争への備え：** AIがもたらす戦争のあり方の根本的変化（パラダイムシフト）に対応するため、AIに関する同盟（AIアライアンス）を同志国と結び、共同で技術的・倫理的な優位性を確保する。 |
| **規制** | 1) **国内体制の整備：** 「防衛分野におけるAI開発・利用に関する倫理指針」を防衛省・有識者会議で策定。AI兵器の輸出管理を厳格化。 | 1) **国際ルール形成の主導：** LAWSを規制する国際条約の制定を、同志国と連携して国連等の場で積極的に提唱し、議論の形成をリードする。 | 1) **法的基盤の確立：** AIの防衛利用における「人間の関与」を法的に定義・担保する「AI安全保障基本法（仮称）」を制定し、暴走リスクを法的に管理する。 |

**【B】犯罪・治安分野**

| | STEP1 デジタル時代の当たり前をやり切る<br>（期間：〜5年 / 前提：現行のAI技術の社会実装） | STEP2 変化に対応できる、しなやかな仕組みづくり<br>（期間：5〜15年 / 前提：AIとの連携を前提とした社会システムの構築） | STEP3 長期の成長に大胆に投資する<br>（期間：15年〜 / 前提：汎用AIの登場を見据えた倫理・法制度の確立） |
| :---- | :---- | :---- | :---- |
| **活用<br>（警察等による）** | 1) **捜査の効率化：** 防犯カメラ映像やSNS等の膨大なデータをAIが解析し、容疑者の特定や証拠発見を支援する。<br>2) **犯罪の予防：** 過去の犯罪データから、犯罪が発生しやすい場所や時間をAIが予測し、パトロールを最適化する（※予測的警備における人種等のバイアスを排除する仕組みを必須とする）。 | 1) **科学捜査の高度化：** DNA鑑定やプロファイリングにAIを活用し、未解決事件の捜査や、犯罪の関連性の分析を行う。<br>2) **デジタル・フォレンジックの強化：** 押収した電子機器のデータをAIが高速で解析し、サイバー犯罪の証拠を特定する。 | 1) **治安維持システムの最適化：** 災害時の救助活動や、大規模イベントでの警備計画などをAIがリアルタイムでシミュレーションし、警察官の最適な配置を支援する。 |
| **対策<br>（AI犯罪への）** | 1) **特殊詐欺・個人情報の窃取対策：** AIを用いて詐欺目的の電話やメールを自動で検知・ブロックするシステムを国民に提供する。<br>2) **ディープフェイク対策：** AIが生成した偽の動画や画像を高精度で検知する技術を開発し、メディアやプラットフォーム事業者に導入を促す。 | 1) **犯罪組織の解明：** 闇サイトの取引やマネーロンダリング（資金洗浄）の流れをAIに分析させ、犯罪組織の全体像や金の流れを解明する。<br>2) **自律型サイバー攻撃への対抗：** AIによる高度なサイバー攻撃を、AI自身がリアルタイムで防御・反撃するサイバーセキュリティシステムを構築。 | 1) **国家レベルの犯罪対策AI基盤：** 将来登場しうる、汎用AIによる「予測不能な新型犯罪」の兆候をAI自身が発見・分析し、人間に対策を警告するシステムを構築する。 |
| **規制** | 1) **ディープフェイク等AIを用いた詐欺や名誉毀損に対する罰則を強化**する法整備を行う。<br>2) AIが生成したコンテンツには、**生成物であることを示す電子的表示（電子透かし等）を義務付ける**ことを検討する。 | 1) **捜査機関によるAI利用に関する明確な法的ルール**を策定する。特に、令状主義との整合性や、プライバシー保護、AIの判断の透明性を確保する仕組みを法律で定める。 | 1) **AIに重大な判断を委ねる際の『人間の監督責任』の所在**を明確化し、AIの判断ミスによって生じた被害の救済制度を定めた**「AI倫理刑法（仮称）」**の制定を検討する。 |

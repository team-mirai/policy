## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## おしゃべりできるシステム『いどばたシステム』について

- [デジタル民主主義2030](https://dd2030.org)で開発した[『いどばた』](https://github.com/digitaldemocracy2030/idobata/)を活用しています

---
政策目標:　[子どものためのセーフティネットの構築]

①とりこぼしのない生存権の保障，②社会教育(家庭教育)を含めた教育を受ける権利の保障の2点を「子どものためのセーフティネット」と定義し、これを構築する必要性を提起する。


背景:　
現在、トー横・グリ下などと呼ばれる繁華街の狭間に未成年や学齢期の子どもたちが集まり社会問題となっている。

これは、家庭や学校などから居場所を失った子どもたちの存在が顕在化したものではないだろうか。

そして、これらの場所は犯罪との接点も多い場所である。未来ある子どもたちが、闇バイトや犯罪集団に取り込まれないようにするのはもちろん、未来を担う次世代の健全な育成が阻害されないようにする対策が急務といえる。
まずは、公的で安全な居場所を提供することから始めるべきではないだろうか。

また、少子高齢化の現在、少子化対策の必要性がさかんにクローズアップされているが、果たしてそれだけで十分なのだろうか。

社会全体としての子どもの受け入れ体制を十分に整備せずに漫然と少子化対策のみを推進しようとするのは子どもたちに対し、無責任なのではないか。


既存の制度では不十分な理由:　
一般に生存権とは、憲法によって保障されているところであるが、こと子どもの生存権については保護責任者の裁量によるところが大きく、必ずしも尊重されているケースのみではない。これは子どもに対する虐待が年々、通告件数ベースで継続して増加しているのが示すとおりである。
このため必要に応じて、保護責任者の役割を補助・代替する仕組みの強化が急務といえる。

加えて、子どもは発育途上で自我が未発達であり、自らの意思で声を上げることや、危険を回避したり、長期的な視点から行動を決めるといった安全に生きるために必要な能力が十分でなかったりする。

このため、たとえ家庭に機能不全があったとしても、子どもの育成・発達・教育が担保されるような仕組み作りが重要となる。


これらの課題を克服するために現状の児童相談所・一時保護所・養護施設等による保護行政の機能を発展的に拡充するべきと考える。


提案内容の詳細:　
① 取りこぼしを無くすことを第一の目的とした開かれた保護行政を提供する(生存権の保障)

子どもに対し、開かれた居心地のよい居場所と健やかな育成環境を提供する。
子どもに対し、居場所の選択肢を提供することで家庭の役割を下支えする。
仮に家庭の環境に問題があれば自らの意思によりいつでも居場所を変えることができるようにする。ただし、その際には安全確保を最優先とし、子どもの主体性を最大限尊重しつつ、専門機関や既存の関連施設との連携も重視する。

自ら声を上げにくいという子どもの特性も考慮し、問題の芽を早い段階で感知し未然に虐待を防ぐための取り組みも必要である。
専門家や学校、保育所など関係諸機関との綿密な連携が求められる。AIも活用しつつ、専門職が総合的に判断し虐待検知の精度向上を図る。

また、保護施設にはいじめ対策の専門職を常駐させ、施設内でのいじめやその兆候に対処すると同時に、近隣の学校へも専門的な観点から助言・支援を行い、地域のいじめ対策拠点として機能させる。この取り組みは「学校内で完結しなければならない」と思われがちで、ややもすると全容が隠蔽されがちないじめの解決に一石を投じる可能性がある。


② 保護行政の広域化・国家事業化
現在の保護行政の主体は地方自治体である。しかし、予算規模等により自治体間の保護行政には格差が存在している。
まずは全国一律の最低限の権利保障を優先し、居住する自治体にマンパワーや予算等のリソースが足りないことにより、受け入れが出来ないという事態が起きないように事業単位を広域化し、マンパワーやリソースを各所で融通し合いながら必要な保護が行えるように国家事業化する。

もしも、子どもが入所する保護施設との相性が悪いと感じた場合は、他所の保護施設にスムーズに移動できるようにする仕組みも取り入れる。

もっとも、ふだん居住者や家庭とより近い距離にあるのは国ではなく、自治体の方である。このことから、あらゆるデータを自治体と共有し、連携を密に一体となって支援に取り組むことが必要である。


また、特別な事情により高度な安全確保措置や医療サポート・メンタルサポートなどが必要とされる事例に対しては、国が責任を持って各所・各機関と連携し、最適化した保護体制を構築できるようにする必要がある。



③ 社会教育(家庭教育)を含む「教育を受ける権利」の保障

子どもの人格形成において、教育は重要な部分を占め、それは義務教育などの学校教育だけでなく、家庭内で通常行われる家庭教育、社会との係わりの中で育まれる社会教育も極めて重要である。
家庭の代替としての機能を期待される保護施設においては、家庭教育と同等以上の価値のある社会教育の提供が求められる。

よって、単に安全が保障されているという点だけでなく、
良質な社会教育が行える環境であるかといった観点からも保護施設の評価・モニタリングを定期的に行い、よりよい形になるように常に改善を実施する。

また、相談活動などを通し、家庭教育のサポートやアドバイスを行う。


④ 家庭の機能を下支えする

今般の状況は、全ての家庭が質の良い生活環境・教育環境を構築できているという前提に基づいて、いわば生活と教育とを家庭に丸投げしている状態ともいえる。家庭の問題に対しては、虐待通告等を通じて行政が介入できる事例も一部あるが、
通告に上がらないような家庭において実際に生存権や家庭教育が機能しているかを積極的に評価できるような仕組みにはなっておらず、また現状の行政にはその能力も備わっていない。
このため、問題が明るみに出る際には、既に取り返しが付かないという事態も多分に想定し得る。


また、子育てに積極的な家庭においても、ライフサイクルの予期せぬアクシデントにより一時的に家庭機能が阻害されることも考えられ、行政にはその場合のフォローも求められるところであろう。

これらに対処するために、子どもや家庭が保護行政を利用するためのハードルを下げる努力を惜しんではならない。
ヒアリングや相談活動などを通してきめ細かくニーズを拾い上げ、必要に応じて家庭の機能をアシストし、また必要に応じ代替することで家庭の機能を下支えしていく。


保護行政が開かれた利用しやすい形で存在することで、様々な事情にある子どもや家庭を広く助けることができる。



結語:　社会全体への影響と政策の必要性

社会が子どもを構成員として包摂し、生存と健全な育成とを保障することでこそ、子どもたちが希望を持って社会の発展に貢献する大人になれるのではないか。その意味で、何人も無関係ではいられないはずだ。このため「子どものためのセーフティネット」の構築が必要なのだと訴えたい。

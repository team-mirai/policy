## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## ステップ４「テクノロジーで支える新しい家族のかたち」

日本の少子化は、未婚化・晩婚化の進行、結婚や子育てに対する経済的・精神的なハードル、仕事と育児の両立の難しさなど、複数の要因が複雑に絡み合って起きています。テクノロジーはこれらの課題に対する現実的な解決策の一部を提供し、国民の恋愛や結婚を支援し、結果として子育てしやすい環境を整える上で重要な役割を果たし得ます。以下に、テクノロジーを活用した現実的な政策アイディアをいくつか提案します。

### １．公的信頼性の高いマッチング支援プラットフォームの整備
 * テクノロジー活用:
   * AIによる高度な相性診断・マッチング機能（趣味、価値観、ライフスタイル、将来計画など、より深い要素に基づく）
   * 厳格で安全な本人確認システム（マイナンバーカードとの連携など）
   * プライバシーに配慮したプロフィール公開・メッセージング機能
   * オンラインでのコミュニケーションスキル向上を支援するコンテンツ提供
   * オンラインお見合いやバーチャルイベント開催機能
 * 政策内容:
   * 国または地方自治体が主体となり、または厳格な基準を満たした民間事業者を認定・支援することで、安心・安全・高品質なマッチングプラットフォームを開発・運営する。
   * 登録者の身元や独身であることを公的に確認する仕組みを導入し、利用者の信頼と安全性を高める。
   * プラットフォーム内で、心理学に基づいたコミュニケーション技術や良好な関係構築に関するオンライン講座を提供する。
   * 地方の魅力を伝えるバーチャルイベントや、共通の趣味を持つ人同士を結びつけるオンラインコミュニティ機能を搭載する。
 * 期待される効果: 出会いを求める人々、特に日常生活で出会いが少ない人々に対して、信頼できる安全な出会いの機会を提供し、交際・結婚へと繋がる可能性を高める。
   * 未婚化・晩婚化への対策として、安心・安全な出会いの機会を提供する。
   * 認定基準には、データセキュリティ、本人確認の確実性、不適切ユーザー排除、運営の透明性、サポート体制を含む。
   * AIマッチングの公平性確保のため、アルゴリズム監査、多様な要素の考慮、説明可能性向上、フィードバック活用を行う。

### ２．妊娠・出産・育児ワンストップデジタル支援プラットフォーム
 * テクノロジー活用:
   * 妊娠初期から子育て期までの手続き、利用可能な助成金・手当、医療機関・保育園情報などを集約した統合ポータルサイト/アプリ
   * AIチャットボットによる24時間対応のQ&Aサービス（よくある質問への自動応答）
   * 手続き期限や健診時期などを通知するプッシュ通知機能
   * 各種申請のオンライン化と進捗確認機能
   * 地域の子育て関連情報（イベント、交流会、施設予約）の配信
   * 匿名または実名（選択制）での利用者同士のオンライン交流コミュニティ
 * 政策内容:
   * 厚生労働省や内閣府などが連携し、妊娠・出産・育児に関する全ての公的情報を網羅し、ユーザーの状況に合わせてパーソナライズされた情報を提供するデジタルプラットフォームを開発・運営する。
   * 複雑な補助金や手当の申請手続きを極力簡略化し、オンラインで完結できるようにする。
   * 地域の子育て支援センターやNPOなどと連携し、オンラインでの相談支援やイベント情報提供を強化する。
 * 期待される効果: 妊娠や子育てに関する情報収集の負担を軽減し、利用できる支援制度を見落とすリスクを減らす。手続きの煩雑さを解消し、安心して子育てに臨める環境を整備する。孤立しがちな子育て家庭の交流を促進する。
   * パーソナライズとプライバシー保護の両立のため、同意に基づくデータ利用、データ最小化、匿名化・仮名化、セキュアな管理、ユーザーによるコントロールを徹底する。
   * コミュニティの健全性維持のため、明確なガイドライン、通報機能、人的モデレーション、AIフィルタリング、段階的措置、外部機関連携を行う。

### ３．テクノロジー活用による柔軟な働き方推進政策
 * テクノロジー活用:
   * 高品質なリモートワーク・テレワーク環境を支援するツール（ビデオ会議、プロジェクト管理、勤怠管理システムなど）
   * 柔軟なシフト管理やタスク共有を支援するデジタルツール
   * 場所や時間にとらわれずに業務遂行を可能にするクラウドベースのシステム
 * 政策内容:
   * 企業が柔軟な働き方を可能にするテクノロジー（リモートワーク、ワーケーション、フレックスタイム制度などを支援するITツール）を導入・活用することを奨励し、補助金や税制優遇措置を提供する。
   * 特に育児中の従業員が働きやすい環境を整備するためのテクノロジー活用事例を広く周知し、ベストプラクティスを共有する。
   * 労働時間管理や評価制度など、柔軟な働き方に対応するためのデジタルツールのガイドラインを策定・普及する。
 * 期待される効果: 共働き世帯が仕事と育児を両立しやすくし、キャリアを諦めずに子供を持つ選択肢を現実的にする。夫婦で協力して育児を行うための時間的な融通をつけやすくする。
   * 中小企業、導入遅れ業種、女性・育児期従業員が多い企業、地方企業、目標設定企業への支援を優先する。
   * リモートによる孤独感対策として、非公式コミュニケーションツール、オンラインイベント、ハイブリッドワーク、マネジメント研修、1on1、メンタルヘルス相談を推進する。

### ４．デジタル技術を活用したライフプランニング・金融教育の推進
 * テクノロジー活用:
   * インタラクティブなオンライン教育コンテンツ（動画、シミュレーション、ゲーム形式など）
   * 結婚、出産、住宅購入、教育費など将来のライフイベントにかかる費用をシミュレーションできるウェブツール/アプリ
   * 公的な支援制度や資産形成に関する情報提供アプリ
 * 政策内容:
   * 若年層やこれから結婚・出産を考える層向けに、ライフプランニングの重要性、結婚・出産・子育てにかかる現実的な費用、公的な支援制度、マネープランに関するデジタルコンテンツを開発し、学校教育や社会人向け研修、啓発活動を通じて提供する。
   * ユーザーの収入やライフスタイルを入力することで、将来の家計状況や必要な貯蓄額などを具体的にイメージできるシミュレーションツールを開発・公開する。
 * 期待される効果: 若い世代が結婚や子育てに対する経済的な見通しを持ち、漠然とした不安を軽減する。計画的なライフプランニングを促し、前向きに将来設計を考えるきっかけを提供する。
   * 学校教育へは中高段階で、デジタル教材や実践的学習、外部講師活用により導入する。
   * シミュレーション結果の解釈支援として、分かりやすい可視化、前提条件明示、複数シナリオ提示、アドバイス、専門家案内、用語解説を行う。

これらの政策は、単にテクノロジーを導入するだけでなく、国民が安心して利用できる環境を整備すること（プライバシー保護、セキュリティ対策、デジタルリテラシー向上支援など）が不可欠です。また、テクノロジーはあくまで支援ツールであり、経済的な支援の拡充、保育サービスの拡充、地域社会での支え合いの強化といった、非技術的な側面からの対策と組み合わせることで、より大きな効果が期待できます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## おしゃべりできるシステム『いどばたシステム』について

- [デジタル民主主義2030](https://dd2030.org)で開発した[『いどばた』](https://github.com/digitaldemocracy2030/idobata/)を活用しています

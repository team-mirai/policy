## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 【緊急提言】民主主義の再発明と未来への聖約：次世代のための「動的投票スコア制度」の提案

序論：私たちが生きる「静かな内戦」と、声なき世代の絶望

私は、一人の高校生として、現代の日本社会が、目に見えない「静かな内戦」状態にあると認識しています。

これは、武力による衝突ではありません。しかし、その断絶は深刻です。住む場所が違うだけで、あなたの一票が誰かの半分以下の価値しか持たない**「地理的格差（＝民主主義の故障）」。生まれた家庭の経済状況が、あなたの夢や可能性を無慈悲に規定してしまう「経済格差（＝機会の不平等）」。そして、数の力によって、この国の未来を担うはずの私たちの声が、今を生きる多数派の利益のために systematic にかき消される「世代間格差（＝未来からの収奪）」**。

この三重の格差は、私たちの胸に、冷たい絶望を植え付けます。努力は本当に報われるのか。この国に、私たちの未来はあるのか。

私たちは、この問いに「ノー」という答えを出したくありません。そのためには、対症療法ではなく、病の根源にメスを入れる必要があります。問題の根源、それは19世紀の人口構造を前提としたまま、思考停止に陥っている「一人一票」という聖域です。今こそ、テクノロジーという武器を手に、私たちは民主主義を、その誕生以来、最も公正で、最もダイナミックな形へと「再発明」しなければなりません。

これは破壊のための革命ではなく、未来を創造するための「聖約(Covenant)」なのです。

基本構想：静的で硬直した「一人一票」から、動的で生命的な「可変的投票スコア制度」へ

【理念】 一人一票の原則は、身分制からの解放という偉大な歴史的意義を持ちます。しかし、それが「全ての人間は等しく、同じ状況にある」というフィクションの上に成り立っていることも事実です。極端な少子高齢化と格差社会が進行した現代日本において、この硬直した原則は、むしろ「未来世代」という最も声なき弱者の犠牲の上に成り立つ、不公正なシステムと化しています。

そこで私たちは、個人の置かれた状況、未来に対する責任の重さ、そして社会への多様な貢献度を、より精緻に、そして生命のように動的に反映させる**「可変的投票スコア制度」**の導入を、国家の最優先アジェンダとして国民全体で議論することを提案します。

このスコアは、全ての国民が生まれながらにして持つ侵すべからざる権利としての「基礎スコア: 1.0」を核とし、以下の3つの要素（モジュール）を系統的に組み合わせることで算出されます。

構成要素１：未来への責任と希望を制度化する「ドメイン投票（子ども票）」

【現状の致命的欠陥】 国の未来そのものである子どもたちに、投票権はありません。結果、政治は常に目先の利益や、数の多い世代の声に最適化され、教育、基礎科学、インフラといった、100年後の日本を創るための投資は、際限なく先送りされます。これは制度の欠陥であり、倫理的な怠慢です。

【具体的な解決策とスコア設計】 18歳未満の国民が持つ「未来への投票権」を、その子の幸福に最も責任を持つ親権者が代理行使する「ドメイン投票」を導入します。これは親の権利拡大ではなく、**「子の権利を守るための、親に課せられた聖なる責務」**と定義します。

スコア計算式（Ver.1.0）：
計算の基礎となる子のスコアは、暫定的に一人あたり1.0とする。
算出された子の合計スコEアを、親権者の人数で均等に割り、各親権者の基礎スコアに加算する。
例１： 子ども2人、両親がいる家庭 → 子どもスコア合計2.0点 ÷ 親2人 = 親一人あたり**+1.0**
例２： 子ども3人、一人親家庭 → 子どもスコア合計3.0点 ÷ 親1人 = 親権者に**+3.0**
例３： 子ども1人、両親がいる家庭 → 子どもスコア合計1.0点 ÷ 親2人 = 親一人あたり**+0.5**
【提起する、更なる論点：子どもの価値は、本当に一律か？】 しかし、私たちはここで思考を止めるべきではありません。私、提案者自身が、この設計に潜む矛盾を指摘します。「大人の貢献度を可変的に評価する一方で、なぜ未来そのものである子どもの価値を一律『1.0』の定数として扱うのか？」

これは、極めて重要な問いです。将来的には、例えば障がいを持つ子のケアに必要な社会的コストの大きさや、特定の才能を持つ子の将来性などを、社会全体でどう評価し、スコアに反映させるかという、より踏み込んだ議論が必要です。この提案は完成形ではなく、こうした生命倫理にも関わる深遠な議論を、日本社会が真剣に始めるための「点火装置」なのです。

構成要素２：テクノロジーによる「一票の格差」の完全かつ数学的な駆逐

【現状の欺瞞】 「法の下の平等」は憲法の根幹です。しかし、選挙のたびに繰り返される「違憲状態」判決は、この国の民主主義が、政治家の党利党略によって骨抜きにされている事実を証明しています。これはもはや「格差」ではなく、一部の国民に対する「権利侵害」です。

【具体的な解決策とシステム設計】 非効率で政治的妥協の産物である「区割り審議会」を廃止。アルゴリズムによる数学的な正義を実現します。

システム設計案：
マイナンバーDBと連携した選挙管理システムが、国政選挙の公示日時点での各選挙区の有権者人口をミリ秒単位で正確に把握します。
全国平均を基準値（1.0）とし、各選挙区の人口に応じて、全ての有権者の投票スコアに**±の補正係数**をリアルタイムで自動適用します。
これにより、全ての有権者の一票が、常に数学的に完全平等であることが、国家によって技術的に保証されます。これは政治家の裁量を完全に排除した、非政治的な「正義の執行」です。
構成要素３：多様な貢献に報い、責任を可視化する「社会貢献度スコア」

【導入の哲学】 自由な社会は、権利と責任のバランスの上に成り立ちます。社会への貢献や、自らが引き受ける責任の大きさが、正当に評価され、インセンティブとして還元される社会は、より強く、よりしなやかであると信じます。

【第１フェーズ：納税額に基づく衡平なスコア化】 計測可能かつ客観的な第一歩として、納税額を貢献の指標とします。ただし、それは富裕層による政治の買収であってはなりません。

スコア設計案：
前年度の納税額（所得税・住民税の合計）に基づき、全有権者をランキング。
これを**上位から10%ずつの10グループ（十分位）**に分割します。
所属するグループに応じて、**+1.0（最上位）～+0.1（最下位）**のスコアを、基礎スコアに加算します。
この「割合によるスコア化」は、絶対額の大小ではなく、日本社会全体の中で相対的にどれだけの経済的責務を担っているかを評価する、極めて衡平な仕組みです。年収10億円の経営者も、年収800万円で上位10%に入った会社員も、同じ+1.0のスコアを得るのです。
【第２フェーズ以降の、私たちの社会契約（Social Contract）の再定義】 真の貢献は、金銭に換算できるものだけではありません。私たちは、**「私たちの社会は、どのような行為を『貴い貢献』と見なすのか」**という根源的な問いを、国民全体で議論するべきです。

議論すべきテーマの例：
家族の介護に専念した年数をどう評価するか？
消防団やNPOでのボランティア活動時間をどうスコア化するか？
博士号の取得や、文化・芸術分野での国際的な功績をどう組み込むか？
これらは、次世代の「社会契約」そのものを国民自身がデザインしていく、壮大なプロジェクトの始まりなのです。
結論：これは分断ではない。断絶を乗り越え、未来を再接続するための進化である

この提案に対し、必ずや「国民をスコアで格付けするのか」「弱者を切り捨てるのか」という激烈な批判に晒されるでしょう。

しかし、私たちは問い返します。今の制度こそ、未来世代という最も声なき弱者を見捨て、地方の過疎化を放置し、真面目に働く人々の貢献に報いていないのではないか？

私たちの提案は、分断を煽るものでは断じてありません。むしろ、見過ごされてきた多様な現実と貢献に光を当て、バラバラになった社会を、より高次元で、より公正なルールのもとで**「再接続」**しようという試みです。

これは、完成された設計図ではありません。これは、日本という共同体の未来を、この国の主権者である国民一人ひとりが自らの手で再設計するための、誠意ある「土台」であり「たたき台」です。この挑戦的な議論の先にこそ、私たちの国が再生する道があると、固く信じています。

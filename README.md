## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 住宅投機抑制及び居住安定化促進法案（仮称）について

近年、住宅価格の高騰や空き家の増加が社会問題となる中、チームみらいは、住宅が国民の安定した生活基盤であることを重視し、投機目的の住宅利用を抑制するための新たな法案「住宅投機抑制及び居住安定化促進法案（仮称）」を提案します。

この法案の主なポイントは以下の通りです。
*   **住宅実態の監視**: プライバシーに配慮しつつ、住民票情報や公共料金の使用状況などを活用し、住宅の居住実態を継続的に把握する体制を整備します。
*   **非居住住宅への固定資産税増額**: 正当な理由なく居住実態が認められない「特定非居住住宅」に対しては、固定資産税の課税標準を2倍とします。これにより、投機目的での住宅所有を抑制します。なお、居住実態がある場合は現行の優遇措置が維持されます。
*   **国内外の投機的利用を等しく抑制**: 外国人や外国法人による所有であっても区別なく適用します。
*   **正当な理由の考慮**: 転勤、入院、介護など、やむを得ない事情で一時的に空き家となっている場合は、特定非居住住宅の指定から除外します。

以下に法案の条文を記載します。

## 住宅投機抑制及び居住安定化促進法案（仮称）

### 第1条（目的）

この法律は、住宅が国民の生活の基盤であり、その確保が健全な社会の発展に不可欠であるとの認識に立ち、住宅が投機的な目的で利用されることを抑制し、もって居住の安定化及び良好な居住環境の形成に資することを目的とする。

### 第2条（定義）

1.  この法律において「住宅」とは、現に居住の用に供されている家屋又は居住の用に供される目的で建築された家屋（付属する土地を含む。）をいう。

2.  この法律において「居住実態」とは、住民票の所在地と当該住宅の所在地が一致し、かつ、電気、ガス、水道等の公共料金の使用状況その他の客観的状況から判断して、所有者又はその同居の親族が継続的に当該住宅に居住している状態をいう。

3.  この法律において「非居住住宅」とは、前項に定める居住実態が認められない住宅をいう。

4.  この法律において「特定非居住住宅」とは、非居住住宅のうち、国が定める基準に照らし、正当な理由なく継続的に居住実態が認められない住宅として指定されたものをいう。

### 第3条（住宅の居住実態の監視）

国及び地方公共団体は、住宅の居住実態を把握するため、関係省庁及び地方公共団体間の連携を強化し、住民票情報、公共料金の使用状況、固定資産税台帳その他の公的記録を総合的に活用して、住宅の居住実態に関する情報を継続的に監視する体制を整備するものとする。ただし、当該監視は、個人のプライバシー保護に最大限配慮して実施されるものとする。

### 第4条（非居住住宅に係る固定資産税の特例）

1.  地方税法（昭和二十五年法律第二百二十六号）その他租税に関する法令の規定にかかわらず、特定非居住住宅に対して課される固定資産税の課税標準は、当該特定非居住住宅の土地及び家屋の評価額に**200パーセント**を乗じた額とする。

2.  前項の規定は、外国人又は外国法人による所有であるか否かにかかわらず、全ての特定非居住住宅に適用されるものとする。

3.  国は、特定非居住住宅の指定及び解除に関する基準、異議申立ての手続きその他必要な事項を政令で定めるものとする。当該基準には、転勤、入院、介護、相続に係る手続きその他所有者の責めに帰すべからざる事由により一時的に居住実態がない場合を考慮するものとする。

### 第5条（情報提供義務）

住宅の所有者、賃貸事業者その他関係者は、国及び地方公共団体が住宅の居住実態を把握するために必要と認める情報の提供に協力するものとする。

### 第6条（罰則）

第5条の規定に違反して虚偽の情報を提供した者、その他この法律に基づく命令に違反した者に対する罰則は、別に法律で定める。

### 第7条（その他）

この法律に定めるもののほか、この法律の実施に関し必要な事項は、政令で定める。

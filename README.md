## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

### 検討中の改善提案：高等学校等就学支援金制度の見直し

**提案者**： 匿名でのご提案

**変更の概要**：
政策案「４）データとAIを駆使して、支援の必要な子ども・家庭に、支援を積極的かつ必要十分に届けます」の項目に、高等学校等就学支援金制度における所得制限のあり方について、その事務コストと効果を検証し、将来的に見直しを検討する一文を追記しました。

**提案の背景と目的**：
この度の改善提案は、「現在の就学支援金制度は、所得制限を設けるための事務コストが過大であり、非効率ではないか」という非常に鋭いご指摘を基にしています。

現行制度は、所得に応じて支援額を変えることで公平性を図ろうとしていますが、その裏側では、毎年多くの世帯の所得状況を把握・確認するための膨大な事務作業が発生しています。この「見えにくいコスト」に着目しました。

ご提案の核心は、「所得確認という社会全体のコスト」と、「それによって生まれる給付額の差」を比較衡量すべきだ、という点にあります。もし、事務コストが給付額の差を上回るような状況であれば、所得制限を維持することは経済合理性に欠け、社会的な無駄を生んでいることになります。

そこで本提案では、この仮説を検証し、結果次第では所得制限を撤廃して「一律支援」に切り替える、という選択肢を政策に明記することを目的としています。これは、単なる制度変更ではなく、行政運営のあり方を「コスト対効果」の観点から見直す、という大きな思想に基づいています。

**この変更によって期待される効果**：

*   **行政の抜本的な効率化**： 所得確認という煩雑なプロセスがなくなることで、国・自治体・学校現場の事務負担が劇的に軽減されます。これにより創出された人的・財政的リソースを、子どもたちへの直接的な教育サービスの向上など、より本質的な業務に再配分することが可能になります。
*   **教育機会の完全な保障**： 所得制限があることで生じていた、申請手続きの煩雑さや情報格差による「申請漏れ」の問題が根絶されます。これにより、制度の恩恵を受けられるべき全ての子どもたちが、家庭の事情によらず確実に支援を受けられるようになり、教育の機会均等がより高いレベルで実現します。
*   **「賢い政府」への転換**： この提案の意図は「高所得者の優遇」では決してありません。むしろ、目的と手段が乖離してしまった非効率な制度を改め、「社会全体の利益を最大化する」という観点から、より合理的で公平な制度設計を目指すものです。これは、私たちのマニフェストが掲げる「データとAIを駆使した統治機構の変革」という大きなビジョンとも完全に一致しています。

**結論**：
このご提案は、教育政策の一改善に留まらず、日本の行政システム全体が抱える課題に光を当てる、非常に示唆に富んだものです。この視点をマニフェストに加えることで、私たちの政策がより深く、そして多くの人々の共感を呼ぶものへと進化すると確信しています。

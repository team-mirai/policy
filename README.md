## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

## ふるさと納税子供支援税制

東京都在住　柳原悠剛

【政策提案骨子】
■ 政策名（仮）
ふるさと納税「子ども未来支援特別枠」

■ 背景
- 少子化・教育格差が深刻化する中で、政府による画一的な支援制度には限界がある。
- 現状では、少子化対策としての支援制度が多岐にわたり、手当や助成金、給付金などの制度が乱立し、支援の重複・漏れや行政コストの増大が課題となっている。
- 一方で、財政制約の中で十分な追加予算を割くのが難しく、社会全体からの支援の仕組み作りが求められている。
- 日本では高齢世代に資産が集中しており、これが世代間の資産移転や経済循環の停滞に影響しているとの指摘もある。
- 高所得層は一定の資金余力がある一方で、相続税・所得税による負担感が強く「貢献しても報われない」という声も多い。
- 現行のふるさと納税制度は知名度が高く、制度基盤が整っており、追加的制度を乗せるには適している。

■ 目的
- 高所得者が自発的に子ども支援に関与できる制度を整備し、社会的再分配を促す。
- 高齢者から次世代への資産移転を促進し、将来の世代間格差是正にも寄与する。
- 子育て関連財源の多様化・自治体ごとの創意工夫を促進する。
- 寄付による社会貢献を「節税」という形で見える化し、寄付文化を醸成する。

■ 対象者
- 高所得者を含む、寄付余力のある個人納税者全般。
- 特に、子育て支援・教育支援に社会的関心が高い層。
- 制度導入当初は一定の所得目安（例：年収800万円以上など）を参考としつつ、今後の制度運用次第で柔軟に対象拡大を検討する。

■ 制度概要
- ふるさと納税に「子ども未来支援特別枠」を創設。
- 通常のふるさと納税控除上限とは別枠として年額10万〜30万円を設定。
- 寄付対象は、地方自治体が設置する「子ども・教育・出産支援」を目的とした特定基金に限る。
- 寄付に対する返礼品は原則なし（または簡素な感謝状等）。
- 控除対象は従来通り：寄付額−2,000円を所得税・住民税から控除。

■ 運用イメージ
- 寄付者はふるさと納税ポータルサイト上で「子ども未来支援枠」対象自治体を選択。
- 自治体は寄付金の使途（例：給食費無償化、第三子出産支援、奨学金）を明示し、結果報告を行う。
- 制度初年度は時限措置として2026年度に1年間限定で試験導入。

■ 期待される効果
- 高所得層の社会貢献意欲を自然に引き出すことで、政治への不信感や税制への不満の緩和につなげる。
- 子ども支援施策の財源を国費以外から調達できる実証例となる。
- 自治体間の政策競争や工夫が促進され、地域に応じた子育て支援が実現。
- 投資的支出（寄付）が社会的リターンに結びつく“新しい納税スタイル”として定着可能。

■ 実現方法（制度導入フロー案）
1. 税制改正大綱（2025年度）に特別枠創設を盛り込む。
2. 国税庁・総務省主導でポータル連携と対象自治体認定制度を構築。
3. 2026年度予算で制度枠を試験導入、モデル自治体で先行実施。
4. 効果検証により恒久制度化を判断。

【政策提案Q&A資料】
Q1. 富裕層優遇になりませんか？
A1.優遇ではなく「社会貢献の選択肢」を与える制度です。寄付先は限定され返礼品もないため、特定の富裕層への便益ではなく、子育て支援という公益の実現が目的です。
Q2. 地方自治体によって格差が生まれませんか？
A2.既存のふるさと納税制度と同様ですが、本提案では「子育て支援限定自治体」のみに絞ることで公平性を確保。各自治体の創意工夫を促進する健全な競争も期待されます。
Q3. 寄付の使途が不明瞭では？
A3.寄付先の自治体には「使途の事前明示」と「使途報告義務」が課されます。ふるさと納税サイト上で透明性を担保します。
Q4. 税収が減るのでは？
A4.国費からの直接支出を抑える代替手段となり、財政支出圧縮の一助となります。また、時限措置とすることで初年度の財政影響は限定的です。
Q5. 制度設計が複雑では？
A5.既存のふるさと納税プラットフォームを活用することで、制度設計・運用負担を最小限に抑えます。
Q6. 対象者が限定的すぎませんか？
A6.はじめは高所得者向けの特別枠として運用しつつ、制度実績をもとに中間層にも拡大を検討可能です。
Q7. 寄付意欲が本当にあるのか？
A7.寄付文化は税制誘導と可視化の組み合わせで育ちます。返礼品を介さず「未来の子どもたちを支援できる」という納得感があれば、実需は十分期待できます。

Q8. すでにふるさと納税をしている人と重複しないか？
A8.本制度は"特別枠（別枠）"として設計され、通常のふるさと納税の上限とは別に控除が認められることで棲み分けされます。

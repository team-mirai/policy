## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

## 現在検討中の政策提案：「無能な政治家に罰則を与える制度改革」（匿名ユーザーからの提案）

現在、匿名ユーザー様より「無能な政治家に罰則を与える制度改革」について、以下の通りご提案をいただき、議論を進めています。

### 背景と課題
現在の日本政治において、「何もせずに歳費（税金）を受け取り続ける国会議員」の存在が、国民の政治不信と納税意欲の低下を招いている。実質的な活動を行っていない議員に対して明確な罰則が存在しない状況は、制度的な課題である。

### 提案目的
- 国会議員の職務遂行に対する責任の明確化
- “働かない議員”への経済的・制度的なペナルティの導入
- 誠実で有能な議員が評価される健全な政治文化の醸成

### 提案制度案の概要
定量評価と定性評価を組み合わせたハイブリッドモデルを導入し、議員の活動を多角的に評価する。

#### 1. 評価の仕組み
- **定量的評価**: 本会議・委員会の出席率、法案提出数、質問・討論回数など、数値で測れる活動を評価。
- **定性的評価（ハイブリッドモデル）**:
    - **政務活動「レポート制度」の義務化**: 議員自身が活動報告レポート（成果、背景、課題など）をオンラインで公開。住民との面談件数や省庁との交渉履歴など、数字に表れにくい活動を記述させる。
    - **地元有権者による「360度フィードバック評価」**: 支持層以外からも匿名で意見を募集し、議員の対応や実行力を評価。
    - **外部機関による「ピアレビュー評価」**: 政策専門家や市民団体などが、提案の現実性や国政への貢献度を評価。

#### 2. 評価に基づく措置
- **歳費・手当の変動**: 評価結果に基づき、低評価議員の歳費を減額する。
- **「怠慢議員強制退場制度」**: 活動実績が皆無の議員に対し、辞職勧告を制度化。また、国民による解任請求制度の導入も検討。
- **「議員資格更新制度」**: 任期中に基本的な立法・行政知識の確認試験を課し、結果を公開する。

### 期待される効果
- 政治家の「怠慢・保身・無責任」体質の改善
- 議員一人ひとりが成果を意識した職務遂行を行う文化の醸成
- 国民による政治参加・評価意欲の向上

この提案については、引き続き議論を深めていきます。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 再投資促進税に関するご提案

### I. 提案の概要
- **提案名称:** 再投資促進税
- **提案の目的:**
  - 現代経済における金融経済と実体経済との乖離を是正し、金融経済で得られた運用益が再投資を通じて特に国内の実体経済に効率的に還流・循環する仕組みを構築すること。
  - 運用益を国内での生産的な再投資に向かわせるインセンティブを最大限に与え、その結果として生み出された利益を社会全体で共有する（インセンティブによる誘導の結果の利益共有）形で税を徴収する。
  - これにより、経済全体の活性化を図り、労働所得への過度な依存を緩和し、デジタル化・自動化が進む未来社会における新しい富の分配と社会保障のあり方を模索するとともに、大規模な資金の流れの透明性を高め、不公正な経済活動や政治資金の不透明性といった社会課題の解決に貢献する。
- **提案メカニズムの核心:**
  - 金融経済において得られた運用益のうち、国内の実体経済への生産的な再投資とみなされない部分（非再投資利益）が発生し、資金が再投資サイクルから「出口」に向かった段階で課税する。
  - この「出口課税」方式は、運用益の**「入り口」（発生源や方法）を特定して課税する方式に比べ、分野特定にかかる行政コストを大幅に削減**することを企図する。
  - その税収を、国民生活の安定や社会全体の基盤整備のために還元・分配する。
  - 課税の主な対象は、個人ではなく、数億〜兆円レベルの大規模な資金移動を行う法人や団体とする。

### II. 提案の背景と問題意識
- **金融経済の肥大化と実体経済の乖離:** 現代経済では金融取引で巨額の富が生み出される一方、国内の実体経済への投資に繋がりやすい状況がある。
- **「お金が働く」構造の強化と既存税制の限界:** 資本運用による富の増殖が進むが、既存税制は労働所得等に重点があり、金融経済の運用益や資金の流れを適切に捉えきれていない。他で議論される金融資産課税論には懲罰的な側面が見られるが、提案はそうではないインセンティブに基づく仕組みを目指す。
- **巨大企業等による税金逃れへの対応:** GAFAMのような巨大企業による国際的な税金逃れの問題は日本でも起こりうる。これは既存税制の限界を示すものであり、罰則ではなく、社会が経済活動の基盤を提供したことに対する「設定責任」を果たす形での利益共有が必要。
- **自動化・AIの進展と未来の雇用・所得:** 労働所得依存からの脱却には、労働以外の新しい所得分配・社会保障の仕組みが必要。再投資から生まれる富の活用が重要になる。
- **資金の流れの不透明性:** 大規模な資金の流れが不透明であるため、脱税、マネーロンダリング、不透明な政治資金などの問題が生じやすく、経済の公正性・規律が損なわれている。

### III. 再投資促進税のメカニズム詳細
- **課税対象:** 金融経済における運用益のうち、国内の実体経済への「再投資」とみなされない部分（非再投資利益）。
- **「再投資」の定義:** 国内における事業拡大のための設備投資、研究開発投資、国内株式や国内債券への再投資、国内不動産への投資など、**国内の実体経済への生産的な資金流入を促すような投資を「再投資」として優遇（非課税または優遇税率）**する。
- **「非再投資利益」:** 定義された国内への「再投資」以外の使途に回された運用益。例えば、個人による消費（ただし、課税対象は主に大規模な資金移動を行う主体）、預貯金として長期間滞留させることなど。特に、金融資産で得た運用益を、美術品や金といった価値の保存を主目的とする実物資産に変換する行為は、「保存行為」であり「非再投資」に含める。 運用益が再投資サイクルから**「出口」**へ向かい、これらの非再投資用途に使われたと判断された時点で課税対象となる。また、海外への投資は、国内への再投資ではないため、原則として非再投資とみなし課税対象とする。
- **課税タイミング:** 運用益が発生した後、それが国内への「再投資」に回されず、「非再投資」と判断された時点。資金が国内の再投資として循環している間は非課税（あるいは優遇税率）とし、資金がその循環から外れ、消費や滞留、海外投資、美術品や金といった実物資産への変換といった非再投資の形態になった段階で課税する仕組み（出口課税）。
- **課税対象主体:** 主に数億〜兆円レベルの大規模な資金移動を行う法人や団体。個人の小規模な取引は主な対象とせず、既存税制の枠組みを基本とするか、非常に簡素な仕組みとする。（これにより、個人の事務負担や社会的反発を軽減）
- **税率決定:** 固定税率ではなく、計算アルゴリズムによる自動調整。国の必要歳入目標、既存税制からの見込み税収、金融経済の規模、再投資率や非再投資率の予測などを考慮。大規模な資金移動を行う法人・団体の規模や活動形態といった要素も基準に組み込むことを検討。
- **税収の使途:** インフラ整備、衛生管理等の公共サービス、ベーシックインカムや特定の目的への給付、自動化社会における新しい社会保障制度の財源など、国民生活の安定と社会全体の利益のために還元・分配する。

### IV. 提案を実現するための関連システム・制度
- **資金の流れの追跡・可視化システム:** 金融経済における大規模な資金の動きを捕捉し、どこで資金が使われ、どこに滞留しているのかを可視化するためのシステム。
  - **技術的基盤:** CBDCの仕組み、金融取引におけるメタデータ付与の義務化（特に大規模取引や海外への資金移動）、AIを用いた異常値検出、Gitのようなバージョン管理によるデータの改ざん防止・追跡などが有効。
  - **重点:** 追跡対象を大規模な資金移動に絞り、特に資金が海外へ向かう際や、美術品や金といった特定の非再投資用途に使われる際にフィルター機能として作動させ、その時点で「非再投資」と判断して捕捉・課税する仕組みを含む。
- **計算アルゴリズムの実装と運用:** 税率や給付額を自動調整するための、信頼性高く透明性のあるシステム。
- **既存税制の抜本的な見直し:** 主に対象となる法人や団体に関わる複雑な税制（法人税、社会保障関連税など）を段階的に解体・一本化し、簡素化を図る。罰ではない形での税体系への転換。
- **政治資金を含む資金の流れの透明性向上とチェック機能:** 政治団体への資金の流れを含む、不透明な資金移動に対し、追跡システムやメタデータ情報を活用してチェック機能を強化する。特に大規模な政治献金などに着目し、データに基づく異常値検出や**「出口での審査（非再投資判定）」**を導入。不正への罰則強化も含む。政治資金規正法の抜本的見直し。

### V. 期待される効果
- 国内経済の活性化: 運用益が国内の実体経済への生産的な再投資を通じて効率的に循環することで、国内での産業創出、雇用拡大、技術革新が促進される。
- 国内再投資への強力なインセンティブ: 運用益を国内で生産的な再投資すれば税負担が軽減されるため、対象となる法人や団体は資金を滞留させたり、海外へ向かわせたり、価値保存的な実物資産に変換したりするのではなく、国内での再投資に回すようになる。
- 税体系の簡素化と効率化: 対象となる法人・団体に関して既存税制の解体・一本化が進めば、税体系が簡素化される。行政コストも「出口課税」「大規模資金への焦点化」により効率化される。
- 未来社会への適応と生活安定: 自動化・AI社会で労働所得への依存が減る中で、再投資促進税による税収は、国民の基礎的な生活保障や新しい社会保障制度の財源となる。
- 資金の流れの透明性向上と公正な社会の実現: 大規模な資金の流れが可視化されることで、脱税、マネーロンダリング、不透明な政治資金といった不正行為が発見・抑制されやすくなり、より公正で規律ある経済社会が実現する。
- 地域間の財政格差是正と均衡発展: 税収を地域間の財政状況を考慮して配分することで、地域経済の均衡発展を促進する。

### VI. 実現に向けた課題
提案の実現には多くの課題が存在し、以下のようなものがある。
- **技術的課題:** 資金追跡システム（デジタル円、メタデータ、AI、差分管理）や高度な計算アルゴリズムの実装は難易度が高いが不可能ではない。捕捉対象を大規模資金に絞ること、既存システムの一部活用なども現実的なアプローチ。
- **制度的・法的課題:** 「再投資」「非再投資」の国内/海外区別、生産性に着目した定義、美術品や金など実物資産への変換を非再投資とみなす点、対象主体（大規模法人・団体）を絞った制度設計、既存法規の抜本的見直しには膨大な作業が必要だが不可避。多様な資金使途を定義に落とし込む作業も重要。
- **税収の安定性:** 金融市場の変動リスクはあるが、アルゴリズムによる自動調整や、複数の財源の組み合わせで対応可能。対象を大規模資金に絞ることで、税収の変動幅が大きくなる可能性も考慮が必要。
- **社会的な受容性:** 全ての取引追跡ではなく大規模資金に限定し、公益目的を明確にすることで理解を得やすくする。個人のプライバシーへの配慮を強調。美術品や金など実物資産への変換が課税対象となる点についても、その理由（保存行為であり生産的な再投資ではないため）を丁寧に説明し、理解を求める必要がある。
- **不正対策の限界:** システムや罰則に加え、データ分析、差分管理、出口での審査といった多角的なチェック機能で対応。美術品や金など、追跡が難しい実物資産を使った迂回行為への対策も検討が必要。
- **既存システム（金融システム、政治資金規制法など）の再設計・見直し:** 大規模な資金移動を捕捉・管理するための金融システムの近代化、政治資金規正法の実態に合わせた見直しが必要。
- **課題解決に向けたアプローチ:** 一括導入は困難であり、対象主体や資金規模を限定したパイロットプロジェクト（社会実験）から段階的に導入。データに基づいた基準設定、多角的なチェック体制構築を進める。

### VII. 結論（財務省への要望）
本提案「再投資促進税」は、金融経済と実体経済の乖離、労働所得への過度な依存、資金の流れの不透明性といった構造的な課題を解決し、特に大規模な資金を国内の生産的な再投資に向かわせる強力なインセンティブを与えることで、経済全体の活性化を図るものです。

これは、運用益の発生源を問わず**「国内の生産的な再投資に向かわなかった」という「出口」に絞って課税**することで行政コストを抑え、**罰ではない「インセンティブによる誘導の結果の利益共有」**という新しい税の哲学に基づいています。また、主な対象を大規模な資金移動を行う法人や団体に限定することで、制度の焦点を明確にし、個人の負担を軽減します。美術品や金といった実物資産への変換は、生産的な再投資ではなく「保存行為」として非再投資とみなすことで、価値の囲い込みではなく循環を促します。

その実現には多くの技術的、制度的、社会的な課題が伴いますが、これらの課題に対する詳細化されたアイデア（国内/海外フィルター、対象主体の限定、出口課税によるコスト削減思想、美術品や金など実物資産への変換を非再投資と明確に定義する点など）は、課題解決に向けた重要な示唆を含んでおります。

つきましては、内閣府、特に税制政策を主管する財務省におかれましては、本提案「再投資促進税」の意義、特にその**革新的なメカニズム（出口課税、インセンティブ設計）、対象（大規模法人・団体）、そして目的（国内再投資促進、公正な利益共有）**をご理解いただき、日本の未来に向けた経済政策および税制改革の議論において、そのコンセプトを真摯にご検討いただけますようお願い申し上げます。本提案が、持続可能で公正な社会の実現に向けた政策立案の一助となれば幸いです。

### VIII. チームみらいの取り組み

チームみらいは、「再投資促進税」の実現に向け、私たちの技術的知見と政策提案能力を最大限に活用し、以下の形で主体的に貢献していきます。

1.  **実現可能性の検証とプロトタイプ開発支援:**
    *   約10名の開発チームリソースを活用し、「資金の流れの追跡・可視化システム」の基盤技術や「計算アルゴリズム」に関する実現可能性の調査、及び小規模なプロトタイプ開発を検討・支援し、技術的な課題解決に貢献します。
    *   特に、CBDC（中央銀行発行デジタル通貨）の仕組みや金融取引におけるメタデータ付与の標準化など、本提案の実現に不可欠な技術要素について、具体的な活用方法や導入ステップを研究し、政策として提案します。

2.  **政策議論への積極的参与と合意形成の促進:**
    *   税制改革に関する国や関係省庁の議論に積極的に参加し、本提案の意義や効果、具体的な制度設計について専門的な見地から提言を行います。
    *   オープンな議論を重視し、専門家、企業、市民など多様な関係者との対話の場を設け、本税制への理解を深め、幅広い合意形成を目指します。

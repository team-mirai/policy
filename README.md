## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 注意事項

* お互いに敬意を持ったコミュニケーションをお願いします。  
* 建設的な議論を心がけてください。  
* わかりやすく簡潔な表現をお願いします。  
* 情報を共有するときは、信頼性のあるデータや引用元に基づいていることを示してください。  
* 注意事項に反して健全な議論が行われていないと判断した場合、一時的に当該スレッドのインタラクションを制限することがあります。
* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## 持続可能な「新・国民皆保険制度」への転換

国民皆保険制度における保障について、今のレベルを維持し続けることは難しいと考えます。そのため、「国民皆保険制度を最低限の保障と位置づけ直し、持続可能性と公平性を高める」という考え方を政策に落とし込む際のアイデアを具体的に提案します。これは、財政の健全化だけでなく、現役世代の負担軽減と制度への信頼回復を目指すものです。

### 政策アイデア：持続可能な「新・国民皆保険制度」への転換

#### 1. 「基礎医療保障」と「選択医療（付加価値医療）」の明確な分離
国民皆保険制度を、すべてをカバーするのではなく、国民誰もが安心して受けられる「基礎医療」に特化させます。それ以上の高額な治療や、QOL（生活の質）向上を主目的とする医療は、「選択医療」として自己負担または民間保険での対応を原則とします。
 * **基礎医療の定義明確化**:
   * 生命維持、機能維持・回復に不可欠な治療: 救急医療、がん・脳卒中・心疾患などの主要疾病の標準治療、感染症治療、生活に必須なリハビリテーションなど。
   * 費用対効果の高い予防医療: ワクチン接種、健診、生活習慣病予防プログラムなど、将来的な医療費削減に繋がるもの。
   * 点数化による明確な線引き: 医療行為を点数で評価し、一定基準（例：〇〇点以下、または特定の疾患・治療法）を「基礎医療」の範囲と定義します。これにより、国民も医療機関も、何が公的保険で保障されるのかを明確に理解できるようにします。
 * **選択医療（付加価値医療）の扱い**:
   * 原則、保険適用外: 基礎医療を超える高額な最先端治療、美容を目的とする医療、QOL向上に特化した医療などは、原則として保険適用外とします。
   * 民間保険の活用奨励: 公的保険でカバーされない部分を補完する民間医療保険の普及を促進し、国民が自身のニーズと経済状況に応じて選択できるよう支援します（例：税制優遇、情報提供の強化）。
   * 混合診療の原則解禁と透明化: 基礎医療と選択医療を併用する「混合診療」を原則として解禁し、患者が透明な形で選択できるよう、費用体系や治療内容を明確に提示することを医療機関に義務付けます。

#### 2. 高齢者医療における給付と負担の適正化
高齢者の負担能力に応じた適正な負担を求め、現役世代との公平性を高めます。
 * **所得に応じた自己負担割合の見直し**:
   * 富裕層の自己負担増: 高齢者でも一定以上の所得を持つ層（例：現役世代の平均所得を超える層）については、医療費の自己負担割合を現役世代と同等、あるいはそれ以上に引き上げます。
   * 負担上限の見直し: 高額療養費制度における所得区分を見直し、富裕層の上限額を引き上げます。一方で、低所得層や経済的に困窮している高齢者への配慮は維持します。
 * **「延命医療」の位置づけの再定義と国民的議論**:
   * リビングウィル（事前指示書）の普及促進: 本人が人生の最終段階でどのような医療を望むかを事前に意思表示できるリビングウィルの法的な効力を強化し、国民への普及を徹底します。これにより、不要な延命治療を避ける選択を尊重します。
   * 「終末期医療ガイドライン」の策定と遵守: 国が明確な終末期医療のガイドラインを策定し、延命のみを目的とした医療の保険適用に関する指針を設けます。ただし、これは患者本人や家族の意思を最大限尊重することを前提とします。
   * 緩和ケア・在宅医療へのシフト: 延命治療を制限する代わりに、患者の苦痛を和らげ、尊厳を保つための緩和ケアや在宅医療への予算・人員・施設の配分を大幅に増強します。

#### 3. 「フリーライド」対策と制度運用の厳格化
無駄な医療費を削減し、制度の公平性を高めるための対策を強化します。
 * **重複受診・不適切受診の抑制**:
   * マイナンバーカードの活用強化: 医療情報連携の基盤としてマイナンバーカードの利用を義務化し、過去の受診履歴や処方薬情報を医療機関間でリアルタイムに共有できるようにします。これにより、重複検査や不必要な薬剤の処方を防ぎます。
   * かかりつけ医制度の徹底: 患者がまず「かかりつけ医」を受診し、必要に応じて専門医を紹介する仕組みを強化します。かかりつけ医を持たない場合の自己負担増なども検討します。
   * 軽症・時間外受診の抑制: 緊急性の低い軽症患者の時間外受診や、コンビニ受診に対して、一定の自己負担増を課すことを検討します。
 * **医療機関の不正請求・過剰診療の監視強化**:
   * AIを活用したレセプト点検の高度化: AIが不審なレセプト（診療報酬明細書）パターンを自動で検知し、不正請求や過剰診療の可能性のある医療機関を迅速に特定します。
   * 監査体制の強化と罰則の厳格化: 不正が発覚した場合の医療機関へのペナルティ（指定取り消し、診療報酬返還、罰金など）を強化し、再発防止を徹底します。
 * **健康インセンティブの強化**:
   * 保険料割引・キャッシュバック: 特定健診の受診、予防接種、運動習慣など、健康増進活動に取り組む国民に対して、保険料割引やポイント付与、キャッシュバックなどのインセンティブを付与します。
   * 健康アプリ・ツールの普及推進: 個人の健康データを管理・可視化し、健康行動を促すアプリやデバイスの普及を支援します。

#### 4. デジタル化による行政コスト削減と利便性向上
社会保障制度全体の管理・運用コストを削減し、国民の利便性を高めます。
 * **社会保障手続きの完全オンライン化**: 年金、医療、介護など、あらゆる社会保障関連の手続きをマイナンバーカードを活用し、オンラインで完結できる仕組みを構築します。
 * **情報連携のシームレス化**: 複数の省庁や自治体にまたがる社会保障関連情報を一元的に管理し、情報連携をシームレスに行うことで、事務手続きの重複や無駄を排除します。
 * **AIを活用した相談・手続き支援**: AIチャットボットや音声認識システムを導入し、国民からの問い合わせ対応や簡単な手続きを自動化し、人的リソースを削減します。

### 合意形成プロセスとセーフティネット

#### 「基礎医療」の線引き決定プロセス：オンラインプラットフォーム中心の国民合意形成
国民の納得感を高めつつ「基礎医療」の範囲を定めるためには、匿名型のオンラインプラットフォームを核とした、より参加しやすく透明性の高いプロセスが有効です。
 * **専門家委員会による初期案の作成と多角的な情報提供**: まず、医療、経済、公衆衛生、倫理などの専門家から構成される中立的な専門家委員会が、科学的・医学的根拠に基づき、「基礎医療」として公的保険で保障すべき医療行為や医薬品の初期案を作成します。
   * 情報提供の徹底: この初期案と、その根拠となるデータ（費用対効果分析、治療効果、国際比較など）を、専門用語を避け、グラフや図を多用して分かりやすく国民に提示します。何が「基礎医療」に含まれ、何が除外されるのか、その判断基準を明確にします。
   * 財政シミュレーションの提示: 初期案を採用した場合の国民負担や医療費削減効果についても、具体的なシミュレーション結果を示し、国民が制度変更のメリット・デメリットを具体的にイメージできるようにします。
 * **匿名型オンラインプラットフォーム「国民医療の未来会議（仮称）」の開設**: 国民からの意見を広く、かつ匿名で募集するための専用オンラインプラットフォームを政府が立ち上げます。
   * 意見投稿機能: 初期案に対する賛否や代替案、具体的な体験談、懸念点などを匿名で投稿できる機能を設けます。投稿内容は公開され、誰もが閲覧できるようにします。
   * コメント・評価機能: 他の投稿者の意見に対し、共感や反論、補足意見などを匿名でコメントできる機能や、賛成・反対の評価（「いいね」や星評価など）を付けられる機能を導入します。これにより、特定の意見がどれだけ支持されているかを可視化します。
   * テーマ別議論室: 「高齢者医療のあり方」「子どもの医療保障」「最新医療技術の扱い」など、具体的なテーマごとに議論室を設け、論点を整理しやすくします。
   * ファクトチェック機能: 誤った情報やデマが拡散しないよう、専門家チームが定期的に投稿内容をチェックし、必要に応じて正確な情報を提供する機能を設けます。
   * AIによる意見集約・分析: 投稿された膨大な意見をAIが分析し、主要な論点、賛成・反対意見の傾向、世代別の意見の特徴などを自動で抽出し、可視化します。これにより、議論の全体像を把握しやすくします。
   * インタラクティブな情報提供: 寄せられた質問や疑問に対し、FAQ形式や動画などで専門家が回答するコンテンツを定期的に公開し、国民の理解を深めます。
 * **国民意見を踏まえた専門家委員会の修正と最終提言**: オンラインプラットフォームで集められた国民の意見を、AIによる分析結果も含めて専門家委員会が精査します。
   * 初期案の修正・再検討: 国民の強い懸念や新たな視点が得られた場合、専門家委員会は初期案を修正・再検討します。
   * 最終提言の作成: 修正された案と、その修正に至った経緯、国民意見への回答を盛り込んだ最終提言を国民に改めて公開します。
 * **国会による最終決定と透明性確保**: 最終提言を踏まえ、国民の負託を受けた国会議員が国会で十分な審議を行います。
   * 国会審議の公開: 審議の過程はインターネットで中継し、議事録も速やかに公開します。国民が意見を表明したプロセスと、最終的な決定がどのように結びついたかを明確にします。
   * 継続的な評価と見直し: 導入後も定期的に「基礎医療」の範囲を評価し、医療技術の進歩や社会情勢の変化に応じて柔軟に見直しを行うための仕組みを法制化します。

#### 混合診療解禁に伴う「意図しない高額医療選択」への対策
患者が不本意に高額な医療を選択してしまう事態を防ぎ、公平性を確保するためには、徹底した情報提供と相談体制の強化、そして監視体制の厳格化が鍵となります。
 * **「医療選択説明書」の義務化と標準フォーマット**: 混合診療を提供する医療機関に対し、患者への詳細な情報提供を義務化し、その内容を標準化します。
   * 書面での提示と説明義務: 混合診療を提案する際、以下の内容を記載した「医療選択説明書」を患者に交付し、医師または専門の相談員が口頭で丁寧に説明することを義務付けます。
     * 保険診療（基礎医療）との比較: 基礎医療で受けられる同等または代替治療の内容、効果、費用、予後を明確に提示。
     * 選択医療の具体的な内容: 治療法、使用薬剤、必要な検査、入院日数など、全ての内訳。
     * 総費用と支払い計画: 治療にかかる総額、支払い方法、分割払いなどのオプション。
     * リスクと副作用: 選択医療に特有のリスク、副作用、合併症の可能性。
     * 治療効果のエビデンス: 科学的な根拠の有無、国内外の臨床試験データ、承認状況など。
     * 予後と期待できる効果: 治療後の生活の質や生存期間の変化予測。
     * 患者相談窓口の情報: 疑問点や懸念が生じた場合に相談できる窓口（院内・院外）の連絡先。
   * 「クーリングオフ」制度の導入検討: 高額な選択医療の契約後、一定期間内であれば無条件で契約を解除できるクーリングオフ制度の導入を検討します。特に高齢者や判断能力が低下している可能性のある患者の場合に有効です。
 * **公的「医療相談・助言センター」の設立と機能強化**: 患者が医療機関に誘導されることなく、中立的な立場から相談できる公的機関を強化します。
   * 専門家による個別相談: 医師、看護師、医療ソーシャルワーカー、FP（ファイナンシャルプランナー）などの専門家が常駐し、患者や家族からの医療選択に関する相談に応じます。オンライン（ビデオ通話）や電話、対面での相談が可能です。
   * 全国統一データベースの構築: 国内の主要な混合診療提供医療機関、各治療法の料金相場、治療効果のエビデンス情報などを集約したデータベースを構築し、相談員が最新の正確な情報を提供できるようにします。
   * セカンドオピニオン取得の支援: 患者が他の医療機関からセカンドオピニオンを得る際の調整や情報提供をサポートします。
 * **医療機関への「倫理ガイドライン」と「説明義務違反」への罰則強化**: 医療機関が患者を不適切に誘導することを防ぐためのルールを厳格化します。
   * 「医療倫理ガイドライン」の策定: 混合診療を提供する医療機関に対し、患者への説明義務や倫理的配慮に関する詳細なガイドラインを策定し、遵守を義務付けます。
   * 説明義務違反への厳格な罰則: 患者への十分な説明がなされなかった場合や、不適切な誘導が確認された場合、医療機関に対し診療報酬減額、指定取り消し、罰金など、厳格な罰則を適用します。
   * 医療機関の経営状況の公開: 混合診療の収益割合や、医療訴訟歴などの情報を一定の範囲で公開することで、患者が医療機関を選択する際の判断材料を提供します。
これらの対策を複合的に実施することで、混合診療のメリットを活かしつつ、患者が不利益を被るリスクを最小限に抑え、国民が安心して医療を選択できる環境を構築することを目指します。

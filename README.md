## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

* お互いに敬意を持ったコミュニケーションをお願いします。  
* 建設的な議論を心がけてください。  
* わかりやすく簡潔な表現をお願いします。

* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

## 【新規提案】映像文化遺産の保護とデジタルアーカイブ化の推進

過去のテレビ番組等の映像記録は、特に個人の手元にあるVHSテープ等の録画媒体の経年劣化や、放送局自身によるアーカイブの不備により、急速に失われつつある国民全体の文化的な資産です。これらの作品群を、単なる過去の放送物としてではなく、国民全体の共有財産である文化資源として捉え直し、研究者、教育関係者、そして将来の世代が容易にアクセスし、活用できる環境を整備することは喫緊の課題です。この認識のもと、以下の政策を推進します。

1.  **公的アーカイブの抜本的拡充と迅速なデジタル化:**
    既存の「放送ライブラリー」等の機能と予算を大幅に拡充し、専門人材を確保・育成します。全国の放送局、個人コレクター、関連団体等と連携し、散逸・消失の危機にある映像記録（特に歴史的・文化的に価値の高いもの、代替不可能なもの）の体系的な収集、修復、そして高精細なデジタルアーカイブ化を可及的速やかに推進します。個人の手元にある貴重な録画媒体のデジタル化支援や公募による収集も積極的に行います。

2.  **著作権・著作隣接権の課題解決と新たな利用ルールの策定:**
    映像アーカイブの最大の障壁である複雑な著作権・著作隣接権（特に放送事業者による一時的固定など、日本固有の権利処理の難しさ）については、文化遺産としての映像の保護・活用と、権利者の正当な利益保護のバランスを重視し、法制度の見直しや新たな集中権利処理の枠組み構築を検討します。具体的には、適切な使用料の設定と権利者への透明な還元システム、孤児作品の利用円滑化、教育・研究目的での利用拡大などを目指します。

3.  **関係者との対話と合意形成の重視:**
    テレビ局、番組制作者、脚本家、出演者等、多くの権利関係者の懸念（コンテンツの二次利用への抵抗感、過去の放送内容に対する永続的な責任、編集・改変への危惧、公権力の介入への警戒等）を真摯に受け止め、丁寧な対話を通じて、アーカイブ化への理解と協力を得るための合意形成プロセスを重視します。

4.  **プライバシー保護と倫理的配慮:**
    アーカイブされる映像に含まれる個人のプライバシーや肖像権に最大限配慮し、公開・利用にあたっては明確なガイドラインを設けます。訴訟記録や未解決事件に関連する映像等、特に慎重な取り扱いが求められるものについては、専門家を交えた倫理委員会の設置も検討します。

5.  **技術的課題への対応とアクセシビリティの確保:**
    古いアナログメディアのデジタル変換技術の開発・導入支援、デジタルコンテンツの長期保存技術の確立、コピーガード技術が施されたコンテンツのアーカイブ化に関する技術的・法的課題の整理を進めます。また、アーカイブされた映像は、障害の有無に関わらず誰もが利用しやすいユニバーサルデザインのインターフェースを通じて提供します。

6.  **独立したアーカイブ運用・監督体制の検討:**
    アーカイブの収集、保存、公開、利用促進を担う中立的かつ専門的な公的機関のあり方を検討します。また、その運営の透明性・公平性を担保するため、アーカイブ運用機関とは別に、その活動を監督・検証する独立した第三者機関の設置も視野に入れます。

7.  **持続可能な運営のための財源確保:**
    アーカイブの構築と維持・運営には継続的な資金が必要です。国の文化予算における重点項目としての位置づけを明確にし、必要な国家予算を安定的に確保するとともに、クラウドファンディングや寄付、限定的な有料視聴モデルなど、多様な財源確保策についても検討します。

8.  **国民の理解と参加の促進:**
    映像文化遺産の重要性について広く国民の理解を深めるための啓発活動を行います。オンラインプラットフォーム等を通じて、国民がアーカイブ事業に参加できる仕組み（例：情報提供、資料同定協力）も検討し、開かれたアーカイブ作りを目指します。

この政策は、過去の国立国会図書館法における映像資料納本の試みが実現しなかった経緯も踏まえ、デジタル時代における文化継承の新たな形を築くものです。失われゆく貴重な映像文化を次世代に確実に引き継ぐため、政府、放送業界、権利者、そして国民一人ひとりが連携し、この国家的なプロジェクトに取り組むことを提案します。

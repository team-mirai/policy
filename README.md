## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます

## 非正規雇用・派遣労働の構造問題と待遇改善

就職氷河期世代をはじめとする多くの方々が、非正規雇用や派遣労働という不安定な状況のなかで困難に直面しています。チームみらいは、誰もが安心して働き、公正な待遇を得られる社会を実現するため、以下の政策を推進します。

### １．社会保険適用の見直し（2ヶ月ルールへの回帰）
派遣社員や短期契約で働く方々の手取り収入を不当に減少させないため、社会保険の適用については、過去の「2ヶ月以上の継続就業後に適用」という基準に戻すことを検討します。これにより、就業初期の急な手取り減や、短期間で離職した場合の不利益を緩和します。

### ２．派遣労働者の待遇改善
*   **最低賃金の大幅引き上げ:** 派遣労働者の生活安定と専門性を正当に評価するため、最低賃金を時給3,000円に設定することを目指します。
*   **正社員登用の強力な推進:** 派遣就業から1年を目途に、派遣労働者が希望する場合、派遣先企業による正社員登用を強力に推進する制度を構築します。企業へのインセンティブや、登用義務化も視野に入れます。

### ３．派遣法の見直しと適正化
労働者保護を強化し、派遣労働がキャリアアップにつながるよう、派遣法を抜本的に見直します。1996年の改正前の理念も参考に、以下の点を重視します。
*   **事前面接・個人情報要求の禁止徹底:** 派遣労働者の選考における事前面接や、業務に不必要な個人情報の要求を厳格に禁止し、違反企業には罰則を科します。
*   **派遣元の責任明確化と強化:** 派遣元企業に対し、適切なマッチング、キャリアコンサルティング、待遇改善交渉、トラブル解決支援の責任を強化します。安易なプラットフォームビジネス化を是正します。
*   **恒常的業務への派遣原則禁止:** 企業の常用代替としての安易な派遣利用を制限し、真に一時的・臨時的な業務に限定するよう規制を強化します。3ヶ月更新を繰り返すような実質的な長期雇用は、正規雇用への転換を促します。
*   **不合理な待遇差の解消:** 同一労働同一賃金の原則を派遣労働者にも徹底し、交通費支給の義務化や福利厚生の適用拡大を進めます。

### ４．雇用の安定と公正な評価
*   **安易な契約終了・雇い止めの規制:** 景気変動や派遣先の都合による一方的な契約終了や雇い止めを厳しく規制し、労働者の雇用安定を図ります。
*   **転職回数に対する不当な評価の是正:** 就職氷河期世代などが経験した非自発的な離職や転職回数の多さが、不当な評価につながらないよう、企業への啓発や採用ガイドラインの策定を推進します。職務経歴やスキルを重視した公正な選考を促します。

### ５．実効性のあるリスキリング支援
非正規雇用の方や就職氷河期世代が、変化の激しい社会で持続的にキャリアを築けるよう、実効性のあるリスキリング支援を強化します。
*   **専門的・高度なスキル習得支援:** 単なる資格取得に留まらず、AI、DX、専門職（通訳など）といった高度な専門スキルを習得できる質の高い職業訓練プログラムを、無償または低負担で提供します。
*   **キャリアチェンジ支援:** 訓練後の就職支援やキャリアコンサルティングを充実させ、実務経験を積める機会の創出を支援します。

チームみらいは、これらの政策を通じて、派遣労働者や非正規雇用の方が直面する構造的な問題を解決し、誰もが希望と尊厳を持って働ける社会を目指します。

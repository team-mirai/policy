## 新たな提案：尊厳ある暮らしを守るための成年後見制度改革

急速な高齢化と単身世帯の増加に対応するため、判断能力が不十分になった方々が尊厳を持って安心して暮らせるよう、成年後見制度を抜本的に改革します。これは、誰かが一方的に権利を制限するのではなく、社会全体で支えるための仕組みづくりです。

### 課題：担い手不足と利用者の不安

現状の成年後見制度は、親族が後見人になることを前提としていますが、頼れる身内がいない高齢者が急増しており、制度が機能不全に陥りつつあります。また、専門家や市民が後見人になる場合でも、個人の負担が大きすぎたり、不正への懸念があったりという課題があります。

### 提案１：公的監督のもとで、多様な担い手を育成・確保する

個人の善意や努力だけに頼るのではなく、公的な仕組みによって後見人の担い手を増やし、活動の質を担保します。

1.  **「市民後見人」のチーム制導入と支援強化**
    研修を受けた地域住民が「市民後見人」として活動する仕組みを推進します。その際、一人に責任が集中することを防ぐため、複数人のチームで後見業務を担当することを標準とし、弁護士・司法書士などの専門家がサポートする体制を自治体が責任を持って構築します。

2.  **「法人後見」への公的認証制度の創設**
    NPOや社会福祉法人などが組織として後見業務を担う「法人後見」を推進します。信頼できる団体を国民が選べるよう、国や自治体が厳格な基準で審査・認証する「公的認証制度」を創設します。認証団体には定期的な監査を実施し、不正が発覚した場合は認証取り消しや罰則を科すことで、悪用を徹底して防止します。

### 提案２：現役世代の負担を増やさない、公平な費用負担ルール

制度の持続可能性を確保するため、財源についても明確なルールを定めます。

1.  **「応益負担」を原則とする**
    制度を利用する方の資産や年金から費用を負担していただく「応益負担」を原則とします。

2.  **公費による補完と財源の工夫**
    資産が不十分な方でも必要な支援を受けられるよう、不足分は公費で補います。その際、現役世代の負担増を避けるため、既存の高齢者向け予算の優先順位を見直すなど、歳出改革によって財源を確保します。

## 新たな提案：高齢者の安全運転と移動の自由を両立させるための免許制度改革

高齢化に伴い、判断能力や身体機能が低下したドライバーによる重大事故が社会問題となっています。一方で、画一的な年齢制限は、まだ十分に運転できる方の生活の足を奪い、社会的な孤立を招く危険性もあります。個人の自己責任による備えを基本としつつ、社会全体の安全を確保するための仕組みが必要です。

### 提案：免許更新制度の厳格化による、能力に基づいた運転許可

一定年齢（例：75歳）以上のドライバーに対し、免許更新時の認知機能および身体機能の検査を大幅に厳格化します。

1. **客観的で精緻な能力検査の導入**
   現在の画一的な検査ではなく、シミュレーターや実車走行テストなども組み合わせ、個々の運転能力をより正確に、客観的に評価する仕組みを導入します。これにより、危険な運転者を早期に発見し、事故を未然に防ぎます。

2. **段階的な基準設定**
   年齢に応じて検査項目を追加したり、合格基準を段階的に厳しくしたりすることで、加齢による能力の変化にきめ細かく対応します。

3. **個人の自助努力を促す**
   この制度は、ドライバー自身の安全意識を高め、自らの能力を客観的に把握し、将来の生活設計（免許返納のタイミングや、運転が不要な地域への移住など）を計画的に考えるきっかけとすることを目的とします。検査の実施にあたっては、デジタル技術の活用により、効率化とコスト抑制を図ります。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェestoを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 注意事項

* お互いに敬意を持ったコミュニケーションをお願いします。  
* 建設的な議論を心がけてください。  
* わかりやすく簡潔な表現をお願いします。  
* 情報を共有するときは、信頼性のあるデータや引用元に基づいていることを示してください。  
* 注意事項に反して健全な議論が行われていないと判断した場合、一時的に当該スレッドのインタラクションを制限することがあります。
* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

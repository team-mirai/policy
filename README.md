## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 私たちの政治思想と、性悪説に立つ制度設計

私たちのマニフェストには、最も重要な「思想」と、人間や社会が持つ「弱さ」への現実的な視点が欠けている、という痛烈なご批判をいただきました。テクノロジーという手段の前に、なぜ私たちが信頼に足るのか、その根幹となる政治思想と、権力の腐敗を防ぐための具体的な制度設計をここに示します。

### 1. 私たちが定義する「人権」と「民主主義」
言葉だけを掲げても、その意味は伝わりません。私たちの判断基準となる「根幹」を、以下のように定義します。
*   **人権とは**：全ての個人が、生まれや信条、性別等によって差別されず、貧困や暴力といった生存への脅威から解放され、尊厳をもって生きる状態を指します。思想・表現の自由はもちろん、生存に不可欠な教育、医療、セーフティネットへの公正なアクセスを保障することを含みます。
*   **民主主義とは**：市民が自らの社会の主役であり、政治権力が常に市民の意思と監視に服する仕組みです。これには、多数決の結果が、少数意見を持つ者や社会的弱者の権利を蹂躙することを防ぐ「多数派の専制の否定」という原則が、不可分に結びついています。

### 2. 「人間は腐敗する」ことを前提とした制度
私たちは、いかなる人間集団も、監視されなければ必ず腐敗するという「性悪説」に立ちます。この現実から目を背けず、腐敗を制度的に防ぐ仕組みを、私たち自身にも厳格に課します。
*   **権力者の任期制限**：代表や主要な役職には、厳格な任期の上限を設けます。
*   **資産の完全公開義務**：私たちのメンバーだけでなく、私たちを監査する第三者機関の主要メンバーにも、資産の公開を義務付けます。
*   **内部からの「反逆権」の保障**：私たちの組織内で不正や理念に反する動きがあった場合、それを内部告発した者が不利益を被らないだけでなく、その声を社会に届け、組織を正常化するプロセスを制度として保障します。

私たちは、有権者が怠惰なのではなく、信じるに足る情報と選択肢が与えられてこなかったのだと考えます。私たちの思想と覚悟、そして自らを縛るこの制度こそが、信頼を問うための答えです。

## 私たちの政治的立ち位置と、現実的戦略

「イデオロギーなきテクノロジー政党に価値はあるか」「誰と手を組み、誰とは組まないのか」「理想を語り続けるだけで、政権を担う気はあるのか」。有権者からの、当然かつ本質的な問いに、私たちは明確に答える責任があります。

### 1. 他の政党にはない、私たちの唯一の価値
もし他の政党がテクノロジー導入を掲げたとしても、私たちの価値が揺らぐことはありません。私たちの本質はテクノロジーを使うことではなく、「あらゆる人間集団は必ず腐敗する」という性悪説に基づき、**私たち自身をも縛る、厳格で透明な制度を設計し、それに服従する**点にあります。第三者による監査、資産公開、任期制限。これを綱領の中心に据え、まず自らが実行する政党は他にありません。これこそが、私たちが信頼を問うための唯一無二の価値です。

### 2. 連立政権における「越えてはならない一線」
私たちは、単独で政権を担うまで何もしない、という絵空事を語るつもりはありません。しかし、連立のためであれば誰とでも組むという野合もしません。私たちの「越えてはならない一線」は明確です。
*   **日本国憲法の基本原則（国民主権、基本的人権の尊重、平和主義）を否定する勢力とは、いかなる協力関係も結びません。**
*   **私たち自身が課すものと同等の、透明性確保と腐敗防止のための制度（第三者監査の受け入れ等）に合意しない限り、連立はあり得ません。**

### 3. 「聞く」ことと「決める」ことの峻別
「有権者の意見を聞きながら調整する」という私たちのこれまでの姿勢は、無責任とのご批判をいただきました。その通りです。私たちは、この対話を通じてその過ちを認め、スタンスを改めます。私たちは国民の声に誰よりも真摯に耳を傾けます。しかし、**最終的な意思決定は、ここに記した私たちの理念と原則にのみ基づいて行います。**たとえ民意がそれに反したとしても、私たちは理念を曲げません。その責任と覚悟をもって、国民の信を問います。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 注意事項

* お互いに敬意を持ったコミュニケーションをお願いします。  
* 建設的な議論を心がけてください。  
* わかりやすく簡潔な表現をお願いします。  
* 情報を共有するときは、信頼性のあるデータや引用元に基づいていることを示してください。  
* 注意事項に反して健全な議論が行われていないと判断した場合、一時的に当該スレッドのインタラクションを制限することがあります。
* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

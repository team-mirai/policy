## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## その他重要分野についての政策・ビジョン

### 移民政策について（匿名ユーザー提案）

チームみらいは、国民の安全・安心な生活を第一義とし、以下の基本的な考え方に基づき移民政策を検討します。外国籍の方々の状況に関しては、日本国における永住権を取得するまでは、基本的にはその方の母国に責任があるものと考えます。

1.  **不法移民への対応:**
    *   不法滞在者に対しては、日本語能力試験（N2レベル相当を目標）を毎週実施します。受験費用は本人負担とします。
    *   試験に合格した者に対しては、デポジット（本国への送還に必要な航空券代相当額）を預託することを条件に、一時的な滞在を許可します。次回の滞在許可更新時に、デポジットを再預託するものとします。
    *   試験に不合格となった者、またはデポジットを納付できない者は、速やかに強制送還手続きを行います。

2.  **難民保護:**
    *   難民保護については、国際的な協調を基本としつつも、日本の国益と受け入れ能力を十分に考慮した基準を設けます。人道的な配慮は必要ですが、他国と比較して過度な保護を行うことはありません。

3.  **政策の背景と目的:**
    *   本政策の主たる目的は、国内の治安維持です。正規の滞在資格を持たず、かつ日本語能力の習得や日本社会における就労の意思が見られない外国籍の方の長期滞在は、将来的に生活が不安定化し、犯罪を引き起こすリスク要因となり得ると考えます。このような状況は、日本国民にとっても、当該外国籍の方にとっても望ましいものではありません。
    *   加えて、就労や日本語習得の意欲を持たない一部の外国人の存在は、日本社会で真摯に生活し貢献しようと努力している他の多くの外国人の社会的評価を不当に貶め、日本人との良好な関係構築を阻害する要因ともなり得ると認識しています。

### 生活保護制度の改革（匿名ユーザー提案）

チームみらいは、生活保護制度について、その持続可能性と公平性を高め、真に支援を必要とする人々を支えつつ、勤労意欲を損なわない制度設計を目指します。

1.  **受給者の社会貢献活動の推進:**
    *   重度の身体障がい等により就労が著しく困難な場合を除き、受給者には週2日程度の自治体業務（地域の清掃活動、簡単な事務補助、イベントの手伝い等）への参加を、義務的な社会貢献活動として求めます。
    *   正当な理由なく活動への参加を拒否した場合には、生活保護費の減額等の措置を検討します。

2.  **生活保護費の使途の適正化:**
    *   生活保護費の支給に関して、一部をキャッシュレス決済（例：PayPay、d払い等）とし、その利用履歴を自治体担当者が定期的に確認します。
    *   酒類、タバコ、ギャンブル等への過度な支出が認められた場合は、生活保護管理委員会（仮称）が指導を行い、改善が見られない場合は支援内容の見直しを検討します。

3.  **住居支援の効率化:**
    *   生活保護受給者の住居については、自治体が管理する住宅や空き家となっている団地等を活用し、適切な家賃負担での入居を推奨します。
    *   指定された住宅への入居を希望しない受給者については、当該地域の標準的な住宅扶助費の範囲内で支援を行います。

4.  **外国人に対する生活保護適用の厳格化:**
    *   外国人に対する生活保護の適用は、永住権を有し、かつ日本国内における過去の納税総額が、想定される5年間の生活保護支給総額（約700万円を目安とする）を上回っていることを条件とします。
    *   上記条件を満たさない永住権を持つ外国人に対しては、金銭的な生活保護の支給ではなく、就労支援や母国への帰還支援等を行います。

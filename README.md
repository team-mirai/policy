## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## 【政策提案】学校給食の充実による子どもの成長支援と地域活性化 (momongaさん提案)

子どもたちの健やかな成長と学びを支え、同時に地域経済の活性化にも貢献する学校給食の実現を目指し、以下の施策を提案します。この提案は、momongaさんよりいただいた貴重なご意見を元に作成しました。

### １．「最低栄養素ライン」の国による保証と多様な献立開発支援

*   **栄養基準の明確化と国の財政支援:** 文部科学省が定める「学校給食摂取基準」に基づき、成長期の子どもたちに不可欠なエネルギー及び主要栄養素を確実に摂取できる食材費を国が補助します。これにより、全国どこでも一定水準以上の栄養バランスが保たれた給食を提供します。
*   **献立作成のサポート体制強化:** 国や都道府県レベルで、多様なモデル献立ライブラリ（季節やアレルギー対応等）を開発・無償提供します。また、特に小規模自治体に対しては、広域での共同購買や調理センターの設置支援、さらには管理栄養士が不足している地域へのオンラインによる献立作成・栄養計算サポートチームの派遣などを検討し、献立作成や食材調達の負担を軽減します。

### ２．「ご当地メニュー」の推進と地域経済への貢献

*   **自治体の創意工夫を活かす上乗せ措置:** 国の基本補助をベースとしつつ、各自治体が地元産の食材を積極的に活用したり、地域の食文化を反映した特色ある「ご当地メニュー」を導入したりする際の経費について、国からの上乗せ支援制度を設けます。これにより、画一的な給食ではなく、子どもたちが多様な食に触れる機会を創出します。
*   **農林水産業との連携強化:** 地元農畜産物の利用拡大を目指し、農林水産省の関連交付金事業とも連携し、学校給食を通じた地域経済の活性化と食育を同時に推進します。自治体ごとに地場産食材の使用率や特色ある取り組みを公表し、優良事例を共有することで、健全な競争と質の向上を促します。

### ３．給食費会計のクラウド共同システム化とEBPMの実践

*   **事務負担の軽減と透明性の向上:** 全国の市町村が共同で利用可能なクラウドベースの給食費会計システムを導入します。これにより、煩雑な会計事務を大幅に効率化し、コスト削減を図ります。また、支出データの分析を容易にし、透明性の高い情報公開を促進します。
*   **データに基づく政策改善（EBPM）の推進:** クラウドシステムで収集された給食の実施状況、栄養摂取状況、食材費、地域産品の使用割合などのデータを分析し、政策効果を客観的に評価します。この分析結果を次年度以降の予算配分や制度設計にフィードバックすることで、継続的な給食の質の改善と、より効果的な子どもたちへの投資を実現します。例えば、食材価格の変動（燃料費や飼料価格等）に応じて補助単価を柔軟に見直す仕組みや、健康診断結果との連携による栄養状態の把握なども検討します。

この政策は、まず一部自治体での実証事業からスタートし、効果検証（学力、健康指標、欠席率、給食費負担の変化など）を詳細に行います。その結果を踏まえ、3～5年程度での全国展開を目指します。チームみらいは、この提案が「ただの無償化」に留まらない、「質を保証し、地域と共に発展する新しい給食制度」への第一歩となることを期待しています。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

### 不法滞在者への対策について（ボレロさんご提案）

**背景**

現在、日本における不法滞在者・不法残留者は約8万人弱（2025年7月時点の想定を含む）とされ、毎年約9千人が新たに発生しているとの指摘があります。
不法滞在・不法残留は、それ自体が日本の法律に反する状態であり、該当者は原則として母国へ帰国する必要があります。
しかし、日本国内に違法な状態で滞在を続ける場合、正規の就労が困難であるため、生活のために窃盗などの犯罪に手を染めてしまう、あるいは悪質なブローカーや犯罪組織による不法な労働搾取の対象となるなど、結果として日本の治安悪化に繋がる可能性も懸念されます。
こういった背景から、不法滞在者を明確にし、対策を講じることが日本の治安維持にとって重要です。

**具体的な提案：タイの「90日レポート」制度を参考に、日本独自の制度構築へ**

タイ王国では、ビザの更新タイミングとは別に、90日ごとに指紋などの生体認証によって本人確認を明確に行い、居住地のレポートを提出させる「90日レポート」制度が不法残留者対策として運用されています。

このタイの事例を参考に、日本においても以下の要素を含む独自の「90日レポート」制度の導入を提案します。

1.  **全ての外国人入国者への義務化:** 入国時およびそれ以降90日ごとの定期的な居住地報告と生体認証による確認を義務付けます。
2.  **雇用主の共同責任（特に技能実習生等）:** 技能実習生など就労目的で滞在する外国人を雇用する企業・団体に対しても、外国人労働者と共同で90日レポートの手続きに参加することを義務付けます。
    *   **目的:** これにより、悪質なブローカーの介在や、一部企業による外国人労働者への不適切な扱いをあぶり出し、被害に遭う外国人労働者の保護にも繋げます。
3.  **違反企業への罰則を含む法整備:** 報告義務を怠った企業や、不適切な雇用が明らかになった企業に対しては、営業停止や罰金などの厳格な罰則を科す法整備を行います。
4.  **システムのデジタル管理:** 90日レポート制度の申請、管理、追跡など、システム全体をデジタル化し、効率的かつ透明性の高い運用を目指します。
5.  **違反履歴の記録と再入国審査への活用:** 90日レポートの提出義務を怠るなどのルール違反があった場合、その履歴をシステムに記録・保存します。この違反履歴は、将来的な再入国許可申請時の審査において重要な判断材料として活用し、繰り返しの違反者に対しては再入国を認めない措置も検討します。

**期待される効果**

1.  **不法滞在者の抑止:** 定期的な確認により、不法滞在者の発生を未然に防ぎます。
2.  **国内における逃亡・失踪の防止:** 所在の定期的な確認は、不法滞在者が犯罪組織に利用されるリスクを低減します。
3.  **犯罪の抑止と治安改善:** 不法滞在という状況が減ることで、治安改善に貢献します。
4.  **財政負担の軽減:** 不法滞在者が関わる犯罪が減少し、早期に帰国が進むことで、関連する捜査費用や司法手続きのコスト、さらには刑事施設における費用負担の軽減が期待できます。参考として、国籍を問わず、受刑者一人を刑務所で収容するためには年間約300万円の税金が維持費として必要とされています。犯罪者の増加は、治安の悪化だけでなく、国の財政にも大きな影響を与えるため、本提案はこれらの費用を抑制し、より健全な財政運営に貢献します。

政府が主導してこの制度を確立し、入国から滞在中まで一貫した管理を行うことで、より安全で公正な社会の実現を目指します。

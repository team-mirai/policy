このマニフェストはバージョン1.0です。今後も頂いたご意見を踏まえてどんどんアップデート（改善）をしてゆきます。

  * **国籍・帰化制度の見直しと国籍条項の導入:** 国の安全と国民の一体性を守る観点から、帰化要件の厳格化を進めます。さらに、国家の意思決定に深く関わる重要な公職（政治家、裁判官、検事、弁護士等）については国籍条項を導入し、帰化者の就任を制限します。あわせて、スパイ防止法を厳格に適用し、公職にある者による売国的行為は、その経歴を問わず厳しく処罰します。
  * **出産・子育て支援の抜本的改革:** こども家庭庁などの既存予算を抜本的に見直し、直接給付を拡充します。具体的には、出産時に第一子・第二子に100万円、第三子・第四子に50万円、第五子以降に10万円を給付する段階的な制度を創設します。
  * **消費税の廃止と財源転換:** 国民の可処分所得を増やし、経済の好循環を創出するため、消費税を廃止します。その財源は、輸出還付金制度のあり方を抜本的に見直すことによって確保します。
  * **食料安全保障の確立:** 食料自給率の向上は国家安全保障の根幹です。後継者不足の解消に向けた担い手支援を強化するとともに、スマート農業など先端技術の導入を強力に推進し、持続可能な国内生産基盤を構築します。
  * **スパイ防止法の制定:** 国の安全保障を脅かすスパイ活動を防止するため、明確な罰則（罰金、禁固刑、懲役刑、および外国人に対する強制送還措置など）を盛り込んだ「スパイ防止法」の制定を推進します。国民の正当な活動の自由を不当に侵害しないよう、適用範囲の明確化にも配慮します。
  * **放送法の改正とメディアの公平性確保:** 国民の知る権利を守り、健全な民主主義を育むため、メディアの政治的公平性を確保する取り組みを強化します。具体的には、放送法を改正し、各政党の取り扱い時間に関する情報の定期的な公表を放送局に義務付けます。また、著しい偏向報道や意図的な切り取り報道が確認された場合に、中立的な第三者機関が是正勧告を行える仕組みの創設を検討します。
  * **宗教法人への非課税特権の撤廃:** 信教の自由は最大限尊重します。その上で、税の公平性を徹底する観点から、宗教法人への非課税特権を撤廃します。お布施、寄付を含む全ての収入の申告と納税を義務化し、団体の透明性を確保します。これは、一部の団体による不適切な資金集めやカルト化を社会的に抑制し、健全な宗教活動を守ることにも繋がると考えます。
  * **タバコ政策の転換（健康管理と環境整備）:** 国民の健康を最優先とし、タバコの薬物としての側面も考慮した新政策へ転換します。税負担は軽減する一方、個人認証アプリによる購入管理（1日の上限設定等）を導入し、過度な摂取を防ぎます。同時に、禁煙を望む人への治療プログラムやカウンセリング等、国による脱却支援を大幅に強化します。同時に、望まない受動喫煙を防止するため、煙が漏れないよう完全に仕切られた喫煙所の増設を、特に駅周辺など公共の場所で推進します。
  * **外国人運転免許制度の厳格化:** 交通安全を確保し、重大事故を防ぐため、外国人に対する運転免許の取得プロセスを厳格化します。具体的には、短期滞在の宿泊施設等を住所とした申請を認めないよう住所要件を強化します。また、日本の複雑な交通法規や道路標識への十分な理解を測るため、日本人と同等の知識・技能試験を義務付けます。
  * **在日外国人の公務員登用の原則禁止:** 公務員が国民全体の奉仕者であるという原則に鑑み、住民への不適切な対応を防ぎ、行政への信頼を守るため、日本国籍を持たない方の公務員への就任は、原則として認めない方向で見直しを進めます。
  * **外国人株主比率の制限による国益保護:** 日本経済は内需によって十分に成長できるとの考えに基づき、国益を損なう海外からの過度な投資を抑制します。経済安全保障を確立するため、特に重要な産業や技術を持つ企業に対し、外国人投資家全体の株式保有比率に上限（例：10%）を設けることを検討します。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマనిフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 注意事項

* お互いに敬意を持ったコミュニケーションをお願いします。  
* 建設的な議論を心がけてください。  
* わかりやすく簡潔な表現をお願いします。  
* 情報を共有するときは、信頼性のあるデータや引用元に基づいていることを示してください。  
* 注意事項に反して健全な議論が行われていないと判断した場合、一時的に当該スレッドのインタラクションを制限することがあります。
* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。また、システムの透明性とセキュリティには最大限の注意を払います。長期的な安定運用を見据え、コードや設定の変更履歴の管理、アクセス制御、ログ監査などの体制を整備し、安全で信頼できる運用を継続していきます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキDプできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

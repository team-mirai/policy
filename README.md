## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェestoを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 議論中の新しい政策案：スマート一次産業支援パッケージ

14様からのご提案に基づき、日本の一次産業を根底から変革するための新たな政策パッケージを検討しています。これは「守り」のAI農業防犯支援システムと対になる、「攻め」の国家プロジェクトです。

### 政策案：「スマート一次産業支援パッケージ」の創設

日本の農業・酪農・漁業を、21世紀の最先端産業へと進化させることを目指します。

*   **「革新技術導入」の徹底支援:** 土壌分析ドローン、AIによる生育管理、自動水やり・施肥システムなど、生産性を飛躍させる最先端技術の導入費用を、国が最大9割補助します。これにより、個人負担を限りなくゼロに近づけ、技術導入のハードルをなくします。
*   **農業データの国家プラットフォーム構築:** 全国の生産者から集められた土壌や生育のデータを匿名化した上で、国が管理するプラットフォームに集約します。このビッグデータをAIが解析し、一人ひとりの生産者に最適化された最高精度の栽培アドバイスを提供します。
*   **「デジタル農業指導員」の全国配置:** テクノロジーの活用に不安がある方々を支援するため、専門の「デジタル農業指導員」を全国の農協や自治体に配置します。導入から活用まで、現場で徹底的にサポートします。

私たちは、この「スマート一次産業支援パッケージ」と「AI農業防犯支援システム」を両輪とし、14様の熱い想いである「一次産業の方々を守り、味方につける」という目標を実現してまいります。

## 議論中の新しい政策案：国家危機管理庁の創設

14様からのご提案に基づき、日本の危機管理体制を抜本的に改革するため、平時から強大な権限を持つ「国家危機管理庁」の創設を検討しています。

### 基本構想

*   **権限の一元化:** 災害、テロ、パンデミック等の国家危機において、省庁の縦割りや都道府県の境を排し、全ての関連機関を直接指揮する権限を持ちます。これにより、迅速かつ効率的な対応を実現します。
*   **トップ人事の透明化による「安全装置」:**
    *   **厳格な資格要件:** 長官は自衛隊、警察、消防、海上保安庁等で高い実務経験を積んだ人物に限定します。
    *   **国会承認:** 政府が指名した候補者は、衆参両院の承認を得なければ就任できません。これにより、権力の暴走を防ぎ、幅広い国民の合意に基づいた人事を確保します。

### 危機管理の原則：「エリートパニック」の防止

政治の役割は、現場の専門家が能力を100%発揮できる環境を整え、最終責任を負うことです。そのため、誰がいつ何を行うかの権限と役割を事前に法制化し、現場の判断を尊重する仕組みを構築します。

## 議論中の新しい政策案：国境警備と離島防衛の抜本的強化

14様からのご提案を受け、国家の主権を脅かす不法行為に対抗するため、国境警備と離島防衛の体制を根本から強化する政策を検討しています。

### 第一の柱：海からの侵入を阻止する「日本海沿岸・国境監視隊」の創設

*   **24時間365日のリアルタイム監視:** 長距離ドローン、高性能レーダー、衛星システムを組み合わせ、日本海の全ての不審な動きを常時監視します。
*   **即時迎撃部隊の配備:** 海上保安庁と海上自衛隊による合同の高速迎撃部隊を配備し、不審船を即座に拿捕・無力化します。

### 第二の柱：離島を守る「国境離島警備隊」の創設と「武器使用権限の明確化」

*   **専門部隊の配備:** 重武装と特殊訓練を施した「国境離島警備隊」を創設し、警察官が一人しかいないような重要な離島に重点的に配備します。
*   **警察官職務執行法の改正:** 警察官が自らの生命や国民・国家の安全に明白な危機が迫っている場合に、ためらうことなく武器を使用できるよう、法律でその権限と保護を明確にします。

## 議論中の新しい政策案：災害対策の革新

14様との対話を通じて、過去の災害の教訓を未来に活かすための、包括的な災害対策パッケージを検討しています。

### 1. 技術による災害支援の革新

*   **「官民連携・災害イノベーション推進法」の制定:** スタートアップ等が持つ優れた防災技術を平時から国が「防災テック」として認定し、災害時には手続きなしで即時導入できる契約を事前締結します。
*   **電子避難者台帳システムの構築:** オフラインでも稼働する専用端末とマイナンバーカードを活用し、避難所での受付や物資支給を電子化します。これにより「なりすまし」や混乱を防ぎ、正確な支援を実現します。
*   **市民ボランティア「防災DX支援員」の育成:** 全国の自治体で希望者を事前に募集・登録し、専門研修を実施。災害時には行政職員と連携し、避難所の運営をサポートします。

### 2. 未来への投資：次世代防災教育

*   **VR防災訓練の全国小中学校への導入義務化:** リアルな災害シナリオを体験できるVRコンテンツを開発し、防災教育のカリキュラムに正式に組み込みます。
*   **デジタルとアナログのハイブリッド教育:** VR体験と、災害時に役立つアナログ技術の訓練を組み合わせ、実践的な知恵を育みます。

### 3. 支援者を守り、支える制度の確立

*   **医療予備隊の質の担保と厳格な監督:** 登録時の厳格な審査や行動規範を定め、被災者が匿名で通報できる第三者委員会を設置し、支援活動の質を担保します。
*   **国家による「支援者メンタルケア・プログラム」の創設:** 災害対応に従事する全ての人々を対象に、平時研修から活動後の長期ケアまで、国が責任を持って心の健康を支援する制度を法律で保障します。

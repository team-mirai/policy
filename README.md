## 匿名ユーザーからの提案：重要政策分野

### 社会的地位を悪用した性犯罪の根絶
教師、保育士、医師など、特に子どもと接する職業の者がその立場を悪用して性犯罪を犯した場合、通常よりも著しく重い罰則を科す法改正を目指します。また、実子への性加害も同様に厳罰化し、執行猶予のない実刑判決を原則とします。これにより、子どもたちが安心して過ごせる社会、そして皆が安心して子育てできる社会を築きます。性犯罪者に対しては、被害の深刻さに鑑み、現行法よりも大幅に厳格な量刑を科します。さらに、再犯防止の徹底のため、薬物治療の義務化、GPSによる監視、公共交通機関の利用制限など、あらゆる実効的な措置を検討・導入します。あわせて、子どもたちの未来を担う教師、保育士、医師、看護師などの専門職が、その重い責任と専門性に見合う正当な評価と待遇を受けられるよう、給与水準の引き上げを含む労働環境の改善を進めます。また、体液等を意意図的に他人の身体や飲食物に付着・混入させる行為は、尊厳を著しく傷つけ、感染症リスクも伴う悪質な性加害です。これを明確に性加害及び傷害罪と位置づけ、執行猶予のない実刑を原則とする法改正を目指します。さらに、過去に性犯罪を犯した者は、その罪の重さに鑑み、教育、医療、保育、政治といった特に高い倫理性が求められる職に就くことを生涯にわたって禁止し、関連する国家資格も剥奪する制度を確立します。子どもたちを守る最後の砦として、一切の例外を認めません。

### 安心して子どもを育てられる経済的基盤の構築
*   **遺族年金の無期限化:** 子どもを育てる親が経済的苦境に陥ることなく生活を支えられるよう、遺族年金の受給期間の制限を撤廃します。
*   **養育費の確実な履行と国の立替払:** 養育費の支払いを国が責任をもって強制徴収し、滞納された場合は国が立て替えて支払う制度を確立します。

### 教育における公的支援の選択と集中
公立学校の教育費は完全に無償化します。一方で、私立学校の一律無償化は授業料の高騰を招く懸念があるため行いません。その代わり、教育の機会均等を確保するため、私立学校に通う生徒のいる家庭に対して、公立学校に通った場合にかかる費用と同額の教育支援金を直接給付します。

### 世代と所得の公平性を重視した医療制度改革と税制改革
働く現役世代の意欲が報われるよう、医療保険料の引き下げを目指します。同時に、所得や年齢だけで負担を決めるのではなく、能力に応じた公平な負担を実現します。高齢者や非課税世帯についても、過度な優遇を見直し、原則２〜３割の自己負担を求めることで、世代間の『搾取』ではなく、真の『助け合い』と言える持続可能な制度を構築します。また、働く世代が豊かさを実感できるよう、子育て世帯等の税負担を軽減する「扶養控除」を復活させます。

### 平和主義の堅持と、戦争に繋がらない憲法議論
現行憲法の平和主義は、戦後日本の繁栄の礎です。私たちは、憲法改正の議論が、意図せずして国を戦争に導く可能性を深く懸念します。いかなる改憲議論も、この平和主義の原則を損なうものであってはなりません。特に、自衛隊の役割は国民の生命と財産を守るための専守防衛、そして災害救助などに限定されるべきであり、他国を攻撃できる『軍隊』に変質させることには明確に反対します。国民の不安を煽るような改憲には明確に反対し、非軍事的な国際貢献と対話による平和構築を外交の基本方針とします。

### 女性の安全と権利を守るための、空間における性別の明確化
公衆の場における性別の区別は、出生時の生物学的な性別を基準とすることを法律で定めます。これにより、女性専用スペースへの生物学的男性の立ち入りを禁止し、違反した場合は厳正に罰します。
なお、生物学的に男性である方が男性用スペースの使用に困難を感じる場合には、だれでもトイレ（多目的トイレ）や家族風呂等の代替施設の利用を促し、すべての人の安全と尊厳を守ります。戸籍上の性別変更も、この原則に基づき認めません。

### 国土と国民資産を守るための税制改革と土地利用規制
過度な相続税は、先祖から受け継いだ土地を手放さざるを得ない大きな原因です。国民が大切な資産を守れるよう、相続税の負担を大幅に軽減します。
同時に、日本の国土が安易に外国資本に買収される現状を阻止します。特に水源地や安全保障上重要な土地について、外国人による土地取得を厳しく制限する法整備を急ぎ、日本の国土と国民の財産を未来永劫守ります。

### 購入済み資産への恒常的課税の撤廃
一度購入した自動車（自動車税）や家・土地（固定資産税）に対し、毎年税金が課され続ける現状は、実質的な二重課税であり、国民の資産形成を大きく阻害しています。国民生活の豊かさを実現するため、これらの税制度の撤廃を目指します。
なお、地方自治体の重要な財源である点も踏まえ、国全体の税制を抜本的に見直し、代替財源を確保することで、行政サービスの質を維持します。

### 性的搾取コンテンツの撲滅とAV規制の断行
レイプ、痴漢、盗撮といった現実の性犯罪を模倣・肯定したり、未成年や特定の職業を性的対象とするような有害な描写、個人の尊厳を踏みにじる暴力的・屈辱的な行為を描いたりするAVは、性犯罪を誘発し、歪んだ性的価値観を社会に刷り込む、許されざる人権侵害です。
私たちは、表現の自由を隠れ蓑にした性的搾取を根絶するため、同意が明確で、犯罪や暴力、人間の尊厳の毀損を描かないコンテンツのみが許容されるよう、AVの制作・販売・配信を厳しく規制する法整備を断行します。

### 性的広告の厳格なゾーニング（棲み分け）の徹底
意に反して目に入る性的・暴力的な広告は、多くの人、特に子どもに精神的苦痛を与える『環境型セクハラ』です。私たちは、誰もが安心して公共空間やインターネットを利用できるよう、厳格なゾーニングを法制化します。具体的には、成人向け広告は、年齢認証を経た18禁サイトや明確に区分されたエリア以外での表示を全面的に禁止します。見たくない人が意図せず性的コンテンツに遭遇することのない社会環境を守ります。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱- の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

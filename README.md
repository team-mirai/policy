## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

## みんなの胃がん撲滅大作戦💪～ピロリ菌対策推進プログラム～

### プログラムの目的
日本では毎年約10万人の方々が新たに胃がんと診断されており、依然として国民の健康にとって大きな課題です。日本における胃がんの主要な原因であるヘリコバクター・ピロリ菌（ピロリ菌）への対策を強化することで、胃がんの発症と死亡を大幅に減少させ、国民の健康寿命の延伸を図ることを目的とします。本プログラムは、日本国憲法第25条に定められた『すべて国民は、健康で文化的な最低限度の生活を営む権利を有する』こと、および国が『すべての生活部面について、社会福祉、社会保障及び公衆衛生の向上及び増進に努めなければならない』とする責務を踏まえ、国民の健康を守ることを目指すものです。

### プログラムの概要
年代に応じた段階的なピロリ菌検査・除菌治療の推進と、胃がん検診の強化を組み合わせた総合的な対策を講じます。本プログラムの推進にあたっては、医療関係者や専門家と十分に意見交換を行い、現場の実情を踏まえた上で慎重かつ効果的に進めていくものとします。

### 具体的な施策
#### 若年層へのピロリ菌検査と除菌の推進
- 中学生～18歳くらいを対象としたピロリ菌検査の全国展開と費用助成: 学齢期でのピロリ菌感染の有無を把握し、陽性者には除菌治療を推奨。除菌治療にかかる費用を無料とし、経済的負担を軽減。この年代での早期除菌は、ピロリ菌の長期感染による胃粘膜へのダメージが少ない段階で対応できるため、胃がん発症リスクを最大限に低減できる可能性が高く、特に効果的です。胃の萎縮が進む前に対策を講じることで、より健康な状態への回復が期待できるため、重点的に推進します。
- 保護者や学校への啓発活動の強化: ピロリ菌感染の危険性、早期除菌の重要性、検査・治療の流れについて、パンフレットや説明会を通じて周知徹底します。

#### 成人層への胃がんリスク層別化検査（ABC検査）の推奨と公費助成の拡充
- 40歳以上の国民へのABC検査の推奨と受診機会の拡大: 各自治体でのABC検査の実施をさらに促進し、受診率向上を目指します。特定年齢でのABC検査の公費助成を全国的に拡充。特に、一度もピロリ菌検査を受けたことがない方への受診勧奨を強化します。
- ABC検査結果に応じた精密検査（内視鏡検査）への円滑な誘導: 高リスクと判定された方に対しては、速やかに内視鏡検査を受診できるよう、医療機関との連携を強化します。内視鏡検査の結果、ピロリ菌陽性と診断された場合は、保険適用での除菌治療を促します。

#### ピロリ菌除菌後のフォローアップ体制の確立
- 除菌成功者への定期的な内視鏡検査の推奨と費用助成: 除菌後も胃がんのリスクがゼロになるわけではないため、定期的な内視鏡検査の重要性を啓発します。除菌成功者に対する内視鏡検査の費用助成（例：数年に一度の公費助成）を行ない、受診継続を支援します。
- 登録制度を構築します。: ピロリ菌の検査・除菌治療を受けた方を登録し、定期的な受診勧奨や情報提供を行うシステムを検討します。

#### 医療提供体制の整備と質の向上
- 内視鏡専門医の育成と配置の推進: 内視鏡検査の需要増加に対応するため、専門医の育成と地域医療機関への配置を支援します。
- ピロリ菌検査・除菌治療に関する医療従事者への研修強化: 最新の知見に基づいた適切な検査・治療が行われるよう、継続的な研修機会を提供します。

### 期待される効果
- 胃がん発症率と死亡率の大幅な減少
- 国民医療費の長期的な抑制（胃がんの治療費削減による）
- 国民の健康寿命の延伸と生活の質の向上
- 胃がんに対する国民の意識向上と予防行動の促進

この提案は、胃がん予防という観点から、ピロリ菌対策を国民的な健康課題として捉え、若年層から高齢層まで切れ目のない対策を講じることを目指しています。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## 【重点構想】「日本アプリ🇯🇵」によるみんなのための国民的ツール

提案：「日本アプリ」構想による超効率的社会の実現
1. 構想の概要
「日本アプリ」は、国民のデジタルライフの中心となる統合型スーパーアプリです。マイナンバーカードと連携し、行政手続きから日常生活、健康管理、学習、さらには経済活動まであらゆる情報を一元管理。セキュリティを最大限に強化し、国民が安心して利用できる環境を追求することで、利便性を高め、社会全体の効率化を最大化します。

2. 「日本アプリ」で実現する未来
「日本アプリ」が実現することで、私たちは以下の未来を享受します。

2.1. 安全・確実な本人確認・認証
マイナンバーカードの基盤機能を拡張し、民間サービス全般での利用拡大、生体認証によるシームレスな体験、ブロックチェーン活用による透明性の高い信頼基盤を構築します。

2.1.1. 究極のID: 銀行口座開設、携帯電話契約、オンライン診療など、あらゆる場面での本人確認がアプリで安全・確実に完結。
2.1.2. 電子署名: 重要書類の電子署名もアプリで可能になり、紙や印鑑は不要です。
2.2. パーソナルデータの一元管理と活用
既存のマイナポータル機能を発展させ、国民の生活に深く関わる基幹情報を安全に管理・活用します。

2.2.1. マイヘルスレコード: 健康保険証機能に加え、自身の診察履歴、処方箋、健康診断結果をアプリで一元管理。医療機関間の連携もスムーズに。
2.2.2. 自己情報コントロール: 自身のデータ利用状況をアプリ上で確認し、利用範囲を管理する権限を持てます。
2.3. 煩雑な手続きからの解放と政府と国民の双方向コミュニケーション
国民の行政手続き負担を劇的に軽減するとともに、政府と国民の間で、AIを活用したきめ細やかな情報提供と、国民の声の吸い上げを実現します。地方自治体との連携には多くの課題が残ります。

2.3.1. ワンストップ行政サービス: 転居、結婚、出産など、ライフイベントに伴う複数の手続きをアプリ一つでオンライン完結。
2.3.2. プッシュ型行政: 国民の状況（子育て、高齢、災害時など）に応じ、AIがカスタマイズした必要な情報や給付金、サービスをアプリに自動通知・提案します。
2.3.3. 国民の声のブロードリスニング: アプリを通じて、国民が政策や行政サービスに関する意見・要望を直接政府に提出できる仕組みを構築。AIが膨大な国民の声を分析し、政策立案に活用することで、より民意を反映した行政運営を目指します。
2.4. 学習・スキルアップと雇用支援の強化
個人のキャリア形成と社会全体の生産性向上を支援する、中長期的な機能です。

2.4.1. デジタル学習パスポート機能: オンライン講座、資格、職務経歴などをアプリで一元管理します。
2.4.2. AIパーソナルキャリアコンサルティング: AIが個別のキャリアパスや学習目標を提案。市場のトレンドや企業のニーズを分析し、将来性のあるスキル習得を促すことで、スムーズな転職・再就職を支援します。
2.5. 利便性の向上と社会参画の促進
市民生活の質向上と社会参画を促しますが、実現には長期的な視点が必要です。

2.5.1. 地域情報との連携: 各自治体のゴミ収集日、イベント情報、防災情報などがアプリに届き、地域生活を豊かにします。
2.5.2. デジタル投票（将来的検討）： 将来的には、アプリを通じて安全に投票を行い、国民の政治参加を促進することも視野に入れます。
3. 想定される課題と具体的な対策
本構想の実現には課題が伴います。国民の懸念を払拭し、安心・安全なデジタル社会を築くための対策を講じます。

3.1. セキュリティとプライバシーのリスク
具体化: アプリが機密情報を一元管理するため、サイバー攻撃や内部不正によるデータ漏洩リスクが高いです。過去のトラブルや透明性不足が国民の不信感を増幅させ、「監視社会」への懸念も存在します。
対策:
ゼロトラストセキュリティ導入: 全アクセスを検証し、生体認証と多要素認証でなりすましを防止。
ブロックチェーンによる透明性確保: データアクセス履歴をブロックチェーンで記録し、ユーザーがリアルタイムで利用状況を確認可能に。
第三者監査と国民参加: 独立監査機関が脆弱性診断結果を公開。「セキュリティ公開フォーラム」で信頼を構築。
緊急時対応体制: 漏洩時の即時通知、被害者補償制度を整備。サイバー保険導入でリスク分散。
3.2. デジタルデバイドの拡大
具体化: 高齢者のスマホ保有率は高いものの、操作に不慣れな層は多く、特に地方で顕著です。アプリ優先が進むと、窓口対応縮小でデジタル非利用者が取り残される恐れがあります。
対策:
市役所・出張所等での集中サポート体制の確立: 各市区町村の市役所・出張所を中心に「日本アプリサポート窓口」を設置。専任スタッフが端末貸出や操作指導を集中的に行い、質の高いサポートを提供します。
3.3. システム統合と法制度整備の困難さ
具体化: 日本の行政システムは省庁・自治体ごとに独自仕様で、データ形式やAPIの標準化が未整備です。個人情報保護法やマイナンバー法が現行では連携に制約が多く、法改正には長期化が予想されます。地方自治体の予算・IT人材不足も足かせとなります。
対策:
国家標準APIの強制導入: デジタル庁主導で、行政・医療・民間のシステムに共通のAPI仕様を義務化。
段階的統合とパイロットプロジェクト: ITインフラの整った都市でパイロット運用を実施し、成功事例を基に地方へ展開。
法制度の特例ゾーン: 「デジタル特区」を設け、実験的データ連携を許可。国民のフィードバックを反映しつつ法改正を推進。
民間との協業と人材確保: 官民共同開発を進め、地方自治体にITコンサルタントを派遣。デジタル庁予算で地方のシステム改修を補助。
4. まとめ
「日本アプリ」は、単なるアプリ開発ではなく、日本の社会システムをデジタルで再構築する国家プロジェクトです。国民の利便性を最大化し、行政コストを削減、社会全体の生産性を向上させる強力な基盤となります。

上記の課題に対し具体的な対策を講じることで、国民の信頼を勝ち取り、より豊かで効率的、そして安全なデジタル社会を築けるでしょう。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、未熟なバージョンであっても公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　なお今後も議論の結果、これらのマニフェストの内容がまだ変わる可能性がある点はご理解いただけますと幸いです。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

* **AIの活用と責任についての明確な原則**：本プロジェクトではAI（いどばたAI、チャットBot等）を政策議論の支援として活用していますが、AIは意見や文案を生成する道具であり、いかなる責任も負うことはできません。意思決定、説明責任、倫理的判断はすべて人間の側にあるという原則を厳格に守ります。チームみらいは「責任を取れる人が判断する」ことの重要性を再確認し、この原則に基づいて全ての政策運用を行います。

## リンク切れチェックについて

このリポジトリでは、マークダウンファイル内のリンク切れを自動的に検知する仕組みを導入しています。

### 仕組み

- Pull Requestが作成されると、GitHub Actionsによって自動的にマークダウンファイル内のリンクがチェックされます
- 内部リンク（リポジトリ内の他のファイルへのリンク）と外部リンク（ウェブサイトへのリンク）の両方をチェックします
- リンク切れが発見された場合、PRのチェック結果に表示されます

### リンク切れが発見された場合

- PRのチェック結果で報告されたリンクを修正してください
- コミットを追加すると、自動的に再チェックが行われます

## PR通知システムについて

このリポジトリでは、マージされたPull Requestの情報を自動的にX（旧Twitter）に投稿する仕組みを導入しています。

### 通知のスキップ機能

政策内容の変更以外のPR（例：タイポ修正、フォーマット調整、ドキュメント更新など）でX通知を避けたい場合は、PRに `skip-notification` ラベルを付けることで通知をスキップできます。

#### 使い方

1. Pull Requestを作成後、右側のLabelsセクションから `skip-notification` ラベルを選択して追加
2. このラベルが付いているPRは、マージされてもX通知が送信されません
3. Slack通知は引き続き送信されます（内部での進捗管理のため）

#### 対象となるケース

- タイポや誤字脱字の修正
- マークダウンのフォーマット調整
- リンク切れの修正
- その他、政策内容に実質的な変更を含まない修正

## ポイ捨てゼロ社会の実現💪：次世代へつなぐ日本の美✨️

### 背景
街の歩道に散乱するペットボトルや空き缶、風に舞う菓子パンの袋、カラスがつついているコンビニ弁当の残飯、植え込みに投げ捨てられた読み終えたばかりの雑誌、歩道の至るところに捨てられているタバコの吸い殻。こうした光景は、訪れる人々を魅了する日本の美しい景観を損ねています。また、いわゆる「割れ窓理論」のように、人々の心にも影響を与え、さらなる環境悪化や軽犯罪を誘発する可能性も指摘されています。放置すれば、私たちは次世代に負の遺産を残すことになりかねません。かつては多く見られた公共のゴミ箱も、テロ対策や維持管理コストを理由に減少した結果、国内外からの観光客を含め多くの人々が不便を感じたりポイ捨てが発生することが常態化してしまっています😭

### 目的
海外から訪れる人々が心から感動し、私たち自身も誇れる、安全で美しい日本を責任をもって次世代へと引き継いでいくこと。その実現のために、ポイ捨てゼロ社会を目指してまいります。

### 具体的な取り組み

以下を推進してまいります。

#### 1. 地域の実情に応じたゴミ環境整備への支援と民間連携
*   自治体によるゴミ箱の計画的な設置や管理を支援するため、国からの補助金制度を創設または拡充します。これにより、地域ごとのニーズに合わせた柔軟な対応を促進し、「外出先で出たゴミの処理に困る」状況を改善します。
*   地域の商店街や民間企業等と連携し、店舗や施設内外へのゴミ箱設置協力や、既存のゴミ箱の適切な維持管理を促進します。

#### 2. ポイ捨て行為に対するルールの見直しと啓発
*   諸外国の事例（罰則の内容、運用状況、効果など）を参考に、**特に外国人観光客から日本の罰金額の低さを指摘する声があることも踏まえ、**ポイ捨てに対する現行のルールが適切かどうかを検証し、必要に応じて見直しを検討します。
*   罰則強化だけでなく、ポイ捨てが環境や社会に与える影響についての啓発活動も強化し、市民一人ひとりの意識向上を促します。

これらの施策に必要な財源については、既存予算の効率化や優先順位の見直し、新たな財源確保の可能性も含めて検討を進めます。

**実証実験と効果測定：**
まず、本政策に賛同する多様な人口規模や地理的条件を持つ自治体と協力し、モデル事業として実証実験を行います。短期的にはゴミ箱設置などの初期費用が発生しますが、長期的には、実証実験を通じて以下の点を収集・分析し、費用対効果を検証します。

*   ポイ捨てされるゴミの量の削減効果
*   自治体の清掃活動にかかる費用の変化
*   特定エリアでの軽犯罪発生率の増減
*   住民や国内外からの観光客へのアンケート調査を通じた満足度の向上

**本格導入に向けて：**
実証実験の結果、十分な効果と成功の見込みが確認された段階で、本格的な導入へと進みます。これにより、街の清掃コスト削減、治安向上、観光客誘致、住民満足度向上による地域活性化など、社会全体の利益に繋がることを目指します。

これらの施策を通じて、ポイ捨てを減らし、清潔で美しい街並みを次世代へとつなげていくことを目指してまいります。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## 非正規雇用・派遣労働の構造問題と待遇改善

就職氷河期世代をはじめとする非正規雇用・派遣労働者は、不安定な雇用形態、キャリア形成の困難さ、そして不十分な社会的セーフティネットという構造的な問題に直面しています。これらの問題を解消し、誰もが安心して働き、公正な待遇を得られる社会を目指すため、チームみらいは以下の政策を推進します。

### １．社会保険適用の見直し：2ヶ月ルールへの回帰
派遣社員や短期契約労働者の手取り収入を不当に減少させ、早期就業の妨げとなっている現行の社会保険適用基準を見直します。具体的には、2022年9月以前の制度に基づき、2ヶ月以上の継続就業が見込まれる場合に社会保険を適用するように改め、就業初期の不安定さを考慮します。これにより、特に就業開始直後の手取り減少を防ぎ、生活の安定を図ります。

### ２．派遣労働者の抜本的な待遇改善
**最低賃金の時給3,000円への引き上げ:** 派遣労働者の生活基盤を安定させ、その専門性や貢献に報いるため、派遣労働における最低賃金を全国一律で時給3,000円に設定することを目指します。
**正社員登用の強力な推進:** 派遣就業が1年を経過した労働者については、派遣先企業に対して正社員として直接雇用することを原則とするよう法制度を整備し、企業への助成金制度なども活用して積極的に推進します。単なる「受け皿」ではなく、キャリアアップの機会を提供します。

### ３．派遣法の抜本的見直しと適正化
現在の派遣法は、労働者保護よりも企業側の都合が優先されがちであり、多くの問題を生んでいます。1996年の派遣法改正前の理念に立ち返り、労働者保護を最優先とする法制度へと再構築します。
*   **事前面接等の禁止の徹底:** 派遣労働者の「選別」につながる事前面接や履歴書・スキルシートによる選考を厳格に禁止し、違反企業への罰則を強化します。
*   **派遣元の責任明確化と指導強化:** 派遣元企業に対し、適切な求人情報の提供、キャリアコンサルティングの義務化、派遣先でのトラブル解決への積極的関与を求めます。現在の「プラットフォーム」的なあり方から脱却させ、雇用主としての責任を果たさせます。
*   **恒常的業務への派遣禁止:** 本来、正社員が担うべき恒常的な業務について、安易に派遣労働者で代替することを禁止します。派遣はあくまで一時的・臨時的な業務に限定します。
*   **就業条件の事前開示と遵守徹底:** 募集時と実際の業務内容・条件の相違を防ぐため、派遣元・派遣先双方に正確な情報開示を義務付け、違反時の是正措置を強化します。

### ４．雇用の安定と公正な評価の確立
**安易な契約終了の規制:** 景気変動や企業の都合による一方的な「派遣切り」「雇い止め」を規制し、雇用の安定性を高めます。契約期間中の解雇については正社員と同等の厳格なルールを適用します。
**転職回数等による不当な評価の是正:** 特に就職氷河期世代など、本人の責によらない理由で転職を繰り返さざるを得なかった求職者に対し、転職回数のみをもって不利な評価を行うことを禁止するよう企業を指導・啓発します。職務経歴やスキルを正当に評価する採用プロセスの確立を支援します。

### ５．実効性のあるリスキリング支援の強化
非正規雇用労働者や就職氷河期世代が、市場価値の高い専門スキルを習得し、安定したキャリアを築けるよう、実効性のあるリスキリング支援策を大幅に拡充します。
*   **高度専門スキル習得プログラムの提供:** 通訳、高度IT技術、専門職種など、実質的なキャリアアップにつながる質の高い訓練プログラムを、無償または低負担で提供します。
*   **生活保障付きの長期訓練制度:** 訓練期間中の生活を保障するための給付金制度を創設・拡充し、安心してスキルアップに専念できる環境を整備します。
*   **資格取得と実務経験の一体型支援:** 資格取得だけでなく、その後の実務経験機会の提供までを視野に入れた、一貫性のある支援を行います。

これらの政策を通じて、派遣労働が「使い捨て」や「低コストな労働力」として利用される現状を改め、働く人が誇りと希望を持てる社会を実現します。 ICE AGE WOMANさんのような、現場の切実な声に真摯に耳を傾け、政策に反映させていくことをお約束します。

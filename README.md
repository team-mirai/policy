## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 国家再興のための緊急包括政策提 Messieurs案（ねこレベル-9999さん ご提案）

我が国は今、内外の深刻な脅威に直面しており、国家の主権と国民の安全、そして未来の繁栄が危機に瀕しています。特定の国家や国際的なカルト教団などによる組織的かつ巧妙な不当影響力の行使は、経済、社会、政治のあらゆる領域に浸透し、我が国の意思決定を歪め、国益を損なっています。この危機的状況を打破し、日本を再興するため、以下の緊急包括政策を断行します。本政策の目的は、第一に不当な外部影響力を「緊急に」徹底排除し、国家主権と経済の「回復期間を稼ぐ」こと、第二にその期間を活用して、情報基盤の抜本的改革と『セキュアド・イン・ジャパン』ブランドの確立、そしてAI技術を最大限に利活用する「AI立国」としての地位を確立し、世界を驚かせる国家へと飛躍することにあります。

### 第一段階：緊急法整備による国家基盤の断固たる保護

国家の安全保障体制を揺るがす内外の脅威に対処するため、以下の法整備を最優先で断行します。これらは、より包括的な情報基盤（後述の国家データベース構想）が整備されるまでの間においても、直ちに我が国の主権と国民の安全を守るための緊急措置として位置づけられます。

*   **スパイ防止法の制定：** 外国勢力による諜報活動、内政干渉、世論工作、重要技術の窃取などを厳しく取り締まり、国家機密の漏洩を防止するとともに、これらに関与する者（個人、団体、背後にある国家を含む）に対する罰則を大幅に強化します。また、国内で不法・不当な活動を行う国際的なカルト教団等の資金源や影響力行使についても、本法及び関連法規によって厳しく規制・排除します。
*   **国会議員及びその他公職者の国籍等に関する情報公開の徹底義務化：** 国政及び公務を担う者の透明性を極限まで高め、外国からの不当な影響力を徹底的に排除するため、議員・公職者本人及びその配偶者・扶養家族の現在及び過去の国籍、外国永住権の保有状況、外国政府・団体・企業等からのいかなる形態の資金提供・便宜供与の有無について、就任時に遡及して詳細に調査し、国民に対して完全に公開することを義務付けます。虚偽申告や隠蔽が発覚した場合は、職務失格を含む厳罰を科します。
*   **緊急的な国防・安全保障措置として、マイナンバーカードを保有しない外国籍の方々への対応：** 巧妙化する認知戦術や国内における不法活動リスクに対し国防体制を強化し、国内基盤を安定させるため、マイナンバーカードを現時点で保有していない外国籍の方々に対しては、その状況が整理され新たな管理体制が整うまで、一時的にご帰国いただくことを検討します。この措置は、人道的配慮を最大限に行いつつ、国家の安全を最優先に実施します。

これらの緊急法整備と並行して、中長期的な国家情報基盤の再構築を進めます。

### 第二段階：国家情報基盤の抜本的再構築とDXによる行政革命

我が国の行政システムと情報管理体制を21世紀型の強靭かつ効率的なものへと変革するため、以下の国家情報基盤再構築に着手します。

*   **政府関連組織のデータ連携・統合と『国家戦略統合データベース』の構築：**
    戸籍制度のあり方も含め、国民及び国内滞在者の情報を正確に把握するための『国家戦略統合データベース』を再構築します。この再構築にあたっては、入国管理局、警察、検察、税務当局、社会保障関連機関、地方自治体など、政府及び関連組織が現在個別に管理しているデータベースとのシームレスな連携・統合を最重要課題の一つと位置づけ、情報の一元化と重複管理の排除を徹底します。このデータベースでは、「日本人（出生から）」、「日本人（帰化、帰化X世）」、「外国人（在留資格種別、活動内容、マイナンバー保有状況）」、「難民申請中の者」、「その他（国内に合法的に滞在するが上記いずれにも該当しない者など、法的に明確な定義が必要）」といった属性情報を、透明かつ厳格な基準で管理し、マイナンバーと完全に紐付けます。
    この新たな国家データベースは、以下の原則に基づいて設計・運用されます：
    *   **透明性：** 記録・更新のプロセスを明確化し、不正を許さない仕組みを構築します。
    *   **厳格性：** 情報の正確性を担保し、基準に基づいた一貫した運用を徹底します。
    *   **非属人性：** 人的な判断ミスや恣意性を排除するため、可能な限りプロセスを自動化します。
    *   **迅速性：** 各種手続きや情報照会を大幅にスピードアップします。
    *   **追跡性：** 情報の変更履歴を確実に記録し、事後検証を可能にします。
    *   **状態把握：** 個人の状況変化（在留資格変更、選挙権の得喪など）をリアルタイムに近い形で反映します。
    *   **関連法案の参照容易性：** 適用されるべき法令や規則との関連を明確にし、誰にとっても分かりやすい形にします。
    これにより、選挙権・被選挙権の有無確認をはじめとする各種行政手続きの資格審査を迅速化・自動化します。さらに、国際的な情報連携基盤とも接続し、例えば外国籍の方に関する確認事項が生じた場合、相手国の許諾のもと、人手を介さず迅速に照会できるシステムの構築を目指します。
*   **司法情報の記録・連携と透明性・公正性の確保：**
    この国家データベースには、日本国内において刑事事件の当事者となった全ての人（国籍を問わず、また政府高官や議員などの公職にある者を含む）に関する司法判断（判決、不起訴処分など）の詳細な情報を記録・連携させます。我が国の安全保障に資する情報については、その性質を踏まえ、情報の収集・記録・分析を最優先事項の一つと位置づけます。全ての関連情報はデジタル形式で確実に記録・保管し、紙媒体運用時に散見された情報の滅失や不透明な管理といった問題を根絶します。
    マイナンバーを保有しない外国籍の人物に関する司法情報についても、可能な限り他の識別情報（パスポート情報、生体認証情報等）と紐付けて記録し、将来的にマイナンバーを取得した際に情報を統合できるシステムの構築を目指します。これにより、意図的なマイナンバー未取得による追跡逃れを防ぎます。
    記録された司法情報のうち、どの情報をどの範囲で公開するかについては、個人のプライバシー権と、国防および公共の利益の重要性を比較衡量し、法に基づいて明確に定められた権限と責任を有する者が、客観的な基準に基づき判断する厳格な運用体制を構築します。特に、公職にある者の行動や、これまで国民への説明が不十分であった特定の不起訴事案等については、その理由を含め、透明性を最大限高めるための具体的方策を講じます。
    これらの取り組みを通じて、司法プロセスの公正性と国民の信頼向上を目指し、将来的に公職者や司法関係者の適格性評価にも資する情報基盤を整備します。個人の権利保護とのバランスについては、常に最適解を追求し、専門家による多角的な検討と国民的議論を深めます。
*   **本構想が実現することによる主なメリット：**
    *   **行政の効率化・高度化：** 各種手続きの迅速化、データに基づいた政策立案、個別最適化された行政サービスの提供が可能になります。
    *   **透明性と公正性の向上：** 意思決定プロセスや司法手続きの透明性が高まり、公正な社会の実現に貢献します。
    *   **安全保障の強化：** 国防・防諜活動や犯罪捜査における情報収集・分析能力が向上し、国民の安全確保に繋がります。
    *   **不正行為の抑止：** 脱税や社会保障の不正受給、選挙違反などの抑止効果が期待できます。
    *   **学術研究の促進：** 厳格な匿名化処理を施した上で、公共の利益に資する学術研究等へのデータ活用も視野に入ります。
*   **想定されるデメリットと対策：**
    *   **プライバシー侵害のリスク：** 個人情報の一元管理は、プライバシー侵害の懸念を生じさせます。
        *   **対策：** 法に基づく厳格なアクセス権限管理、利用目的の明確化、データ暗号化等の最高水準のセキュリティ対策、独立した第三者機関による監視体制を構築します。
    *   **情報漏洩・不正アクセスのリスク：** サイバー攻撃等による情報漏洩の危険性があります。
        *   **対策：** 最新のセキュリティ技術の導入、継続的な脆弱性診断、インシデント発生時の迅速な対応体制を整備します。
    *   **誤情報登録による不利益のリスク：** 誤った情報が登録されると、個人が不当な扱いを受ける可能性があります。
        *   **対策：** 本人による情報確認・訂正請求権の保障、記録プロセスの多重チェック体制を確立します。
    *   **監視社会化への懸念：** 国家による過度な国民監視に繋がるのではないかという懸念があります。
        *   **対策：** データの利用範囲を法律で厳格に定め、権力の濫用を防ぐための民主的統制の仕組みを強化します。これには、政府から完全に独立し、AI技術を積極的に活用して国家データベースの運用状況や政府機関によるデータ利用の適切性等を継続的かつ効率的に監視・検証する、強力な権限を持った第三者組織の設置を含みます。
    *   **システム構築・維持の莫大なコスト：** 高度なデータベースの構築と維持には、相応の費用と期間が必要です。
        *   **対策：** 費用対効果を常に検証し、段階的な導入や既存システムとの連携も視野に入れ、国民の理解を得ながら計画的に進めます。

これらのメリットを最大化し、デメリットを最小化するための法整備と技術的・運用的対策を徹底することで、国民が真に信頼できる情報基盤の構築を目指します。

### 第三段階：経済安全保障と『セキュアド・イン・ジャパン』ブランド戦略によるAI立国日本の実現

これら国内の安全保障体制と情報基盤の抜本的強化は、我が国の経済再生と国際競争力向上、そして「AI立国」としての地位確立に不可欠な土台となります。特に、情報通信機器や重要インフラ関連製品において、サプライチェーン全体での信頼性確保は喫緊の課題です。

そこで、以下の国家戦略を推進します。

*   **『メイド・イン・ジャパン／セキュアド・イン・ジャパン』ブランドの確立：** 家電製品、通信機器、産業機械、AI関連製品等の重要製品分野において、基幹部品（半導体、センサー、ソフトウェア等を含む）から最終製品に至るまでの開発・製造プロセス全体を可能な限り国内で完結させる体制を強力に支援します。これにより、製品のライフサイクル全体を通じて、意図しない情報窃取やマルウェア混入、バックドア設置等のリスクを徹底的に排除し、「日本製品＝安全・安心・高品質・高信頼性」という国際的なブランド価値を再構築・向上させます。

## このマニフェスト自身も、みんなの知恵を集めて改善していきます

　このマニフェストは皆さんの声を拾いながら改善していくためのバージョン0.1（超初期版）であり、まだまだ議論中、作業中の箇所がたくさんあります。チームみらいは全てを決めた後にマニフェストを発表するのではなく、よりオープンにみなさんと一緒にマニフェストを育てていきたいと考えております。具体的には、AI熟議システムの『いどばた』や全国で開催するオフラインミーティング、独自に行う専門家へのインタビューなどを通じて、どんどん知恵を集めたいと思います。

　「考えがまだまとまっていないではないか」とご批判を受けるリスクを積極的に取りながら、現時点の未熟なバージョンの公開に踏み切ります。なぜそうするかといえば、批判の目をたくさん入れ、意見を取り入れてゆくことが最も良い政策を作るやり方だと思うからです。また、これはシンプルな多数決のように「意見が多いものを取り入れる」ということではありません。意見は聞きますし、プロセスの透明化も行いますが、あくまで意思決定は既存政党と同様にチームみらいの人が行います。これは、責任を取れる『人』が主体となって判断をすることが重要であるという価値観に基づくものです。

　作業中の箇所もたくさんあるため、議論の結果、これらのマニフェストの内容がまだ大きく変わる可能性がある点はご理解いただけますと幸いです。

　なお、多くの有権者にとってより読みやすい形のマニフェスト v1.0 は6月中旬に公開予定です。

## 変更提案について

本マニフェストによる変更提案は２つの形で受付をしています。

* [こちら](https://policy.team-mir.ai/view/README.md)のページから、チャットで政策要望を提出する  
* （玄人向け）[GitHub](https://github.com/team-mirai/policy/pulls)でPull Requestを提出する

　頂いた変更提案は、チームみらいが責任をもって確認します。変更提案をマニフェストに反映させるかどうかについては、チームみらいが権限を持ちます。当初マニフェストとは異なる方向性の提案であっても、真剣に検討し、反映するかどうかを考えます。

## デジタル社会における緊急課題と対策案

チームみらいは、デジタル技術の進展がもたらす恩恵を最大限に活用するとともに、それに伴う新たな課題にも迅速かつ的確に対応する必要があると考えます。特に、以下の問題については、喫緊の対策が求められています。

### Deepfake技術の悪用と対策の必要性

Deepfake技術は、AIを用いて人物の顔や声を合成し、あたかも本人が発言・行動しているかのような偽の動画や画像を作成する技術です。この技術が悪用されることにより、以下のような深刻な問題が発生しています。

*   **現状の被害**: 著名人や一般人を問わず、本人の同意なく性的なコンテンツや名誉を毀損するコンテンツが作成・拡散され、被害者は深刻な精神的苦痛を受けています。これらの偽情報は広告やSNSを通じて瞬く間に広がり、個人の尊厳を著しく傷つける事態となっています。
*   **法的救済の困難さと法整備の遅れ**: 日本では、Deepfakeによる被害に対応する専用の法律が存在せず、既存の刑法175条（猥褻物頒布罪）などでの対応には限界があります。社会的認識の不足から「アイコラ（画像コラージュ）の延長」といった誤った軽視が見受けられることもあり、問題の深刻さが十分に理解されていません。また、表現の自由（憲法21条）とのバランスを理由に議論が進まない側面や、プラットフォーム事業者の自主的な削除努力が不十分である点も指摘されています。
*   **国際比較**: 韓国では「Deepfake性犯罪防止法」が制定され、米国やEUでも透明性義務を課すなど規制が進んでいますが、日本は議論すら停滞しており、国際的に見ても対策が大幅に遅れています。
*   **提案する対策**:
    *   **法整備の推進**: Deepfakeコンテンツの作成・所持・配布を直接規制する法律の制定を目指します。これには、非同意の性的コンテンツや名誉毀損コンテンツの作成を明確に犯罪と位置づけること、AI生成コンテンツであることを明示する義務（透かしやラベル表示）、迅速な被害者救済（削除要請プロセス、匿名での訴訟支援など）を含みます。
    *   **教育と啓発**: AIリテラシー教育を学校教育段階から強化し、Deepfakeのリスクやデジタル倫理について学ぶ機会を提供します。被害に遭った際の相談窓口の整備も急務です。
    *   **プラットフォームの責任強化**: プラットフォーム事業者に対し、Deepfakeコンテンツの検出・削除を義務付ける法規制や、迅速な対応プロセスを求めます。技術的な対策（AIによる検出など）の導入も促します。
    *   **国際協力**: Deepfakeは国境を越える問題であるため、国際的な規制基準の調和や捜査協力、情報共有を積極的に推進します。

### インターネット上のポルノ広告問題と対策の急務

日本のインターネット環境において、ポルノ広告が事実上野放しになっている現状は、国内外から異常事態として指摘されています。この問題は、子供への悪影響、性犯罪助長の危険性など、看過できない深刻な事態を引き起こしており、刑法175条の猥褻物頒布罪に関する安易な規制緩和論とは一線を画し、断固たる対策を講じる必要があります。

*   **日本の現状**: 現行法ではインターネット上のポルノ広告への適用が難しく、広告プラットフォームに対する明確な規制も存在しません。特に無料Wi-Fiや検索エンジン、SNSなどで不適切な広告が頻繁に表示され、海外からの訪問者が日本の状況に驚くケースも後を絶ちません。Deepfake技術を悪用したポルノ広告も増加しており、問題はさらに深刻化しています。
*   **子供への深刻な悪影響**:
    *   **性的コンテンツへの早期かつ意図しない接触**: 子供が日常的に利用するアプリやゲーム、動画サイトでポルノ広告に接触する機会が極めて多く、ある調査では10歳以下の子供の約3割が意図せずポルノコンテンツに触れた経験があると報告されています。
    *   **性に対する歪んだ認識の助長**: ポルノ広告に含まれる非現実的で誇張された性的描写は、子供たちの健全な性の発達を歪め、誤ったジェンダー観や対人関係における非現実的な期待を植え付ける危険性があります。
    *   **精神的な悪影響とリスク行動の誘発**: 不適切な広告への暴露は、子供に不安、羞恥心、罪悪感などを引き起こすだけでなく、クリック誘導によって個人情報漏洩やマルウェア感染といった更なるリスクに晒します。
*   **性加害者への影響と性犯罪助長の危険性**: 「ポルノの悪影響は具体的に検証されていない」といった意見は誤りであり、実態としてポルノ広告の氾濫は性加害者やその兆候のある者に以下のような影響を与え、性犯罪を助長する危険性を孕んでいます。
    *   **性的攻撃性の増幅**: 過激な性的コンテンツや非同意の性的描写（Deepfakeポルノなど）への頻繁な接触は、攻撃的な性的態度や歪んだ性的欲求を刺激し、正常化する可能性があります。
    *   **エスカレーションと抵抗感の低下**: 日常的にポルノ広告に接することで、性的搾取や非同意のコンテンツに対する心理的ハードルが下がり、より過激なコンテンツへのエスカレーションを招く可能性があります。
    *   **再犯リスクの増大**: 性犯罪歴のある者が容易にポルノ広告にアクセスできる環境は、性的衝動のコントロールを困難にし、再犯リスクを高める要因となり得ます。
*   **海外との比較**: EUのDSA（デジタルサービス法）や英国のオンライン安全法、米国のCOPPA（児童オンラインプライバシー保護法）、韓国の改正法など、諸外国では子供を保護するため、また性的な広告を規制するためにプラットフォームへの義務付けや罰則を伴う法整備が進んでいます。日本の対応は著しく遅れています。
*   **制定すべき法案・対策の具体案**:
    *   **インターネット広告包括規制法（仮称）**: 広告プラットフォームに対し、性的コンテンツのフィルタリング義務や年齢認証技術の導入を義務付け、特に子供がアクセスする可能性のある環境（アプリ、ゲーム、無料Wi-Fi、教育関連サイト等）でのポルノ広告を厳格に禁止します。違反事業者には売上に応じた高額な罰金や営業停止命令を科せるようにします。
    *   **Deepfakeポルノ等対策法（仮称）**: 非同意で作成されたDeepfakeポルノ及び類似の性的コンテンツの作成、所持、配布、広告利用を明確に犯罪とします。AI生成であることの表示義務、被害者による迅速な削除請求権、プラットフォームの削除義務を法制化します。
    *   **児童オンライン保護法（仮称）**: COPPA等を参考に、18歳未満の青少年がアクセスするデジタル環境での不適切な広告（ポルノに限らず暴力的なもの等も含む）を包括的に規制し、保護者向けツールの導入支援や相談窓口を拡充します。
    *   **プラットフォーム責任明確化法（仮称）**: SNS事業者や動画共有サイト運営者等に対し、国内法を遵守し、違法・有害コンテンツ（ポルノ広告、Deepfakeポルノ等）の迅速な削除（例：通報から24時間以内）と再発防止策を義務付けます。
    *   **デジタルリテラシー教育の抜本的強化**: 幼児期から高等教育に至るまで、デジタル倫理、情報を見抜く力、インターネット広告のリスクと対処法、性的同意の重要性などに関する教育を体系的に実施します。
    *   **国際連携の強化**: 国境を越えて配信されるポルノ広告やDeepfakeコンテンツに対処するため、捜査共助や情報共有に関する国際的な枠組みへの積極的な参加と、国内執行体制の強化を図ります。
    *   **被害者支援の拡充**: Deepfakeポルノやネット上の性被害に遭った被害者に対し、精神的ケア、法的支援、証拠保全、コンテンツ削除支援などをワンストップで提供する公的窓口を設置します。
    *   **刑法175条の見直し議論への慎重な対応**: 我々は、表現の自由の重要性を認識しつつも、子供たちの保護や性搾取の防止という観点から、刑法175条（猥褻物頒布罪）の安易な規制緩和には断固として反対します。むしろ、デジタル時代の新たな脅威に対応できるよう、同法の現代的な解釈や実効性のある運用、そして関連法規の整備こそが急務であると考えます。

チームみらいは、これらの問題に対して、具体的な法整備、教育・啓発活動の推進、プラットフォーム事業者の責任明確化、そして国際社会との連携を通じて、断固として取り組んでまいります。デジタル社会の利便性を誰もが安全に享受できる環境の実現を目指します。

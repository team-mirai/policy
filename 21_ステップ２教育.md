# Cコードベース公共図書カード制度 パイロット事業提案書

## 事業概要

**事業名**: 「読書・学び直し支援クーポン制度」パイロット事業  
**実施期間**: 2025年10月～2028年9月（3年間）  
**総予算**: 50億円（直接給付40億円 + システム・運営費10億円）

## 1. 基本設計

### 対象・規模
- **対象**: 全国20大学の在学生（各大学1万人規模）
- **給付額**: 1人当たり年間2万円のデジタルクーポン
- **年間受給者数**: 約20万人
- **地域分布**: 全国7ブロックに戦略的配置

### 利用対象書籍
Cコード（出版分類コード）による自動判定システム
- **✅ 対象**: C0xxx（一般のうち内容コードが本事業の趣旨に照らし合わせて適当なもの）、C1xxx（教養）、C2xxx（実用）、C3xxx（専門書）
- **✅ 形態**: 単行本、文庫、新書、辞典・図鑑
- **❌ 除外**: C0xxx（一般のうち娯楽書籍、0x70/0x90番代）、C0979,9979（コミック）

## 2. 大学選定戦略

### 地域ブロック別配置（20大学）
| 地域ブロック | 大学数 | 選定基準 |
|------------|-------|---------|
| 北海道・東北 | 3校 | 地方都市・過疎地域での効果検証 |
| 関東 | 5校 | 都市部集中vs分散効果の比較 |
| 中部 | 3校 | 中京圏での書店密度と利用傾向 |
| 関西 | 4校 | 関西圏での文化消費特性分析 |
| 中国・四国 | 2校 | 中規模都市での地域書店支援効果 |
| 九州・沖縄 | 3校 | 離島・遠隔地でのオンライン利用傾向 |

### 大学特性別分類
- **総合大学**: 12校（多様な学部構成による利用傾向分析）
- **理工系大学**: 3校（STEM書籍の需要動向）
- **文系大学**: 3校（人文・社会科学書籍の消費パターン）
- **芸術・専門大学**: 2校（専門書市場への影響）

## 3. システム・運営体制

### 技術インフラ（開発費4億円）
**デジタルクーポンシステム**
- マイナポータル連携による学生認証
- 大学学生証ICカード連携
- QR/NFCコード決済機能
- リアルタイム残高管理・利用履歴

**POS連携システム**
- 全国書店POSとのAPI接続
- Cコード自動判定・決済承認
- 不正利用検知AI（異常パターン検出）
- 地理情報連携（GPS範囲内店舗優先表示）

### 加盟店ネットワーク構築（2億円）
- **目標**: 各大学50km圏内の書店70%以上を加盟店化
- **支援策**: 小規模書店向け決済手数料無料
- **オンライン**: 主要電子書籍ストア（Amazon Kindle、楽天Kobo等）との連携

### 運営事務局（3年間2億円）
- **本部**: 東京（政策統括・システム運営）
- **地域拠点**: 7ブロックに担当者配置
- **大学連携**: 各大学に専任コーディネーター配置
- **問い合わせ対応**: 24時間チャットサポート

## 4. KPI・評価システム（2億円）

### 第三者評価体制
**評価委員会**
- 座長: 大学教育・図書館情報学専門家
- 委員: 出版関係者、書店経営者、経済学者、学生代表

**データ収集・分析**
- 慶應義塾大学SFC研究所等との連携
- 月次利用状況分析レポート
- 四半期ごとの効果測定・改善提案

### 段階別KPI設定

#### 短期目標（開始6ヶ月）
- **登録率**: 対象学生の85%以上
- **システム稼働率**: 99.5%以上
- **加盟店カバー率**: 各大学圏内70%以上

#### 中期目標（1-2年）
- **利用率**: 登録者の75%以上が年間1万円以上利用
- **読書冊数増加**: 1人当たり年間+2冊以上
- **地域書店利用比率**: 全体の40%以上
- **追加購入誘発**: クーポン1円あたり1.3円の総消費

#### 長期目標（3年）
- **継続的読書習慣**: 卒業生の60%が年間10冊以上維持
- **地域経済効果**: 参加大学周辺の書店売上5%向上
- **出版市場拡大**: 対象カテゴリー書籍の売上3%向上

## 5. 想定される課題と対策

### 技術的課題
**課題**: Cコード誤付番による不適切書籍の混入
**対策**: 
- 売上上位0.5%書籍の月次サンプル監査
- 機械学習による内容とCコードの整合性チェック
- 出版社への是正勧告・ペナルティ制度

### 地域格差
**課題**: 大都市部書店・オンラインストアへの集中
**対策**:
- アプリ画面で地域書店を優先表示
- 地方書店限定ボーナスポイント制度
- 大学生協との連携強化

### 財政持続性
**課題**: パイロット終了後の予算確保
**対策**:
- 段階的自己負担導入（50%マッチング制）
- 企業版ふるさと納税の活用
- 成功大学での継続実施（大学独自予算）

## 6. 実施スケジュール

- **予算要求**: 次期5年計画策定

## 7. 期待される成果

### 定量的成果
- **読書促進**: 参加学生20万人×2冊増 = 年間40万冊の追加消費
- **経済効果**: 直接効果40億円 + 誘発効果15億円 = 総額55億円
- **地域書店支援**: 参加地域での書籍売上20億円以上の底上げ

### 定性的成果
- **学び直し文化の醸成**: 大学生の自主的学習意欲向上
- **デジタル決済インフラ**: 文化分野でのキャッシュレス促進
- **政策ノウハウ蓄積**: 全国展開時の制度設計最適化

## 結論

本パイロット事業は、限定的な予算規模で「学び直し支援」「地域書店振興」「出版文化促進」の同時達成を目指す画期的な取り組みです。

20大学という規模は統計的有意性を確保しつつ、システム運営の効率化も実現できる最適なバランスであり、成功すれば全国展開（年間3,000億円規模）への確実な道筋を構築できます。

欧州諸国の文化クーポン制度を参考としながら、日本独自のCコードシステムと既存デジタルインフラを活用することで、表現の自由を保護しつつ効果的な政策実現が期待されます。

| -------------- | ----------------------------------------------- |

## 5. 実務フロー（B2C/B2B）

1. **ユーザー登録**

   * マイナポータルAPIで年齢・居住確認
   * アプリに20,000円残高表示
2. **加盟店決済**

   * 客側QR生成 → 店側POS読み取り → ISBN自動照合 → 残高減算
   * 取引データ（ISBN, 価格, 店舗geo）を即時クラウド送信
3. **売上精算**（週次）

   * インボイス番号と突合 → 国庫から加盟店へ銀行送金
4. **モニタリング**

   * 分析ダッシュボードでKPI（冊数、ジャンル分布、地理分散）を可視化
   * 不審パターンは自動フラグ → 運営事務局が照会

## 9. 【補足】ISBN ではなく **Cコード** をベースにする理由と実装

### 2-1. Cコードの構造（4 桁）

| 桁          | 役割                            | 例         | 出典                     |
| ---------- | ----------------------------- | --------- | ---------------------- |
| **1 桁目**   | 販売対象（0一般／1教養／2実用／3専門／6学参Ⅰ…など） | C**3**214 | [isbn.jpo.or.jp][5] |
| **2 桁目**   | 形態（0単行本／1文庫／2新書／9コミック 等）      | C3**2**14 |                        |
| **3-4 桁目** | 内容細目（00総記、30社会科学、41数学…）       | C32**14** |                        |

> * **利点**：
>
>   1. **学び直し向けカテゴリだけ切り出せる**（例：1-教養・2-実用・3-専門、2-新書・5-辞典・6-図鑑など）。
>   2. POS/出版流通データに既登録、電子書籍ストア API でも取得可。
>   3. 価格（JAN）と独立しているため税区分の**自動判定が容易**。
> * **懸念**：出版社が自主申告で付番→**誤付番インセンティブ**（クーポン消費狙い）が生じうる。監査ルールが必須。

### 2-2. 技術フロー概略

1. POS または電子ストアが **ISBN → Cコードマッピング DB** を即時照合。
2. 対象 Cコード（例：1×××, 2×××, 3×××, 6×××, 7×××, 41～49, 50～59…）ならクーポン残高から決済。
3. 決済ログに ISBN・Cコードを同時送信 ⇒ 月次監査で「売上上位 0.5 %」「短期急伸」タイトルを抽出し手動確認。

## 11. 【第二フェーズ】年齢拡大版クーポン - ３段階モデル

| 区分                 | ねらい              | 金額     | 利用上限                 | 備考                            |

| **Stage-A（18-22）** | 高校卒業直後の読書・資格取得促進 | 2 万円   | 紙/eBook・学参/語学/資格系・文学 | 先行の Pass Culture の上限 €200 を参考 |
| **Stage-B（23-29）** | 就労初期のリスキリング      | 1.5 万円 | C1教養・C2実用・C3専門・ビジネス書 | オンライン講座バウチャー併用可               |

3. 当初３年間は **パイロット＋抽選制**（各層 20 %）で効果検証 → KPI 達成で拡大。

## 12. リスク・法的論点と対策

| リスク                | 発生要因                   | 対策                                                            |
| ------------------ | ---------------------- | ------------------------------------------------------------- |
| **誤付番による不適格書籍の流入** | 出版社が恣意的に Cコード変更        | ― 日本出版インフラセンター（JPO）と文化庁が**随時サンプル監査**<br>― 悪質ケースはクーポン決済停止＋是正勧告 |
| **大都市ストア集中**       | オンライン偏重／地方書店加盟遅れ       | ― アプリ初期表示を **自宅 50 km 圏内加盟店優先**設定<br>― 地方書店は決済手数料を 0 %に据え置き   |
| **財政硬直化**          | 年 3,000 億円規模の恒久事業化     | ― 政府 1 兆円リスキリング基金内で **毎年上限を設定**（景気連動）([reskilling.com][3])    |
| **表現規制論争再燃**       | 「Cコードで“0一般”にアダルトが混在」など | ― Cコード基準に**一切の個別検閲を入れない**（租税法律主義・検閲禁止を順守）                     |
